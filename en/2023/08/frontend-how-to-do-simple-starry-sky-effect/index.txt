9:T1283,<p>When I'm developing my self software service, there is a background with stars and nights effect.</p>
<p>I thought it is difficult to develop when I was designing the background (I do design and development in same role)</p>
<p>However, it finished with an easy way, so today I would like to share the tips.</p>
<h2 id="requirement-introduction">Requirement Introduction</h2>
<p>This is my design of the background by Figma.</p>
<p><img src="https://minayu0416.files.wordpress.com/2023/08/screenshot-2023-08-18-at-8.34.34-pm.png" alt="starry-sky-by-figma"></p>
<p>This is a starry night background by using linear-gradient and draw the stars by hands.</p>
<p>Here is the Home Page of the service that I'm developing.</p>
<p><img src="https://minayu0416.files.wordpress.com/2023/08/screenshot-2023-08-19-at-12.50.59-am.png" alt="miri-home-page"></p>
<p>Based on this background and I add some elements, then this will be a good divination service. This is the first front-end design of the service that I self-learn by using Figma and Front-end skills.</p>
<p>I got 2 methods to develop it from my colleagues.</p>
<ul>
<li>Use CSS to draw each line and points.</li>
<li>Put Image directly.</li>
</ul>
<p>In the beginning, I thought if only put an image, the front-end will look fake, but seems I was wroung xD.</p>
<h2 id="resolution">Resolution</h2>
<p><img src="https://minayu0416.files.wordpress.com/2023/08/screenshot-2023-08-18-at-8.34.34-pm.png" alt="starry-sky-by-figma"></p>
<p>So, we are gonna to see the Figma, we need following steps.</p>
<ol>
<li>Use linear-gradient to render night space color in background.</li>
<li>Export the stars to SVG file by using Figma Plugin.</li>
<li>Integrate SVG file and background.</li>
</ol>
<p>So first, we are gonna use linear-gradient to render night scene, here are code.</p>
<pre><code class="language-css">/* We use three colors starts from top to bottom and render the night space effect.  */
.wallpaper {
  background: linear-gradient(
    to bottom,
    rgba(23, 18, 50),
    rgba(49, 22, 56),
    rgba(53, 53, 119)
  );
}
</code></pre>
<pre><code class="language-html">&#x3C;div class="wallpaper">&#x3C;/div>
</code></pre>
<p>So that can done the good night effect.</p>
<p><img src="https://minayu0416.files.wordpress.com/2023/08/screenshot-2023-08-18-at-9.21.00-pm.png" alt="night-effect"></p>
<p>Second, we need to export stars to SVG file by Figma Plugin.</p>
<p>I use a tool called SVG Export, you click your componenets (don't forget to group by them.) and then click the plugin, you can download it as SVG file, we will use it later.</p>
<p><img src="https://minayu0416.files.wordpress.com/2023/08/screenshot-2023-08-18-at-9.22.27-pm.png" alt="export-step-of-stars"></p>
<p><img src="https://minayu0416.files.wordpress.com/2023/08/screenshot-2023-08-18-at-9.23.01-pm.png" alt="export-svg-tool-in-figma"></p>
<p>Third, now we need to integrate them as a background.</p>
<p>I tried to put img tag on div directly, but it will cover other divs, and we can not click other divs. I also use <code>z-index</code>, however in one page, I have a lot of cards with z-index arrangement, it will be a nightmare to adjust each z-index sequences.</p>
<p>So, I use <code>before</code> and <code>relative</code> + <code>absolute</code> to reach our requirement, please see following code.</p>
<pre><code class="language-css">.wallpaper {
  background: linear-gradient(
    to bottom,
    rgba(23, 18, 50),
    rgba(49, 22, 56),
    rgba(53, 53, 119)
  );
  /* We need to add position: relative, and make SVG file as absolute, so the SVG can be attached on background. */
  position: relative;
}

.wallpaper-color::before {
  content: "";
  position: absolute;
  /* Because the SVG Export still transfer some space, so we need some adjustment position to put the SVG to right place. */
  top: -46px;
  left: 0;
  width: 100%;
  height: 100%;
  /* This is the SVG file we exported by Figma Plugin. */
  background-image: url("./static/stars.svg");
  background-repeat: no-repeat;
  background-position: center;
  background-size: contain;
}
</code></pre>
<pre><code class="language-html">&#x3C;div class="wallpaper">&#x3C;/div>
</code></pre>
<p>Remember adding the <code>position:relative</code> in background and adding <code>position:absolute</code> to your SVG css layer, so that we can achieve the effect we want.</p>
<p><img src="https://minayu0416.files.wordpress.com/2023/08/screenshot-2023-08-18-at-8.34.34-pm.png" alt="starry-sky-by-figma"></p>
<p>That's the tips I want to share to you!</p>
<p>Because I'm a self-learned Front-end and learning by doing projects, so if you have other good resolution on this topic, please share to me and I will update my post.</p>
