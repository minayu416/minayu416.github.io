2019年新年新希望

June 17, 2019 minayu416

1 minute read

  • Twitter

  • Google+

  • Facebook

  • Reddit

  • LinkedIn

  • StumbleUpon

  • Pinterest

  • Email

一個月前，焦慮症又復發，我發現自己給自己的壓力太大了。

我不經意的就加了許多Loading在身上，甚至超載了。

我反覆Review了以後，我認為我給自己太多目標跟夢想

我什麼都想做，卻什麼都還沒開始做
一直不斷的思考及反反覆覆

經過了大約一個多月對於焦慮的治療，以及最近對生活的改善以及目標的控管

我已經慢慢的又康復，隨著這段時間我也重新審視了自己給自己的目標，並將他們加以修改。

這是去年11月給今天訂定的目標，後來有在做修正

2018訂定的2019年新年新希望

2019年新年新希望 - 第一季 Review

舊的文章有些自己激勵人心的前言，可以點擊上面的文章觀賞

再訂定自己接下來的半年要做些什麼事之前，一樣講些激勵人心的話 ~

出社會八個月了，時而忙碌時而打蚊子

忙碌的時候我就哀哀叫，空險的時候我就特別緊張我沒事做沒有學習 ..

絲毫著憂慮與焦慮如果一刻沒事做，就一直不會進步

當然我也會抓時間偷閒，前陣子演變成: 沒事做真好，狂看網站跟放鬆

可是說真的，我雖然腦袋這樣想，但我的內心與身體卻呈現十分焦慮的狀態

我不可能在公司做些私人的事情，但任由每天八個小時空閑的虛度

我竟也開始覺得 是不是該換個地方學習比較好？

就這樣一直拖，雖然沒事做就有錢拿，可是這真的是我要做的嗎？

易經我也抽了，不要離開繼續做，那我該怎麼辦？

我終於在某次上網找，找的內容是不忙很空閑，沒事做，卻又不想這樣一直虛度。

看到了某樓某句話，他是這樣說的

你以為你每天在公司沒事做是放鬆, 不用做事又沒事的度過了一天
殊不知你在浪費自己的時間

看到這句話，我整個人突然就醒來了

對吼，這些時間都是我的呀！我可以好好的利用這些時間來做學習

於是，當我沒事做的時候，我就開始打開IT的學習文件開始學習，也順便問問資深的同事怎麼學習某個技術。

就這樣，我慢慢開始學了些IT的新技術，開始研究技術並且運用

也打開了外語的文法網站，沒事就多看幾下

然後別為了流量發文，我只筆記比較關鍵的部分，就不用天天為了網站流量而憂慮。

少了大把的時間產初低品質的文章，我有更多的時間學習及規劃目標，我有更多的時間運動，我有更多的時間把自己的人生過好，而並非刻意做些無意義的事。

以下，是我這一個多月改動的目標：

目標清單

關於個性上

由於前一週焦慮復發，我歸納了幾點，最為主要想先改善的

  1. 相信自己，對自己有自信，相信自己所做的行為及抉擇
  2. 維持專一自己訂定的目標與道路(最終回歸自我)
  3. 不要亂受他人影響，相信自己
  4. 擁有開闊的思想及正面思考，思想不要時常被侷限住，坦然面對一切

  • 要有自信！要讓自己有自信，勇敢的表達自我/主動與人溝通/主動爭取，勇敢的做自己，有自己的想法

  • 正向思考, 正向看待, 正面接受, 樂觀

  • 學著不要直接反應內化所有的害怕, 焦慮與不安，包括任何源自於自卑, 沒自信與壞的思想，放下不安，害怕與焦慮，擺脫原生造成的行為與思想，什麼事都不會發生

  • 變成一個有獨特風格(內涵)/ 獨特思想/ 有個性的人

  • 變得更獨立，更堅強不要軟弱，不要容易被影響，內心堅定點，更對自己的人生負責

  • 國外的思想，要有開擴的心思，接受外來的意見但也堅持自己的想法不要隨波逐流，不對任何事情有先入為主的思想，保持開放的態度

我打算學著依照自己的喜好做抉擇，並負責自己做的決定

我想要改掉從別人口中評論自己的壞習慣，學著活出自己的價值

面對它, 解決它, 放下它

關於興趣/工作上

主要想持續的目標領域為: 語言, 資訊, 運動

資訊

目標當更厲害的培根(Back-End)工程師 或 學習DevOps技術

部落格 (對自己的壓力太大，擱置中)

  • 擴展新的部落格, 並在設計/文章/SEO都比舊的好!(先暫時擺著了，因為經營部落格給我的壓力太大)

研究Hugo的css, Golange語法, 將部落格改成想要的樣子: 嘗試使用 Hugo Future Imperfect 及Tranquilpeak主題 (已經辦到)

嘗試開發自己的AI軟體機器人

這…其實是我自己醞釀好久，看著惡靈古堡的紅后以及鋼鐵人的賈維斯還有Apple 的Siri 跟 Google助理 都是 AI，從我研究所接觸りんな開始，綜合先前累積的 夢想 的 能量，開始有了想要開發自己的AI的動力

但也許是研究所的知識與經驗累積不足，還不知從何下手。有幸畢業後進了開發AI的公司，了解AI 是怎麼做開發與怎麼運作，也慢慢的讓我燃起先前的AI 夢。

雖然不知道什麼時候會開始，想先記錄下來，這個目標跟Youtube影片都是近期新增的目標與夢想，期望在未來有機會開始執行！
