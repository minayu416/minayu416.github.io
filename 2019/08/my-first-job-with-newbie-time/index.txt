9:T37d5,<p>我已經好久沒更新文章了</p>
<p>六月份開始我就很少寫文章，因為我在公司從原本開發產品的團隊，調到另外一個快要結案的專案努力奮鬥。</p>
<p>就這麼的忙了快三個月，每天幾乎都加班完成工作，我也沒什麼空閑時間寫文章</p>
<p>當然這跟我後期沒什麼在學新的電腦技術也有關係，也不會特別發技術文章</p>
<p>雖然說沒在學技術，但整個回過頭來回顧，我整體的能力是有往上進步的，那麼就讓我來記錄一下作為職場新鮮人的第一份工作回顧吧。</p>
<p>再過不久，我即將要從我現在的公司 <code>畢業</code> 了，之所以使用 <code>畢業</code> 一詞，是我認為我在這家公司的奉獻與經歷讓我作為一個職場新鮮人有個很完美的體驗，到了這個階段，我也準備要邁向下一步了，所以我用 <code>畢業</code> 一詞來作為這段 <code>職場旅行</code> 的結束。</p>
<h1 id="第一章---面試">第一章 - 面試</h1>
<p>一年前的求職過程以及剛踏入公司的回憶還歷歷在目，轉眼間已經過了一年了。</p>
<p>還記得一年前我是一個剛從研究所畢業的職場新鮮人，抱著自己下定決心好好衝刺拼了兩年半參與很多專案與在校輝煌紀錄的經驗，我用心的做了我的履歷，並附上我那時還使用的 <a href="https://minayu0416.wordpress.com/">wordpress</a> 部落格，投了十家公司，五家有回音，面試了三家公司，短短不到兩週，我就進入了這家公司工作。</p>
<p>對於公司的第一印象是 <code>美商</code>，且十分神秘，直到二面才邀請我進入辦公室參觀。</p>
<p>二面一進門，整個辦公室透露著新創年輕的室內擺設，走進當時的 rd 區，一大片窗戶對外，陽光透進來，旁邊緊鄰著不小的陽台讓在這工作的員工可以時不時走向陽台休息，甚至還有外國同事抱著電腦就坐在陽台地板曬太陽。</p>
<p>rd 區就像剛蓋好的圖書館自修室，十分安靜，每個人都顧著自己的電腦，也有些同事用好奇的眼神看著我走進小會議室面試。我坐在小會議室裡顯得有點緊張，不斷的告訴自己沒事，且環境很溫馨讓我情緒緩和下來。</p>
<p>人資打開門走了進來是一個年紀看起來很輕的女生，我們面談以及填寫資料後，打電話跟即將要從台灣起飛去中國的人資長聊了一回兒，當時的人資長我覺得很親切，很像我在 Juniata 認識的美國教授，他立即問我能不能下週就來上班，我說可以。</p>
<p>走出小會議室，看著這些 rd 朋友，其中一位女生 rd 對著我笑，我整個就被煞到 XD。另外一個男生 rd 我對他的印象就是 <code>他一定是一個台大的學生</code> XD，好好奇自己居然會有這樣的想法迸出</p>
<p><code>如果上了，我一定要來這家公司</code></p>
<h1 id="第二章---初入職場">第二章 - 初入職場</h1>
<p>剛進到公司的第一天下大雨，我的鞋子全都濕掉了，我穿著正裝前往公司報到第一天。</p>
<p>我進入職場的第一份工作開始了</p>
<p>先前往人資處報告填寫資料後，就走到先前面試的 rd 區，選了個窗旁邊的位置坐下，我十分喜歡這個位置。</p>
<p>不久，面試我的主管向前來帶領我進小會議室，好好的了解我的能力以及我應該要先做些什麼事。</p>
<p>面對第一次見面的主管，真心覺得他跟電話面試想像中的聲音完全不同呀 XD</p>
<p>我還記得那天主管問我，有什麼問題嗎？ 我則是回他 <code>下雨可以穿夾腳拖嗎？</code>，確實奇怪但我的腦袋中想法其實是因為鞋子濕掉了腳會很臭，會影響到旁邊的人。</p>
<p>沒想到主管聽到立馬大笑，笑聲是足以嚇到我的境界，也許他覺得著個問題很有趣吧。</p>
<p>與另外一個看起來很資深的工程師與我同一天報到。</p>
<p>最先接收到的任務是 <code>了解 git</code> 以及手邊有本書是 <code>D3 與 資料分析</code> 要在很迅速的時間內看完。</p>
<p>接著我就用在 <code>讀書</code> 的方式吸收這些資訊</p>
<p>如果是看那本書的話，主要學的部分是 <code>restful api</code>, <code>pandas</code>, <code>request</code> (天阿我那時候怎麼這麼菜 ~ )</p>
<p>但就最簡單的 <code>request 爬蟲</code> 我都理解了很久，當然這些都是對於剛踏入職場的新鮮人再新鮮不過的知識了</p>
<p>還記得那時候打開我的 wordpress 部落格永遠掛在瀏覽器上，只要學習到任何知識包括 linux 指令我都寫在部落格上紀錄，晚上特別留的晚一點就是編輯以及回顧今天學習了什麼知識。</p>
<p>我還記得剛踏入職場的兩三個月，我每天頭腦都很漲，因為我每天都吸收大量的知識進腦內，甚至根本覺得出社會工作真的跟在學校學的完全 gap 超級大，更何況我是一個在校期間都在接案實作的人。當時真心覺得在學校的程度，真的完全不能與職場相比，差太多了。期望學校能夠多一點實作與社會接軌就太好了！</p>
<p>還記得第一天上班就覺得十分疲勞，剛開始上班的一個月都覺得一天過得非常久，但是學習到很多知識。</p>
<p>我們 rd team 每天中午都一起吃飯，我很快的大致上了解每個人的角色以及一起談論日常生活。</p>
<p>在我進來公司沒多久後，分別來了同年齡新的人資妹妹以及兩位新的 pm，等於我們是同一時間進來的，但是他們的辦公室在另外一邊，當時沒有業務上往來也很少走過去另外一邊，也就和 pm 與人資區不熟悉，往後會比較慢才認識熟悉。</p>
<h1 id="第三章---新鮮人小試身手">第三章 - 新鮮人小試身手</h1>
<p>沒幾天，主管又出了幾個小功課開始讓我實作，像是 用 <code>flask 框架寫 api</code> 以及 <code>測試如何在一秒對mysql打入4000次資料</code>，學到了 <code>flask</code>, <code>route</code> 與 當時還不太曉得是什麼的 <code>api</code>，還有學著進入 <code>mysql</code> 以及使用 <code>jmeter</code> 打 <code>mysql</code></p>
<p><code>這樣吧~ 你寫兩個小工具, 一個driver 一個用orm，去Insert資料</code></p>
<p>你知道什麼是 <code>sqlalchemy</code> 嗎？ 不知道的話就去研究。 去用 <code>sqlalchemy</code> 和 <code>pymysql</code> 寫 資料庫 Insert 指令。</p>
<p>然後我就和 <code>sqlalchemy</code> 玩耍了好幾天...</p>
<p>對於當時的我，完全霧煞煞，每個名詞都十分的新穎，也不知道該如何下手，這些都還只是基本的小功能，還有常常主管我請我幫忙寫一些小工具，像是 <code>讀檔轉換字串寫入檔</code> 以及 <code>一些小陣列</code>。</p>
<p><img src="https://minayu0416.files.wordpress.com/2019/08/selfiecity_20180910141735_save.jpg" alt=""></p>
<p><code>累了往後看外面，壓力煩惱通通趕走!!</code></p>
<p><code>每個菜鳥工程師必經之路，就是剛學習時頭腦大量接受訊息呀!!</code></p>
<p>主管雖然沒對我兇過，但是他給我的壓力是蠻大的，對於一個原本就不知道如何抗壓的小女生加上初入職場的新鮮人，僅僅只是一句 <code>學得如何?</code> 都能讓我壓力很大。</p>
<p>由於主管看起來都很忙碌，我也不好意思都問他，慢慢的我開始會去詢問周圍同事。</p>
<p>詢問了坐在主管面前跟我同一天報到的資深 Java 工程師，由於不曉得為什麼使用<code>jmeter</code> 打 <code>api</code> 十萬筆資料會開始有掉資料的現象。詢問了資深前輩，他很熱心的教我如何 <code>debug</code>， 什麼是 <code>debug</code> 的流程。</p>
<p>這個掉資料的 bug 大概讓我解了四天還是不了解為什麼，最後從前輩的口中得知 <code>connection pool</code> 與 <code>thead</code>，兩個好深奧的名詞，完全霧煞煞的 qq。</p>
<p>有點忘記後續是如何了</p>
<p>與同年齡的資料科學家討論該怎麼處理 <code>python 的 字典取值</code>。喔對了，在最一開始的時候， <code>python的字典型態如何取值</code> 簡直讓我傷透了腦筋。</p>
<p>在實作另一個 <code>轉換文字的小工具</code> 時，和當時的資料科學家討論後，他則是建議我可以使用 <code>enumerate</code> 的語法來解決問題。</p>

<h1 id="第四章---我的第一個專案-北京示範天使線網頁">第四章 - 我的第一個專案: 北京示範天使線網頁</h1>
<p>好快的進入了第二個月，主管也一直不斷地宣告第三個月之後我要開始長大堅強了</p>
<p>讓我好緊張很擔心自己，沒自信認為自己無法勝任。</p>
<p><code>北京示範天使線</code> 是我進公司第一個接觸的專案，他是一個大型的網頁，有高達 11 個不同的場景，主要是實作中國那邊的工廠資料分析與資訊顯示。像是工站監視，良率不良率計算，錯誤警報，人流場景，員工資料 ... 等等。</p>
<p>這次的任務變成了在三週內將 80 幾個 task 完成</p>
<p>當然第一次接觸工業類型的專案讓我感覺好新奇，在學校絕對接觸不到的專案。但是在計算良率, kpi 以及其他工業工程相關的計算時，也讓我十分的頭痛。</p>
<p>我開始運用我的 python 技術能力來完成專案，這其中我學習了 <code>pandas</code>, <code>read_csv</code>, <code>requests</code>, 還解謎了原來是前端的程式要一直不斷 call 我的程式才能達到 <code>不斷的讀取資料秀在畫面上</code>，對 python 的了解更上一層樓，也慢慢的累積的些基本能力。</p>
<p>當初十分害怕無法在三週內完成任務，資深前輩還幫助我一起開發分擔了一些任務。</p>
<p><code>前輩前輩，為什麼我看你好像都沒什麼工作！公司請你來是負責哪個工作呢？</code></p>
<p><code>- 公司請我來，是來教你的啊</code></p>
<p>我滿臉問號的回到了我的座位，繼續寫未完成的任務。</p>
<p>進來公司一兩個月了，總覺得人資很活潑，常常舉辦好多活動，我也工作得十分開心，壓力是有的，但我也漸漸地很喜歡公司。中秋節時一起煮火鍋，剛進公司就和 rd 同事以及 pm 們一起唱歌，還有猜謎活動。</p>
<p>當然每天中午，我們 rd 團隊就一起出發吃飯，一起聊天一起分享想法，很團結很溫馨，很像家人。</p>

<h1 id="第五章---主管離職了">第五章 - 主管離職了</h1>
<p>三個月到了，像是一個宣告期。</p>
<p>rd 的主管開始很少出現在辦公室，我有一股很奇怪的預感，總覺得什麼快消失的感覺，我提早感受到主管要離職了呢。</p>
<p>他開始一對一對每個同事都聊過一次天，在他離開公司前一天傍晚安排跟我聊天。</p>
<p>大致上都已經遺忘了，只是我當時十分擔心我男友自己創業快沒錢。</p>
<p>而主管則是跟我說，<code>他創業如果燒到沒錢了，你先是笑死他</code></p>
<p><code>- 然後如果是我，我會養他</code></p>
<p>聽完後，我沒什麼質疑，就結束了我們的談話。</p>
<p>主管給我的最後一個功課是 <code>研究並架設LDAP</code>。</p>
<p>主管離職後沒多久，我做的 <code>北京示範天使線</code> 專案也被停案了，頓時我沒有工作可以努力，突然變得茫然與困惑。</p>
