[工作旅行記] 我的第一份工作(1) - 幼嫩小女孩

這是我的第一份工作，也是我第一次挑戰新創公司，而我即將要從這個公司畢業了，回顧這一年的經驗帶給我很完美的體驗，以及許多感人的回憶。所以我決定好好地將它記錄下來。

我已經好久沒更新文章了

六月份開始我就很少寫文章，因為我在公司從原本開發產品的團隊，調到另外一個快要結案的專案努力奮鬥。

就這麼的忙了快三個月，每天幾乎都加班完成工作，我也沒什麼空閑時間寫文章

當然這跟我後期沒什麼在學新的電腦技術也有關係，也不會特別發技術文章

雖然說沒在學技術，但整個回過頭來回顧，我整體的能力是有往上進步的，那麼就讓我來記錄一下作為職場新鮮人的第一份工作回顧吧。

再過不久，我即將要從我現在的公司 畢業 了，之所以使用 畢業 一詞，是我認為我在這家公司的奉獻與經歷讓我作為一個職場新鮮人有個很完美的體驗，到了這個階段，我也準備要邁向下一步了，所以我用 畢業 一詞來作為這段 職場旅行 的結束。

第一章 - 面試

一年前的求職過程以及剛踏入公司的回憶還歷歷在目，轉眼間已經過了一年了。

還記得一年前我是一個剛從研究所畢業的職場新鮮人，抱著自己下定決心好好衝刺拼了兩年半參與很多專案與在校輝煌紀錄的經驗，我用心的做了我的履歷，並附上我那時還使用的 wordpress 部落格，投了十家公司，五家有回音，面試了三家公司，短短不到兩週，我就進入了這家公司工作。

對於公司的第一印象是 美商，且十分神秘，直到二面才邀請我進入辦公室參觀。

二面一進門，整個辦公室透露著新創年輕的室內擺設，走進當時的rd區，一大片窗戶對外，陽光透進來，旁邊緊鄰著不小的陽台讓在這工作的員工可以時不時走向陽台休息，甚至還有外國同事抱著電腦就坐在陽台地板曬太陽。

rd區就像剛蓋好的圖書館自修室，十分安靜，每個人都顧著自己的電腦，也有些同事用好奇的眼神看著我走進小會議室面試。我坐在小會議室裡顯得有點緊張，不斷的告訴自己沒事，且環境很溫馨讓我情緒緩和下來。

人資打開門走了進來是一個年紀看起來很輕的女生，我們面談以及填寫資料後，打電話跟即將要從台灣起飛去中國的人資長聊了一回兒，當時的人資長我覺得很親切，很像我在Juniata 認識的美國教授，他立即問我能不能下週就來上班，我說可以。

走出小會議室，看著這些rd朋友，其中一位女生rd對著我笑，我整個就被煞到XD。另外一個男生rd我對他的印象就是 他一定是一個台大的學生 XD，好好奇自己居然會有這樣的想法迸出

如果上了，我一定要來這家公司

第二章 - 初入職場

剛進到公司的第一天下大雨，我的鞋子全都濕掉了，我穿著正裝前往公司報到第一天。

我進入職場的第一份工作開始了

先前往人資處報告填寫資料後，就走到先前面試的rd區，選了個窗旁邊的位置坐下，我十分喜歡這個位置。

不久，面試我的主管向前來帶領我進小會議室，好好的了解我的能力以及我應該要先做些什麼事。

面對第一次見面的主管，真心覺得他跟電話面試想像中的聲音完全不同呀XD

我還記得那天主管問我，有什麼問題嗎？ 我則是回他 下雨可以穿夾腳拖嗎？，確實奇怪但我的腦袋中想法其實是因為鞋子濕掉了腳會很臭，會影響到旁邊的人。

沒想到主管聽到立馬大笑，笑聲是足以嚇到我的境界，也許他覺得著個問題很有趣吧。

與另外一個看起來很資深的工程師與我同一天報到。

最先接收到的任務是 了解 git 以及手邊有本書是 D3 與 資料分析 要在很迅速的時間內看完。

接著我就用在 讀書 的方式吸收這些資訊

如果是看那本書的話，主要學的部分是 restful api, pandas, request (天阿我那時候怎麼這麼菜 ~ )

但就最簡單的 request 爬蟲 我都理解了很久，當然這些都是對於剛踏入職場的新鮮人再新鮮不過的知識了

還記得那時候打開我的wordpress 部落格永遠掛在瀏覽器上，只要學習到任何知識包括 linux 指令我都寫在部落格上紀錄，晚上特別留的晚一點就是編輯以及回顧今天學習了什麼知識。

我還記得剛踏入職場的兩三個月，我每天頭腦都很漲，因為我每天都吸收大量的知識進腦內，甚至根本覺得出社會工作真的跟在學校學的完全gap超級大，更何況我是一個在校期間都在接案實作的人。當時真心覺得在學校的程度，真的完全不能與職場相比，差太多了。期望學校能夠多一點實作與社會接軌就太好了！

還記得第一天上班就覺得十分疲勞，剛開始上班的一個月都覺得一天過得非常久，但是學習到很多知識。

我們rd team 每天中午都一起吃飯，我很快的大致上了解每個人的角色以及一起談論日常生活。

在我進來公司沒多久後，分別來了同年齡新的人資妹妹以及兩位新的pm，等於我們是同一時間進來的，但是他們的辦公室在另外一邊，當時沒有業務上往來也很少走過去另外一邊，也就和pm與人資區不熟悉，往後會比較慢才認識熟悉。

第三章 - 新鮮人小試身手

沒幾天，主管又出了幾個小功課開始讓我實作，像是 用 flask 框架寫 api 以及 測試如何在一秒對mysql打入4000次資料，學到了 flask, route 與 當時還不太曉得是什麼的 api，還有學著進入 mysql 以及使用 jmeter 打 mysql

這樣吧~ 你寫兩個小工具, 一個driver 一個用orm，去Insert資料

你知道什麼是 sqlalchemy 嗎？ 不知道的話就去研究。 去用 sqlalchemy 和 pymysql 寫 資料庫 Insert 指令。

然後我就和 sqlalchemy 玩耍了好幾天…

對於當時的我，完全霧煞煞，每個名詞都十分的新穎，也不知道該如何下手，這些都還只是基本的小功能，還有常常主管我請我幫忙寫一些小工具，像是 讀檔轉換字串寫入檔 以及 一些小陣列。

累了往後看外面，壓力煩惱通通趕走!!

每個菜鳥工程師必經之路，就是剛學習時頭腦大量接受訊息呀!!

主管雖然沒對我兇過，但是他給我的壓力是蠻大的，對於一個原本就不知道如何抗壓的小女生加上初入職場的新鮮人，僅僅只是一句 學得如何? 都能讓我壓力很大。

由於主管看起來都很忙碌，我也不好意思都問他，慢慢的我開始會去詢問周圍同事。

詢問了坐在主管面前跟我同一天報到的資深 Java 工程師，由於不曉得為什麼使用jmeter 打 api 十萬筆資料會開始有掉資料的現象。詢問了資深前輩，他很熱心的教我如何 debug， 什麼是 debug 的流程。

這個掉資料的bug 大概讓我解了四天還是不了解為什麼，最後從前輩的口中得知 connection pool 與 thead，兩個好深奧的名詞，完全霧煞煞的 qq。

有點忘記後續是如何了

與同年齡的資料科學家討論該怎麼處理 python 的 字典取值。喔對了，在最一開始的時候， python的字典型態如何取值 簡直讓我傷透了腦筋。

在實作另一個 轉換文字的小工具 時，和當時的資料科學家討論後，他則是建議我可以使用 enumerate 的語法來解決問題。

進來公司的第一份大餐XD

滿漢全席!!

第四章 - 我的第一個專案: 北京示範天使線網頁

好快的進入了第二個月，主管也一直不斷地宣告第三個月之後我要開始長大堅強了

讓我好緊張很擔心自己，沒自信認為自己無法勝任。

北京示範天使線 是我進公司第一個接觸的專案，他是一個大型的網頁，有高達11個不同的場景，主要是實作中國那邊的工廠資料分析與資訊顯示。像是工站監視，良率不良率計算，錯誤警報，人流場景，員工資料 … 等等。

這次的任務變成了在三週內將 80幾個task 完成

當然第一次接觸工業類型的專案讓我感覺好新奇，在學校絕對接觸不到的專案。但是在計算良率, kpi以及其他工業工程相關的計算時，也讓我十分的頭痛。

我開始運用我的python技術能力來完成專案，這其中我學習了 pandas, read_csv, requests, 還解謎了原來是前端的程式要一直不斷call 我的程式才能達到 不斷的讀取資料秀在畫面上，對 python 的了解更上一層樓，也慢慢的累積的些基本能力。

當初十分害怕無法在三週內完成任務，資深前輩還幫助我一起開發分擔了一些任務。

前輩前輩，為什麼我看你好像都沒什麼工作！公司請你來是負責哪個工作呢？

- 公司請我來，是來教你的啊

我滿臉問號的回到了我的座位，繼續寫未完成的任務。

進來公司一兩個月了，總覺得人資很活潑，常常舉辦好多活動，我也工作得十分開心，壓力是有的，但我也漸漸地很喜歡公司。中秋節時一起煮火鍋，剛進公司就和rd同事以及pm們一起唱歌，還有猜謎活動。

當然每天中午，我們rd團隊就一起出發吃飯，一起聊天一起分享想法，很團結很溫馨，很像家人。

聽聞主管會 塔羅牌，我驚覺我們的頻率應該是很相近的，因為我會看紫微斗數，我們當時對 命理 都有一套興趣。

我最近在學占卜，主管有沒有推薦哪套塔羅牌？

- 沒有推薦，我家有十幾副塔羅牌，我已經可以再抽到的當下，還沒翻開就感應到是哪張牌卡

第一次和設計師一起聚餐吃韓式烤肉，rd和設計師們。

第五章 - 主管離職了

三個月到了，像是一個宣告期。

rd的主管開始很少出現在辦公室，我有一股很奇怪的預感，總覺得什麼快消失的感覺，我提早感受到主管要離職了呢。

他開始一對一對每個同事都聊過一次天，在他離開公司前一天傍晚安排跟我聊天。

大致上都已經遺忘了，只是我當時十分擔心我男友自己創業快沒錢。

而主管則是跟我說，他創業如果燒到沒錢了，你先是笑死他

- 然後如果是我，我會養他

聽完後，我沒什麼質疑，就結束了我們的談話。

主管給我的最後一個功課是 研究並架設LDAP。

主管離職後沒多久，我做的 北京示範天使線 專案也被停案了，頓時我沒有工作可以努力，突然變得茫然與困惑。

中秋節煮湯圓唷!!，哈哈主廚是我們技術主管唷 ~

坐在旁邊的同事 (就是我說很像台大生的那位) 是資深的前端工程師。

我對他的第二印象是一個很活在自己世界寫程式，幾乎都在聽歌封閉寫程式。煙癮很重，長相十分痞像流氓XD，但又帶著一點藝術感在身上。

我們第一次講話是主管去沖繩，好像偷偷拜託了這位資深的前端工程師照顧我，讓我如果在 天使線 專案萬一卡住了，可以跟他說。

在經歷短短四天的與他相處，他是一個很樂意教人的工程師，短短四天我也學會很多知識，也很感謝他照顧我教導我。

和我討論 enumerate 的資料科學家跟我同年齡，是台大統計系碩士畢業。常常吐槽我講的話，但真的十分理性，講出來的話語都十分具有邏輯與科學證實，和他聊天可以學到很多科學上的知識，還有如果自己講話天馬行空可是會被糾正的唷，是個很有趣的男生。

坐我前面的女生是公司的QA測試工程師，也就是一開始對我微笑，我被煞到的女生XDD!!

身材高挑，有著很少見的單眼皮，外表是天梁 坐命的經典樣貌，時常掛著微笑與爽朗的笑聲。同是月亮水瓶的喜好，常常晚上一起留到很晚聊天與工作。

我每天都在公司留到很晚，我會把今日學習到的知識與技術，整理好紀錄在我的部落格內，享受晚上剩下一個人在辦公室的時光，整理一整天所學的各種知識與經驗
