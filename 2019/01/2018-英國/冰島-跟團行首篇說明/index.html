2018 英國/冰島 團體行 目錄

到達倫敦 - 倫敦蓋威克機場/希爾頓飯店

追尋極光第一夜 - 飛往冰島/ 藍湖 Blue Lagoon

去年2018年2月就前往冰島/英國遊玩

起初只是家人不知道新年想去什麼地方，就提議去看極光

但只跟日本熟的我們，沒有日本以外的自由行經驗，加上地理, 天氣, 文化, 語言…種種原因，就找了許久未跟團的旅行社，來去冰島探險啦！

由於隨團只有附贈一家子wifi, 其實根本不夠用，常常早上剛出門流量就用完，我也無法得知我們現在究竟在什麼地方，沒有網路，也就無法一邊玩一邊撰寫我的旅記。

回國後拖拖拖到一年過去，又要去新的地方探險了，才卯起動力想要回憶跟上網查詢來紀錄去年的冰島/英國行

網路流量低，沒網路的情況下，只能夠上車睡覺，下車遊玩，也因為旅行社大部分都是走馬看花式遊玩，自然自己就像被載在貨車上動物，任由他們載往哪邊就下車踏青

雖然起出要認景點時有點困難，因為毫無地理概念(沒網路看google地圖，也不知道究竟是去了何處)

好險剛剛終於找到了相似行程的網站: 可樂旅遊.冰島/英國, 我不知道連結會不會失效，若失效，就接下來看遊記吧！

先上圖片，好險找到這張圖片，我簡直太開心了qq

＊圖片參考至可樂旅遊

我感受到我在重溫冰島旅的感動，真的很像在冰島久待qq

我突然有一種動力想把滿滿的照片砸好砸滿xD，我還在想滿滿的照片要放哪

這次的英國冰島行，我們是報名可樂旅行團，全旅程十天，可是扣掉前後兩天飛行時間剩下八天, 不過老實說，真正待在冰島好像只有4天。

應該說，整個旅程我的印象的是

桃機坐上飛機 >> 到英國的機場 >> 英國的旅館睡覺 >> 起床搭飛機去冰島 >> 冰島待四天 >> 坐飛機回倫敦 >> 倫敦待半天 >> 回旅館睡覺 >> 飛回台灣

寫一下整個旅程的時間表，雖然全都在網站上了，但怕失效，所以還是寫過來

寫完了旅程，我覺得怎麼我印象沒這麼多xD，都有去到，不過導遊應該有把一些行程稍微變動

本來由於是想憑印象寫，所以只打算寫冰島的部分，並分成四篇寫

但既然找到網站，且我拍的影片也都好好的躺在雲端，我想就找幾天完整的假日，邊回憶邊把所有的旅程都紀錄

準備好了嗎!?

準備好進入回憶了嗎(XD)
