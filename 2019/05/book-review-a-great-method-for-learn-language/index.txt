<p>在書籍尾端，秋山討論了這個話題。</p>
<p>當我從他的 Youtube 影片得知他會阿拉伯文開始，我就一直很好奇他對 <code>阿拉伯語</code> 是什麼看法，而我決定將這個我很在意的話題留到最後做討論。</p>

<p>初次看到這本書是在於 <code>秋山燿平</code> 的 Youtube 中，某次我在看 Youtube 影片時，推薦清單 show 出了他的影片</p>
<p><code>會說十國語言的東京大學語言天才告訴你語言學習的經驗和秘訣【秋山雙語時間特別期】</code></p>
<p>影片連結: <a href="https://www.youtube.com/watch?v=SK3wOaaTIN8">https://www.youtube.com/watch?v=SK3wOaaTIN8</a></p>
<p>開頭 秋山(以下稱作者) 就用好幾種外語串成的開頭來介紹他自己，特別是看到阿拉伯語時，我的眼睛為之一亮。對於想學很多語言的自己，<code>想說一口流利的外語</code> 一直都是自己最終的目標。</p>
<p>看到作者本身可以流利自由切換這好幾種語言，我開始覺得興奮，原來同時會多國語言是成立的，而非每次旁人所說 <code>一次顧好一個吧</code></p>
<p>才看完開頭介紹，我已經熱血的準備某日衝去買書</p>
<p>網路上有許多文章是關於他的 Youtube 影片以及介紹影片的內容。但看得懂日文的我直接把日文原文書買下來回來品嘗內容，並在這篇文章中分享書中的內容與自己的一些想法！</p>
<p>書名是 <code>世界一シンプルな外国語勉強法: 純ジャパの僕が, 10カ国語を話せた</code>，<a href="https://taiwan.kinokuniya.com/bw/9784478104880">紀伊國屋書店</a> 有賣，或者你可以像我一樣衝動跑去在台北, 南京復興的微風廣場的紀伊國屋書店有賣!</p>
<p><code>會說十國語言耶？那他是不是花了很多時間在學習語言上？</code></p>
<p>曾經有動過念頭想要念語言學系，甚至在考大學時填了四個志願在日文系上，卻不小心走了資管系(這又是另外一個故事了。想知道究竟怎麼樣才能夠學會這麼多語言，並流利使用。</p>
<p>在紀伊國屋書店語言區繞了一圈後，完全沒看到書本的蹤跡，直到店員從書櫃後方繞出來拿本書到我眼前時，我還不敢相信原來這本書這麼小本！</p>
<p>這樣小本的書籍裡面有滿滿的 <code>如何學好外語</code> 嗎？我腦中不斷的蹦出疑惑，看到書的當下，有種失落感，也許是一開始我意想到的是一本較厚稍大本的書籍吧！</p>
<p>但內容呢？內容有沒有因為書的大小而打折扣呢？</p>
<h1 id="內容小描述">內容小描述</h1>
<p>我想書中所要傳達大部分的內容，影片裡其實已經介紹的很清楚了(所以作者應該算自己嚴重劇透(?))</p>
<p>其實有趣的反而是其他細節，包括作者決定鑽研語言的起承轉合，包括作者對於幾個他在學習的語言做評比</p>
<p>當然還有這本書所要傳達的 <code>學習語言</code> 的中心思想，以及對於這些 <code>方法</code> 的細節描述</p>
<p>當然師傅領門在先，最重要的還是我們讀者/看的人要怎麼去改變觀念去接受並實踐這些對自己來說是新的方法來學習語言。</p>
<h1 id="作者對於語言的起承轉合">作者對於語言的起承轉合</h1>
<p>翻開這本書，覺得有趣的是一小本書內部像是筆記本般，已經透露作者希望語言學習者可以 <code>自己製作</code> 一本屬於自己的 <code>語言學習書</code>。作者將內容像是在寫筆記般撰寫在排版為筆記本的版型中。</p>
<p>開頭先是介紹自己的背景，包括雙親英語能力才 300 分 , 完全沒有任何短期滯留國外的經驗的 <code>純日本人</code> (純ジャパ) 以及沒有環境的薰陶下該如何學習外語。</p>
<p>從小就喜歡足球的作者，目標想成為足球選手。利用上學來回約四小時的時間學習足球強國的語言 <code>西班牙文</code>，通勤時間都很認真在背單字學文法持續一陣子，但是卻無法開口好好的和別人講西班牙文，這讓他十分的挫折 XDD</p>
<p>(是說，上下學四小時有夠久...雖然說我之前我來回大學+研究所 每天三小時六年也是蠻厲害的啦, 四小時每天怎麼辦到的辣 XD)</p>
<p><code>雖然感到十分挫折，但還是很努力的學習，一年後考取西班牙文初級檢定</code></p>
<p><code>雖然背了一堆文法跟單字，但卻還是沒辦法講好西班牙文，非常的挫折，在考大學時停止學語言</code></p>
<p>確實蠻痛苦的，我在準備托福時也是每天苦背 1500 托福高頻率單字，外加背熟文法，雖然<code>閱讀</code>跟<code>寫作</code>真的漸漸變好，但 <code>口說</code> 跟 <code>聽力</code> 真的是我托福中一直壓低分的項目。</p>
<p>作者大學是藥學科，畢業後覺得不太對，為了<code>追求自我</code>，他開始把 <code>多語</code>(Multilingual) 當作目標努力！</p>
<p>而這次作者選了法語當作第二個學習的語言，但有感之前 <code>學習西文</code> 的經驗，他決定用別的方法學習。</p>
<p>之後大概就是在描述這個新方法的成功讓他能夠用外文和當地人進行日常對話</p>
<p>作者的個人經歷就寫到這。</p>
<p>說說自己經歷吧~</p>
<p>想起從小英文就很差，小三還小四時的小學英文單字考還只考 <code>31分</code>，看我記憶力多好 XD！小五雖然開始補科見美語，英文有變好。然而我覺得國中的英文考試太坑爹了，一直到高中, 大學，這樣的英文考試制度與教學，完全不會讓學生 <code>會英文</code>，反而是 <code>會考英文</code>。</p>
<p>直到我國三畢業的暑假去日本，激起對日文的熱情後，開始自學日文。而自學日文的方法也就是作者書中提及的方法，雖然從以前到現在，這中間我開始慢慢對日文的動力減弱，畢竟我看到了會讓我動力減弱的原因 XD(講啥)，然後拉拉扯扯沒很認真繼續自學，但現在至少能和日本人日常會話，拿起字典還能組成語句聊聊天，看起日文書大約 70%懂，但我覺得這樣的效果已經很好了。</p>
<p>可見我們從小到大接觸到的語言教育，洗腦我們學習語言的很嚴重外，還不會增加語言能力(會增加討厭考試的動力 XD)</p>
<h1 id="那麼要怎麼學習語言呢">那麼要怎麼學習語言呢？</h1>
<p>這就是大部分讀者想了解的地方，但我覺得在<a href="https://www.youtube.com/watch?v=SK3wOaaTIN8">Youtube 的影片</a>中，作者就已經講得差不多了，所以如果看不懂日文的讀者/觀眾，多把影片看幾次應該就可以。</p>
<p>那我還是延續書中的內容分享幾點</p>
<h3 id="建立成就感方程式">建立成就感方程式</h3>
<p><code>本書的語言學習方法，不會有挫折感，如何建立並延續成就感呢？</code></p>
<p><code>記住最低限度的文法及單字</code> >> <code>使用記住的單字與文法試著說話</code> >> <code>講出話了/能順利組出話，很開心</code> >> <code>想學更多想講更多話</code></p>
<p>大概是這樣的循環建立你在語言學習中的成就感，保持你對學習外語的動力</p>
<h3 id="方法的-know-how">方法的 Know-How</h3>
<p><code>背這麼多單字真的都會使用到嗎？ 不要去記不需要背誦的單字/不會用到的單字</code></p>
<p>一開始你只需要記住 <code>最低限度的單字和文法，並慢慢的增加必要學習的單字及文法即可</code></p>
<pre><code>Part 1
學習 `最初階級` 必要使用的
200個單字 30個表現文法
(狗爬式的單字/文法) *註
</code></pre>
<p>↓</p>
<pre><code>Part 2
使用狗爬式的單字/文法
並慢慢增加必要學習的語言
(增加會/需使用到的語彙)
</code></pre>
<p>註: 作者將 最低限度使用(最少得學習這些才能組成簡單句子) 的文法與單字稱作 <code>狗爬式的單字與文法</code></p>
<p></p>
<p><code>不管是什麼外語, 3個月後都能講出該國外語</code></p>
<ul>
<li>不需要去購買市面上的單字本</li>
<li>為了日常會話，禮貌型文法不是必要學習的</li>
<li>日文去推斷(這邊帶入各國母語)，列出／選擇 最低限度能組成一句話所需學習的 單字與文法</li>
<li>學習萬人共通的 <code>狗爬式單字</code> (就是學會這些單字可以與不同人聊天)</li>
<li>試著介紹自己與自己的興趣</li>
<li>學習最基礎的 30 個文法</li>
<li>自己製作自己的單字本</li>
<li>一定要寫 <code>例句</code></li>
</ul>
<h4 id="不需要購買市面上的單字本-自己製作自己的單字本">不需要購買市面上的單字本, 自己製作自己的單字本</h4>
<p>基本上你買一本單字本，你根本不可能全部背完！</p>
<p>作者舉例 <code>高麗菜</code>，除非你是廚師或研究食譜，不然你記高麗菜做什麼？</p>
<p>這麼多單字，你根本不會在 <code>會話/聊天</code> 時全部都用到，事實上你根本不會用到這麼多的單字</p>
<p>因此，作者推薦大家買好筆記本，自己製作自己的單字本 (他本人學習是這樣學, 而我剛開始學日文也是自己紀錄單字居多)</p>
<p>製作規則大概為:</p>
<p><code>外文單字</code>, <code>中文翻譯</code>(代換自己母語), <code>外文例文</code>(可以用母語筆記), <code>注意事項</code></p>
<p>作者自己的單字本結構是:</p>
<p><code>外文單字</code>, <code>日文翻譯</code>(他是日本人), <code>外文例文</code>, <code>日文例文翻譯</code>, <code>備註/注意事項</code></p>
<p>用這樣的方法去製作自己的單字本</p>
<h4 id="狗爬式的單字與文法">狗爬式的單字與文法</h4>
<p>「犬かきの単語」，是作者在書中所使用的日文。翻了翻估狗跟文章我翻成 <code>狗爬式的單字與文法</code>，就像游泳初學者，只要學會 <code>狗爬式</code>，都能在泳池/水裡前進。</p>
<p>使用 <code>狗爬式</code> 的意義是，最低限度使用(最少得學習這些才能組成簡單句子)。換句話說，就是學習這能夠組成簡單的句子的單字與文法，而這些句子能夠初期的和 <code>萬人</code> 溝通/聊天</p>
<p>舉例的<code>狗爬式單字與文法</code> 大概為</p>
<p>| 單字  | 文法         |
| ----- | ------------ |
| name  | I am ~       |
| eat   | my name is ~ |
| think | I want to    |
| go    | I go to      |
| buy   | I buy        |
| like  | I like       |
| know  | I know that  |</p>
<p>舉例而已，有的是 <code>行為表現</code> 文法，還有 <code>時間表現</code> 與其他</p>
<p>比方說 I am Mina, come from Taiwan.</p>
<p>My interest is learning language.</p>
<p>諸如這種你和 <code>萬人</code> 都能聊天的句子</p>
<p>接著，再根據需求/需要 去擴增需要學習的單字與文法</p>
<p>想是如果用 <code>I am ..., come from ...</code> 已經不夠用了，就去擴增些新的語彙。</p>
<p>像是 <code>my interest is learning language, I learn japanese, because Japan ....</code> 或者 <code>The Avenger is my favorite movie, because ...</code></p>
<p>從剛開始只能介紹自我開始, 慢慢去擴展可使用的語句</p>
<p>恰好符合上述迴圈 <code>使用狗爬式的單字/文法</code> >> <code>並慢慢增加必要學習的語言</code></p>
<p>建立成就感</p>
<p><code>使用簡單的單字文法組成句子</code> >> <code>覺得很有成就感，並持續擴大學習，學習更多的句子/單字</code></p>
<h4 id="使用運用-會話聊天">使用/運用, 會話/聊天</h4>
<p>還有一個方式是從起床開始，你可以每句話都嘗試用你所學習的外語做表達，像是 <code>早安</code>，可以學習 <code>Good Morning!</code>， <code>我要準備去公司</code> = <code>I am preparing to go to company</code>，像是這樣融入生活的方式學習外語。</p>
<p>還有的是說可以利用洗澡時間跟平時休閒時間 <code>自己和自己對話</code> ，假裝自己有自己跟另外一個人，互相聊天這樣用外語自言自語，我之前學日文/英文也常常這樣練習。</p>
<p>你不一定要用很高超的語言技巧去說話，反之你只要能夠用最簡單最能達到辭意的文法/單字去左表達就好，講再爛再差都沒關係，畢竟最初的重點還是 <code>開口並講出話，學習組成句子</code></p>
<p>最後一個學習方法就是多聽多講，還有讓自己融入當地環境，可以試著將手機調至該語言，或者聽該語言的廣播或玩該語言的遊戲，讓語言融入你的環境。</p>
<p>像是我之前學日文會玩 pigg 是一個日本社交網頁遊戲，還有狂看日劇跟日本綜藝節目/電視，主要是有興趣啦 XD 沒興趣就不會有動力學習。</p>

<h3 id="中文">中文</h3>
<p><code>號稱全球第一難學的語言: 中文</code>，究竟在學習時，有什麼感想！(但其實日文字跟中文字還算有些小相似，不會比純拉丁語系國家的人學中文難)</p>
<p>目前是世界各國都需要會中文的人才，作者主要是學習 <code>簡體字</code>。</p>
<p>文法上比較單純，完全沒有些有的沒的規則 XDD 名詞沒有分性別，沒分單複數，覺得文法比較簡單。</p>
<p>但遇到的困難是 <code>聲調問題</code> ，像是某個字聲調上揚跟下降意思就會完全不同，對於學習中文的日本人來說這種發音的聲調問題還是比較困難棘手 (西方人也是說聲調很難 XD)</p>
<p>因為是用拼音學習，而和日文相似的是，每個中文字可以用拼音做筆記，而日文則是每個字有假名做筆記，這樣每個字每個字分割的發音比較相似，比較好做筆記，讓他覺得學習起來比較簡單順利。</p>
<h3 id="法語">法語</h3>
<p>對於日本人來說，法語算是難度較高的語言</p>
<p>像是 <code>主詞的運用不同</code>, <code>名詞分男女性別</code>, <code>冠詞</code> 亦不同，更難的是 <code>發音</code> 與 <code>聽力</code>！</p>
<p>我有朋友學法語也是說 <code>說出來的話</code> 跟 <code>書上使用的句子</code> 是不同的，沒接觸過法語沒有太多的想法 XDD</p>
<h3 id="西班牙文">西班牙文</h3>
<p>作為作者第一個入門的外語，他認為西班牙文的發音十分簡單！</p>
<p>而西班牙文最需注意的是 <code>動詞</code> 因為動字會隨著主詞變來變去，西文會把主詞省略寫動詞，而這個動詞你就能夠判斷這是在指哪個主詞</p>
<p>像是 <code>hablo</code> 代表的是 <code>我說話</code>， <code>hablas</code> 代表的是 <code>你講話</code>.. 大概是這樣。</p>
<p>我記得之前在學西文時也是被動詞搞到煩 XD，除了六-七個主詞動詞變化不同，文法的動詞變化也不同 QQQ</p>
<p>大部分都是有規則的變化！</p>
<h3 id="韓語">韓語</h3>
<p>對於日本人來說，韓語大概超級容易學習</p>
<p>不僅文法相同，助詞使用大致上相同，在日本許多文章也有完整的韓文翻譯，所以學習上很容易！</p>
<p>再來是韓語很多發音跟中文很像，所以作者也是用學中文字做筆記的方式來筆記韓文字。</p>
<p>以前就有聽說韓文字很多跟中文發音很像，還記得有幾次去聽韓文歌都會聽到些中文發音，也覺得很有趣 XD</p>
<h3 id="德文">德文</h3>
<p>德文在歐洲是使用人口做多的語言</p>
<p>超級複雜的名詞使用，十分混亂 XDDD</p>
<p>名詞除了男性女性還有 <code>中性</code>，最重要的是還有 <code>變格</code></p>
<p>德文文法和英文跟日文都不同，初期學習覺得<code>語順</code>很困難，加上複雜的名詞變化，使得要日常會話變得十分困難</p>
<p>前陣子想去德國留學，有初期接觸德文，當時有個圖在諷刺德語的名詞又臭又長，連推特的文字限制讓德語使用人士很無奈 XD</p>
<p>而德文的名詞常常讓我不知道該怎麼發音，舉個例子 <code>setzt</code> 知道這怎麼發音嗎 XD 我也不知道</p>
<p>更多的是又臭又長的單字 <code> kontinentalwestgermanischen</code>, <code>Deutschschweiz</code>, 覺得可怕，我是從 wiki 上隨便抄下來的，但確實初級學習時還不會接觸到很長的單字！</p>
<p>不過光是背 <code>Die</code>, <code>Das</code>, <code>Der</code> 就翻肚了，雖然我也蠻喜歡德語發音的，但我現在暫時先擺一邊了 XD</p>
<h1 id="對於阿拉伯文的想法">對於阿拉伯文的想法</h1>
<p><code>史上最大難關: 阿拉伯文</code></p>
<p>曾想學習阿拉伯文的自己，也做過許多科普，難是難不倒我的，依據我的個性一定會克服過去。</p>
<p>來看看作者怎麼說吧！</p>
<p>語彙系統和日文, 英文及其他語系完全不同(畢竟是閃米語系(?))，文字書寫系統很困難，雖然阿文只有 28 個字母，但是只要<code>點</code>錯, <code>形狀</code>錯了, <code>位置</code> 不同都會影響意思。</p>
<p>看過阿拉伯文嗎，<code>اللغة الأَلمانيَّة</code>, 你能想像把它分割成 28 個字母嗎 XD 能想像他也是一個一個字母嗎，只是現在連起來了，而且我最近翻了我買的阿拉伯文，字母書寫在句首, 句中, 句尾都不同!!! 連 <code>連起來</code> 都會變化 QQ !</p>
<p>名詞單複數不同, 有時候還會到三個變化以上，分男女場合不同，且格位跟主詞都得看動詞的前後判斷。</p>
<pre><code>最強にコスパの悪い言語：アラビア語
CP值最低的語言: 阿拉伯語
</code></pre>
<p>最後，我們大家在書上學的都是 <code>標準阿拉伯文</code> 是專門書寫, 可蘭經跟清真寺內才會用的文字，不同國家口說有不同國家的<code>方言</code>。</p>
<p>沒錯這就是最困難也很討厭的，就是你克服了一切難關學習阿拉伯文只能在書上文章使用，但是若是要做溝通跟口說，不同國家有他們自己的方言，意思是你拿埃及的阿拉伯文到摩洛哥聊天是不會通的(?)，這我這證實，若有朋友有感想也歡迎聯絡及留言。</p>
<p>但這也是打退我想學阿拉伯文的動力！老實說我一開始也只是想說看得懂 <code>可蘭經</code> 就好，畢竟對宗教有些許興趣想去研究。當然，能夠 <code>順便</code> 學會在該地區能夠溝通是最好的了 xDD</p>
<p>這麼看來學了只能運用在書寫上，還不能夠到處講，好像...就有點學習動力流失啊！！</p>
<p>不過會阿拉伯文的人，我還是抱以崇拜的心看待之！！</p>
