[2019 北海道 札幌/道東] 行程說明與DAY 1 札幌/大通/王子飯店

February 10, 2019 in Travel Diary, 日本－北海道

我還在思考，我要不要趁現在發文章，因為折騰了整個早上五點起床到現在晚上七點才順利搭上前往網走的鄂霍次克二後列車，等了整天。 以往都是趁坐車寫文章，回去台灣也未必有時間靜下心來把這次的自由行寫完，因為真的蠻刺激（?）的。

按照慣例將本次自由行資訊交代一下，接著就進第一天行程：

本來還在猶豫這個春節要去什麼地方，既然我爸已經決定了要去美國了，就剩下我跟我媽，於是這次我媽就提議來完成去年原本要計畫去北海道看流冰，最後卻因為我爸覺得大雪一定誤點，而後來去冰島遊玩！

去年規劃時其實有想說拉長去的時間，並跑去 北海道最北點 稚內 朝聖。但無奈無論是計畫5-7天，要環整個北海道還是完全不夠的，若是有朋友有計畫環整個北海道，我認為9-10天，加上可能需要拉一點車才能足夠去達成環北海道的行程。

因此這次沒有去稚內最北點，但我跟我媽都覺得既然第二次自己去北海道，就不要再跟上次行程一樣，因此這次，我完全建議可以採取大拉車的方式，從札幌出發到 網走~釧路~帶廣，最後回千歲，完成算是從中間切入環島的行程，那我們就來看一下這次行程！

本次行程全程由我媽訂定 — Patliao, YHEH-ER 參與旅行社的兩日一夜自由行計劃, 購買機票，贈送的一夜，就決定住在札幌 

本次行程，依序是千歳 → 札幌 → 網走 → 釧路 → 帯広 → 千歳
但中間因大雪特報使行程改變，不過目前已順利搭上前往網走的列車，前進網走的詳細方法可參考第三篇：

[2019 北海道 札幌/道東] DAY 3 (尚未連結)

 

目前行程更改為 

千歳 → 札幌 → 旭川 → 網走 → 釧路 → 帯広 → 千歳

台灣時間早上五點就起床，準備前往桃園機場check in 來去北海道玩拉！ 通常出國的前後兩天行程都不太知道怎麼安排，而這次我們搭乘約9點半的飛機到達北海道千歲機場約莫13點，大約飛三個小時而已（我怎麼記得之前要飛四個小時（?））

搭乘長榮航空 EVA AIR 前往北海道千歲機場

相比歐美，日本的飛行時間真的是很短了，而在飛機上打發時間，我做了 1. 剛起飛睡覺 2. 玩俄羅斯方塊 破其他人紀錄 3. 觀看 隔離島 電影 4. 吃飛機提供的餐點

 

每次飛行時最喜歡玩俄羅斯方塊跟踩地雷來破人家紀錄，然後直接在排行榜上打上 Mina Yu Taiwan 哈哈

隔離島也讓我在日本遊玩的第一天陷入電影的沉思，還有想要研究二戰歷史的慾望，我決定回程再看一次！

長榮飛日本班機除了機票是三麗鷗, 腰枕, 餐具也都是喔！, 有興趣記得把餐具暗摃回家XD

到達北海道後，時間還很早，這天完全沒有安排什麼景點（有安排大通公園的冰雕展）

一下飛機迫不及待想去玩雪，但拜日本的運輸系統所賜，我們出關後，直接拿著之前買的suica 刷JR進去後就搭乘了 快速エアポート 列車從機場到札幌

到了札幌後，第一件事情就是要把北海道鐵路周遊卷 5天份 買好，並把後續幾天的長程車程給預定位置（買了周遊卷, 預定列車指定席不用額外花錢，更改指定席也不用有額外花費）

＊要注意一下地下鐵 (Metro) 還是要自己買票/ 使用suica, icoca卡.. 等IC卡通過

 

外國人購買北海道鐵路周遊卷的地點不在綠窗口（みどりまど），而是旁邊專門設給外國人的information 遊客中心，有會英文的服務員和您溝通(幸運的話會遇到講中文的)

當時在內心整個吶喊，在台灣跟法國同事天天練英文，日文早已退步許多，剛開始還在英文模式，總是能說出一口流利的英文偶爾夾雜日文（?） 也許是觀光客很多，所以大部分服務人員都會說一些英文，或想辦法讓你知道他在說什麼！所以漸漸地簡單的英文能溝通囉！但當然有時候必要還是要學習日文（因為我很多時候也會用到日文溝通）

不過因為我常常用英文很不正經的跟同事開玩笑，所以我不會放過任何一個可以講英文的時機，並且輕鬆的多和對方聊聊，這是我在使用日文時比較無法達到的境界（?）

晚上的旅館是 王子飯店，一咕狗發現居然要走半個小時，只好硬著頭皮在2度溫度下走路到旅館。

札幌JR車站旁邊有之前在台灣鬧的沸沸揚揚的 紀伊國屋書店，第二天有去，整整兩層都是書，超級龐大！！ 我還買了書呢！

大通公園隔日2/4開始有雪慶典，現在都能看到冰雕作品，可以在這看看夜晚的冰雕展覽，跟白天絕對不同，但白天也很有一翻風味，第三天早上有前往至大通公園看雪慶典，下下篇日記在詳細介紹，但最後走了一個小時才到王子飯店。

看到王子飯店時整個內心大撒花，必應在2度的天氣下走了一個小時嗚嗚

 

千萬不要學我們

我準備詳細如何前往王子飯店的教學！

【五分鐘交通攻略！】搞懂札幌市營地下鐵，怎麼搭最優惠！

欲從札幌到王子飯店的朋友，參考此地圖

從札幌JR站走出後，麻煩跟著指示前往地下鐵

搭乘『南北線』前往真駒內方向，搭乘一站在大通下車，這一站包含大通公園，狸小路這站附近

轉車至『東西線』前往宮の沢方向，搭乘一站至西11丁目下車

出地鐵後有一條大馬路就可以看到王子飯店

其實我發現我也有拍地圖，實際的地圖與地鐵可以參考此圖

在旅店check in 稍作休息後，前往札幌晃晃，一樣依照上面的路線（這次學乖了），搭乘東西線地下鐵到大通站轉南北線 麻生/さっぽろ方向搭一站到札幌站後

我們從西口綠窗戶旁邊的咖啡館下樓梯到達一個地下室都是餐廳，晃了幾圈後，我媽提起要吃湯咖喱，於是找到了在大通站狸小路附近有3-4間湯咖喱，然後就出發會大通站吃晚餐

路上的美景

原本是想去 湯咖喱 GARAKU 結果發現大排長龍後，走附近的 Kanako湯咖喱用餐，剛進店就發現服務生很悠閒緩慢的依照他的做事順序做事，所以代位點餐都較緩慢，感覺來到一個步調比較慢的餐廳

我點的是 店家推薦綠咖喱 （有香菜！）加上 蝦湯（エビスープ） 15分辣，超級好吃

沒吃辣的人 15分 算是挺辣會需要一直喝水，但若對於有吃辣的朋友們15分剛剛好，也許能再往上點

位置真的蠻少的，整間店座位約30位，但整體還算不錯，很推薦！

吃飽後也晚上八點多了，就順著道路直直走一大段路，會抵達王子飯店。

若對於旅程中需要早起或者較睡不著身體欠佳的朋友們，我每次去日本真的覺得身體狀況差都會去便利商店買補給飲料來喝，原則上會有一整個櫃子是賣補給飲料，我是早上起床喝BB 會很有精神，晚上睡覺前買C來喝！對於出來玩又因為時差或時程上/身體疲勞都建議可以買來喝！

那麼今天的行程就到這拉！雖然僅僅只是第一天行程，但也感覺很充實呢！

晚上睡王子飯店的房間
