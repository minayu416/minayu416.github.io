9:T2712,<p>最近幾次的旅行多半是去日本的鄉下: 像是四國及北海道，還有歐洲與美國，通車時間起碼都要 1 小時起跳，所以安排行程上會變得十分寬鬆，因為得因應交通所花費的時間以及各種突發事情。</p>
<p>很久沒去像是東京這樣擁擠方便的城市了，由於公司的同事在尾牙抽到前往東京的機票，但又沒出國過日本，小小的一個單人女子要怎麼面對陌生的國家，於是會日文又有好幾次日本自由行經驗的我，便提議一同前往東京遊玩</p>
<p>本次的日本東京/輕井澤自由行時間為五天, 以下就來針對本次行程做介紹與說明囉！</p>
<h1 id="廉價航空-vs-一般航空">廉價航空 vs 一般航空</h1>
<p>這次的行程是小妹妹我第一次嘗試廉價航空 XD，由於以往跟家人出門，我爸十分鄙視廉價航空，鄙視到提到就會用很諷刺的態度噗哧一笑，但由於這次的機票旅館錢是我，小資窮女為了省錢加上想要搭乘紅眼班機(半夜起飛)，就選擇了廉價航空。</p>
<p>提到廉價航空，許多沒搭乘過廉航的朋友們大改第一印象就是: <code>機票這麼便宜會不會很不安全？會不會墜落？</code></p>
<p>剛開始出發前我也是很擔心，難道我的人生就要？ (呸呸別亂說)，但實際體驗後會發現其實根本沒想像中可怕，我來簡單做一個表格比對提供給大家看</p>
<p>若說一般航空什麼都有，那廉航比對可說是 <code>什麼都沒有</code></p>
<p>| 航空公司類型 |                     廉價航空                     |                       一般航空 |
| ------------ | :----------------------------------------------: | -----------------------------: |
| 航空公司     |                   捷星 Jetstar                   |  長榮 Eva /華航 China Airlines |
| 價位         |               低(本次行程 2500 元)               |        較高(本次行程 10000 元) |
| 飛行時間     |                 通常是半夜或早晨                 |     正常時間, 比較沒有半夜航班 |
| 行李托運     |                      需加價                      |                 每人一件或兩件 |
| 餐點         |                      需加價                      |  會有飛機餐, 可以選不要牛肉/素 |
| 選位         |              Jetstar 要加價才可選位              |                         可選位 |
| Check-In     |                 沒啥感覺都差不多                 |               沒啥感覺都差不多 |
| 機上服務     | 送餐給加購的乘客, 入境卡在一進飛機時拿, 不會走動 |     送餐, 發入境卡, 按鈴可呼叫 |
| 機上設備     |       無螢幕機器, 不知即時地點, 無毛毯枕頭       | 有螢幕機器, 即時地點, 毛毯枕頭 |
| 位置舒適度   | 位置蠻大, 身高 167 坐著, 膝蓋與前座隔一個手掌大  |                       位置普通 |</p>
<p>大致比對了本次體驗的各項細節，所以廉價航空的部分大概就是比一般航空少了服務與吃喝玩樂，單純就是有個位子跟上方的櫃子，你可以安身立命(?)飛往目標地區。</p>
<p>至於下次會不會體驗廉價航空，答案是我認為 <code>短期飛行</code> 就能到的國家，只要帶個頸枕就能出發，特別是半夜及一早的飛機，能夠偷到第一天整整一天的時間旅行，而且通常也不大需要 <code>吃飯</code>。</p>
<p>但通常半夜出發，若沒在機上好好睡的話，第一天也是累翻，像我第一天中午的時候去逛晴空塔，感覺就是靈魂被抽掉的跟著同事走而已 XD，因為忘記戴頸枕機上座位頭的地方不好睡。</p>
<h1 id="票價與交通">票價與交通</h1>
<p>本次東京/輕井澤五日的交通費約莫為日幣 17000 左右, 大約台幣 6000 多，其實交通蠻貴的，因為我們要去<code>清井澤</code> ，光是從 <code>上野</code>坐新幹線 去 <code>輕井澤</code> 的單程票價就要 5180, 因此本次旅行的交通決定購買<a href="https://www.jreast.co.jp/tc/tokyowidepass/">JR 東京廣域周遊券</a>，但是只有三天 qq，所以只好用在來回輕井澤的那兩天</p>
<p>交通費加起來大概是:</p>
<ul>
<li>
<p><a href="https://www.jreast.co.jp/tc/tokyowidepass/">JR 東京廣域周遊券</a> 10000 日幣</p>
</li>
<li>
<p>來回成田的<a href="https://www.keisei.co.jp/keisei/tetudou/skyliner/tc/skyliner/index.php">京成電鐵</a> 5000 日幣左右</p>
</li>
<li>
<p><a href="https://www.jreast.co.jp/tc/pass/suica.html">suika 卡</a> 東京地鐵 Metro/ 公車/ 輕井澤私鐵 (suika 卡) 約 2500 日幣左右</p>
</li>
</ul>
<p>大約 17000 日幣付清我們的交通，JR 東京廣域周遊券在東京可以通行 JR 線 但不通行地鐵與其他列車，所以要搭乘其他列車就還是得乖乖拿出 suika 卡</p>
<p>購買 <code>JR 東京廣域周遊券</code> 的話，請大家可以看清楚一下官網上的<a href="https://www.jreast.co.jp/tc/tokyowidepass/index.html?src=gnavi">地圖</a>，乘坐新幹線很貴!，所以若會大量使用到 JR 列車站或者你要去的地方有列在圖上(且需搭乘新幹線)，那就可以考慮買<code>JR 東京廣域周遊券</code>!!</p>

<p>實際上, 我當初把我們很馬呼的規劃傳給我爸的時候，連他老爹都嫌怎麼行程這麼少</p>
