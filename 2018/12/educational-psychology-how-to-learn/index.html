[教育心理學] 如何學習

December 8, 2018 minayu416

1 minute read

  • Twitter

  • Google+

  • Facebook

  • Reddit

  • LinkedIn

  • StumbleUpon

  • Pinterest

  • Email

拖了差不多一週的稿

起源於上周會議上，前輩基於公司有幾個較資淺的工程師，所以分享了如何學習的技巧

我覺得我很需要，也覺得原來學習與心理學是這麼回事，所以我想分享出來

從小學到大學，我們的教育普遍是

老師教書 >> 學生聽 >> 背起來 >> 要考試

到了研究所的時候，我學資訊管理要寫程式，我的學習方式變成了

自己查做法 >> 自己了解 >> 實作

很明顯的研究所前，學校只教我們背起來，沒有教我們如何運用，沒有去思考為什麼

而「考試選才」的制度，造成教育為了因應考試，因應家長希望學生能獲得「高分」，才能進好學校，許多老師開始開發考試的技巧，教導學生們套公式或如何背誦才能速成獲得高分。

考試考不好不代表學習能力不好，只代表考試能力不好 XDD

這樣的教導方式沒有去將知識實作，亦沒有培養思考

在離開學校後的路途後，會有一段陣痛期，才會開始會去做思考，慢慢去學習實作跟導正觀念

接下來分享前輩所介紹的「如何學習」

學習最重要的**要素**是**動機** 學習最重要的**技術**是**後設認知**

後設認知的意思，根據Wiki, 「就是對自己的認知過程(記憶, 感知, 計算, 聯想)的思考」

在Vygotsky for Educators這本書則是解釋為

to plan and monitor our behavior and to evaluate its outcomes.

再來看一下這張圖

這是截至這個網站的圖

左邊那排 The Knowledge 為知識維度

知識的類型為

事實，概念，程序，後設認知

事實就是具體的人事物，舉凡人名, 地名, 日期都算

概念就是原理，原則

程序就是技巧，技術，策略，怎麼完成一件事

最後，後設認知，結合前三者知識，並不斷的調整到符合期望成果

每一個步驟都是後者需要前者

「後設認知」 是可以訓練的

前輩訓練我的方式就是，像灌籃高手櫻木被訓練般

錄影，回顧，自述

透過將自己的學習錄影，回顧並作檢查與調整，表達調整後的學習，下次會在調整的地方停住去思考這邊錯了要改正

不過我想要講的不是上面的XD

根據最上面的

可以看到在右邊那排 The Cognitive Process Dimension , 也就是所謂的認知過程

認知過程從最右下開始數

背起來 >> 了解  >> 運用 >> 分析 >> 評估 >> 創造

上半部提及學校的教育大部分只停在第一個階段，順利的話達第二階段

而社會卻需要達到第三階段以後，也就是學校沒有教如何運用，結果入社會馬上就要運用

能達到「創造」想必是前面的階段都很熟悉，能達到的人較少

學習不光只是一昧的背起來，還需要了解得知原理運作，透過大量的練習「運用」來熟悉知識與原理，再持續下去

也許第一步是「記得」，不過後續的原理及練習運用可以幫助“記得”知識，不要一直停在第一層，持續的一步步走下去，堅持並搭配正確的學習就能運用自如並達到創造新道理/新知識的階段囉！

learning how to learn

把自己當家教/老師，將自己所理解所學習的知識反覆講／教導給別人聽

透過後設認知，將概念/ 程序結合不斷的去練習

從現在的自己去檢視過去的自己

最後分享前輩提出的小黃鴨偵錯法

在軟體工程內有一個小黃鴨調試法

當程式出錯或思想混亂時，擺一隻小黃鴨在電腦旁

對著小黃鴨解釋自己的目的/ 過程 …總總，並且從原本自己的認知來去反覆檢視一次

往往就能找出錯誤跟新的解法

Think out loudly

說出來, 自己聽, 同伴聽

將內⼼的思考具體化

這避免「後設認知」難以習得的缺點

比對想法與實作是否⼀致

程式是照著自己寫的跑，不是照想像的跑

中間有提到一個模糊焦點，專注焦點(我不知道這個名詞對不對)，意思是說當你很專心在程式碼的時候，你沒辦法解出這個Bug

但當你沒有專注去做別的事，模糊了思想後，往往Bug就解出來了

我不知道這個知識什麼，不過前輩有分享就是，期望之後能找到

2018.12.26 [實例]

我最近意識到原來生活上有好多都跟後設認知有關的例子，經過前輩的訓練後，我將訓練後設認知的方法開始廣大運用在我的每個學習

像是有的時候，當我把所有的事情跟媽媽或者我的朋友再訴說一次時，這個過程中我會再一次將事情回想，整理並說出來，往往許多時候，就會突然有新的idea或者新的解決方式產出

亦或者是最近再工作上有些問題，像是溝通與情緒的問題，當事情與情緒混砸在一起的時候，我們當下會覺得情緒實在是爆爛。但我無法知道究竟是為什麼爆爛。

由於我先前有點焦慮的傾向，我不想靠西藥跟中藥去治癒，我希望能夠自己解決，所以每當事情一發生，我當下會很有壓力很焦慮，然後出現了一些焦慮的身心狀況。開始我慢慢地去分析會造成焦慮反應的所有情緒與因果，有機會再開一篇分享，經過多次不斷的分析練習後，我才發現每一次的焦慮反應是至少有著2種以上的情緒產生，有可能是一件事情牽動著兩種情緒，隨著腦中的想法增多情緒會越多來越多，直到最後由一個焦慮的情況來概括所有的大雜燴

而最近在工作上發生了很多事情，讓我的心情很鬱悶，昨日我不斷地跳針跟我男友包括跟自己對談

我就是覺得很生氣，很不爽，但我真的無法指出哪點不爽，重點是哪個

我就發現，這一切都怪怪的，也許是先前我對抗焦慮時學習到的概念，我開始去一步一步釐清究竟是哪點造就我的不爽，我才發現事情真的沒有想像中這麼的單純，而我也一步一步慢慢的把事情跟混雜的情緒做一個清單，希望能夠透過這些清單中找出問題所在，並尋找解決方式，然後下次避免這個情況發生
