垃圾情緒

April 25, 2018 in Life Healing

你有過一種經驗，是當你面對一個人，然後你開始滔滔不絕的抒發一大串悲觀，困難，障礙，疲勞，抱怨。

而聽你抱怨的人，總是回答你：「想太多了啦，不要糾結了，你真的想太多了」

再開始之前我們先貼一篇網路上的文章

https://www.google.com.tw/amp/s/kknews.cc/essay/nn4b6g.amp

我有時候也會做這樣的事情

對象是我一個好朋友Tina跟我媽

但是，當我在抒發這一長串長恨歌的時候，我會希望我是帶有灰階笑話的去描述這樣的抱怨

抱怨本身很沈重，或許還會影響對方的心情

但我想如果我用比較戲劇化甚至誇張化的肢體與聲調來豐富化我的抱怨，聽者除了會跟我一起體驗這北七又好笑的抱怨外，也不大會把它當作是垃圾反而是一個有趣的小故事

上面是一段小小的題外話

今年初開始發生了一長串事情

讓我不得不開始著手研究這些狀況

而別人帶給自己的垃圾情緒不知不覺也深深影響著自己

讓我產生了大約一段時間的「小劇場」「不安全感」然後連續跟男友吵了幾天，現在想起來真的覺得這樣很傷感情，尤其吵的一次比較嚴重的，差點讓我覺得我快失去他，因此我好好的檢討自己，並讓自己不要再重蹈覆轍。

當然我還沒踏入所謂的婚姻，所以針對婚姻的關係，我不好發表什麼

也經由這年初發生的事情，讓我深深的相信，人與人之間的相處跟溝通是很神奇的

尤其是情侶或者朋友，只要一方有改變，也會帶動另外一邊的改動

因為交男朋友而帶出了價值觀的打架期

後來我得出來的一個結論

當一個強勢的女人，不如當一個有智慧的女人

一直以來，我深受家裡的狀況及價值觀影響

有個價值觀一直深深的烙印在我的內心跟腦海

就是「女人一定要強勢，一定要當一個工作很強的女人，經濟一定要獨立，精神一定要自由，不能全部依靠男人，不能辭掉工作全部都在家給男人養，男人實在不靠譜，會外遇」

我被灌輸這樣的想法，直到我現在交了一個男朋友，我開始產生了價值觀的打架

為什麼？ 因為家裡的狀況，始終讓我覺得男人不靠舖。談了戀愛的結局就是悲劇，不會有喜劇。

講到這裡，我猜很多人都知道家裡是怎樣的狀況。

最近，看到最真實的例子是我的一個朋友(他也是讓我想發這篇文的始作雍者)

他是目前沒工作也沒有生小孩待在家裡養貓，每天都在玩線上遊戲的一個女生

昨天稍微跟她聊了一下，發現其實他還蠻悲憤的，尤其當他提及他小時候成長的那個環境，被媽媽精神虐待＋高壓控制的環境。

不敢叛逆跟反抗的她，大概花了22-23歲的時間都是被擺佈的，以至於沒有摸索自我，不曉得自己是誰，也沒有摸索自己的目標跟夢想。

(這我還蠻慶幸的，也許是自己有那個叛逆的個性，所以會受不了主動去爭取自己想要的)

現在他結婚了大約有兩三年，我聽了一下覺得不大對勁。原來當初她老公追她，她用戒菸作為一個門檻，戒掉才在一起，結果去年他老公開始鬼鬼祟祟的躲陽台講電話，被他發現因為工作壓力大而開始抽菸。

他跟我形容發現的當下

就像是被我抓到他外遇一樣，他之前承諾好的，答應我戒掉抽菸，結果現在我卻發現他在抽菸，當下我真的覺得什麼承諾跟信任全毀掉了，就像被背叛一樣。

她邊講，我看到她的眼裡泛著淚光，但又強忍著。

但看到眼前的她，我真的想起一個月前的我媽。

這樣的歇斯底里，眼神透露出悲傷及恐懼。

什麼時候女人才能夠從這樣不安，恐懼，鑽牛角尖甚至是歇斯底里的心裏被解放出來呢？

這樣我覺得活得很辛苦呢，每天都要擔憂，每天都要不安，每天都要懸著一條線，然後隨著累積的負面能量爆發後，開始抓狂跟歇斯底里。

為什麼明明一個人的時候，我們可以過得很開心，充滿自信，每天都充滿活力。

為什麼多了一個男人後，女人會變得這麼的瘋癲(抱歉，我不是故意要用這字眼)？

有人想過這個問題嗎？

不過其實是他倒了一堆垃圾情緒給我，有影響到我，讓我回家後心情不大好

在這我想打住一下

其實愛情就是兩個不同背景不同個性的人生在一起陪伴一起努力建構未來，一起渡日子互相陪伴。

也就是說，不是1+1=1, 而是1+1=2

看到這1個人+1個人是2個人

所以應該是雙方都各自有各自的生活才對

所以應該不會是一方犧牲掉自己的人生去全力依靠另外一半

也許有的人會說，女生結婚了就是要辭職在家帶小孩煮飯做家事

可是，當其中一方犧牲所有的人生只依靠另外一個，另外一個人得活出兩個人的人生

或者，其中一方沒有自己的人生，他的注意力將會全心的放在另外一個身上

於是產生了控制，產生了胡思亂想。

雖然我自己是女生，但我會現在真的覺得，讓一個女人待在家是很恐怖的一件事XDDD

為了你的自由，為了你的耳根子，為了你的精神與身體著想

你還是讓女生有自己的生活比較好XDDD

舉一下自己的例子，我有陣子不太想待在家，覺得自己只要待在家就會一直受爸媽的關切，關心及碎碎念，還有時不時被爸爸掌握跟控制自己的未來

爸媽控制你的未來，有時候是想安排一段安安穩穩沒有危險自以為覺得好的道路給自己的小孩。曾經我利用半年的時間來試圖做到符合爸媽認為對你好的道路的標準，大概是(2016⁄9-2017⁄2)

後來我發現，這樣的生活真的非常非常的無聊，就是乖寶寶狠安穩，似乎沒什麼挑戰性，沒什麼變化性。

於是，有段時間，將近快一年(2017⁄2~2018)，我都常常窩在學校和學校的實驗室，而且我巴不得家當帶過去直接睡在實驗室，在這裡我就不受到爸媽的干擾，為了有一個自己的空間可以自由做自己想做的事，可以任意發揮自己想發揮的，可以照著自己的想法來去挑戰人生，所以開始找教授訓練自己的英文作文，開始回去找指導教授接案操自己，開始去挑戰外系的課程，並在交通時間充份的學習英文。

雖然時常挑戰人生把自己逼到牆角很痛苦，但卻也學到不少，而且進步神速，經歷了不少。

讓我們想一下

小時候，我們都有被爸媽管過的經驗對吧？

也有被爸媽碎碎念跟控制的經驗

但你喜歡這些經驗嗎？

我最近發現，婚姻好像真的會複製自己爸媽的狀況。

但就因為如此，我才一直在思考怎麼樣才能夠改善甚至是改變未來，不要重蹈覆轍。

所以，我們將小時候被管想逃想自由的狀況挪用過來

爸媽擔心小朋友的未來 >> 對小孩碎碎念 >> 怕小孩受傷 >> 偷看小孩的東西 >> 掌控小孩 小孩發現自己沒隱私 >> 壓力很大 >> 想逃 >> 跑到外面 >> 待在哪都好

套用過來

女生擔心抽煙喝酒對男生健康不好 >> 開始碎碎念 >> 女生擔心他在外面有別的女人/ 不安全感 >> 想探男人的隱私/查手機 >> 想掌控男人 男人方面>> 發現自己沒隱私什麼都被查 >> 你不相信我 >> 想逃>>被壓得喘不過去>>外面的世界好>>往外跑吧

其實我不敢說能不能這樣套用XD

但我有時候聽到女生朋友歇斯底里的樣子，我難免會覺得，有這麼可怕嗎？

而我們會發現

許多人活在過去，擔心著未來

也許

你沒有意識到你沒有活在當下

我們總是用過去的經歷綑綁自己，然後用悲觀塗抹著未來

其實就連我自己有時候也是會這樣沒安全感，不安甚至是亂亂想。

所以，當我在亂想的時候，我就來

  1. 寫程式

  2. 發部落格

要知道這兩件事所耗的時間都蠻多的

尤其第一項跟電腦有關的，我自己算半個軟體工程師，當自己全神貫注在寫程式的時候

那個專注那種六親不認的狀態，是蠻可怕的

常常代碼敲著敲著，怎麼就…晚上了XD

常常邏輯想著想著，阿…就忘了今天有課XDDD (好啦這是今天發生的事情，好險被我媽提及的時候還是早上，我是下午的課)

尤其是在趕專案的時候，真的很想躲起來狂敲

時間沒有個4小時完全不想動工

不要當強勢的女人、當有智慧的女人

不管如何

我們沒辦法回到過去改變過去的事

也沒辦法到未來先去看未來的事情

我們能做的就是過好當下，遇到困難跟障礙就想辦法去解決

就像姐數學題目一樣，有千百萬種解法

以女性的角度來看

常常看文章會知道

男人本來是什麼樣子就是那個樣子

別指望會改變他

本來他是怎樣的人，他就會是怎樣的人，結婚後可能還會更劣化

不然當初愛他的時候，他就是這樣，為什麼後來要硬是去改變他呢

所以，這次我就覺得，不要硬是改變對方

而且我也不知道當初喜歡/愛上的時候對方就是那樣，為什麼在一起之後反而要對方改善什麼呢？

而我自己抱持的想法是

不但不大可能改善，還可能惡化

但只要知道這點，心就放寬許多

人生是他的，你不能操控他怎麼活

可是你可以操控你自己怎麼活

別人的話聽聽就好，不需要拿來當作寶給自己小劇場混淆自己，自己怎麼想比較重要。

當然，有一種說法是，當你把自己過得更好提升自己的時候，你的另外一半其實也會慢慢的受你的影響去提升他自己唷！

好好的經營跟過好自己的人生，然後和另外一半好好一起經營/度過人生的每一刻吧
