s = Record.objects.filter(student_id = request.POST['student_id'])
s.update(check='TRUE')

大概就是這樣，前面的表單要用hidden藏好student_id，接著回傳後撈該生的資料，來做修改他其他資料欄位的值

if 'file' in request.FILES and request.FILES['file']:

如果有收到檔案，那麼就在那位學生的check欄位打個勾(Boolean)
