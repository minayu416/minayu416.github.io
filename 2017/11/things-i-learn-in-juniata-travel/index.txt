9:T30df,<p>醞釀了許久</p>
<p>重複思考該怎麼寫這篇文章</p>
<p>終於有一個好的時間能夠坐下來細細的撰寫這次出國的啟發</p>
<p>這篇文章，我從旅行初就開始構想</p>
<p>我在這趟旅程能夠獲得什麼？我該怎麼撰寫這篇文章</p>
<p>簡直耗盡腦力，因為要有邏輯的組織好我想表達的事務</p>
<p>有太多東西想寫了</p>
<hr>
<p>有別於以前都是去日本</p>
<p>這次是我長大後大一次去美國(小學五年級有去過，不過沒印象)</p>
<p>長大過程中，由於我爸太崇拜美國了，引起我一波又一波的反感</p>
<hr>
<p>在這趟旅程前，我已經失去對未來的希望以及對生活的熱情</p>
<p>總覺得我過的每一天都像機器人般，隨著上天或者命運安排的路程走過</p>
<p>會覺得自己不管做在大的夢，最終都還是得回到現實</p>
<p>被我指導老師給貶入地獄</p>
<p>在這之前我有寫過一篇文章</p>
<p><a href="https://minayu.site/2017/10/my-treasure/">My Treasure</a></p>
<p>我將他訂在首頁，因為我覺得我已經失去對生活的熱忱跟希望</p>
<p>這並不是憂鬱，而是想做什麼卻無法做什麼的無奈感</p>
<p>這篇文章是我在極度失去自由，必須得應付大部分都被綁住的時間，渴望自由渴望自我決定的一篇文章。</p>
<p>在留學夢消失之後，我時常面臨這樣失去熱忱跟失去夢想的窘境</p>
<p>沒有夢想的支持，我就會對人生沒有熱忱</p>
<p>但夢想又很怕不會實現，一整個猶豫拉扯哈哈</p>
<p>所有新奇的事情都會吸引我，但隨著經驗跟心智增長，我會慢慢知道我要的是什麼</p>
<hr>
<p>如果有從我 Juniata 旅行文的第一篇開始看</p>
<p>當然如果沒看也沒差</p>
<p>不過可以看到我從第一天開始就有很多很多的啟發</p>
<p>隨著每天旅行遊記的描述</p>
<p>可以看到我整個人糾結的過程</p>
<p>每次覺得遇到很多新的事物，我就會多出一堆想法</p>
<p>很容易就被外面的環境干擾與影響</p>
<p>開始就會定下一堆看似毫無頭緒的目標跟夢想</p>
<p>可是後來又會因為與現實不符</p>
<p>最後還是會回去原本的道路</p>
<p>在美國可以做很大很大的夢</p>
<p>可是我一回台灣就馬上被環境跟氣氛感染</p>
<p>像是被衛福部案子綁死，面對這個案子的各種憤怒與無奈</p>
<p>像是被教授綁死，大部分的時間，隨時 On call</p>
<p>感覺我不管遇到多麼不錯或者多麼好的知識跟經驗</p>
<p>我依舊還是只能跟著命運安排的走</p>
<p>我依舊看不太到我的未來</p>
<p>因為我沒有時間去做夢，只能忙於現實，目前被綁住的時間</p>
<p>我指導教授常常跟我說</p>
<p>或者</p>
<p>的確</p>
<p>這就像我之前學網頁一樣，後來覺得應我應該要去學人工智慧</p>
<p>但這就不會是我該走的路，我只是不小心被新的事物給吸引到，暫時離開了視線</p>
<p>而這次面對新的方向物聯網，不可能在一朝一夕就能學會</p>
<p>不可能將現在的東西完全放下，再重新學習一個完全全新的東西</p>
<p>現在的時間很少很少，要忙著做當下的事情</p>
<p>在這趟旅程中，我常常突發奇想</p>
<p>覺得英文很重要，覺得西文很重要，覺得日文也很重要，還想要再學一個語言</p>
<p>專業方面，覺得物聯網是新的趨勢，所以我現在就開始學新的趨勢，以備未來接軌</p>
<p>現實中，我就是硬生生的沒有時間</p>
<p>所以我必須割捨掉一些事情跟好奇心</p>
<p>我得根據我的未來跟我的現況來做評估</p>
<p>即便我真的很有愛對語言很有興趣</p>
<p>想跟上電腦的流行潮流</p>
<p>但我都得根據我現有的時間，現有的狀況與未來的發展作為評估</p>
<p>沒有一件事情是能夠快速飛天的</p>
<p>所有的成功及所有的成就，都是慢慢的一步一步做改變，一步一步的累積，一步一步的進步與成功</p>
<p>就好像我在 2015 年的時候還是一個短髮妹妹頭很幼稚的小女生</p>
<p>一步一步的，隨著經驗，時間，事情的累積</p>
<p>我心智受到各種磨練，以及一步一步慢慢的改變我的外觀及想法</p>
<p>不可能一次飛天</p>
<p>任何的事情都需要一步一步慢慢的去改變</p>
<p>像是學英文一樣</p>
<p>一開始背單字練閱讀</p>
<p>然後半年後，磨練寫作</p>
<p>現在決定這半年來磨練聽力跟口說</p>
<hr>
<p>接下來，我稍微總結一下上面提到的</p>
<p>跟補充我在此次旅程學到的道理</p>
<p>先來標記一句，駐美教授給我的建議</p>
<p>不同於我爸</p>
<p>我其實不會像他崇拜美國崇拜的覺得美國什麼都好</p>
<p>但反觀我比較欣賞的是</p>
<p>美國是個大熔爐，什麼文化背景的人都有</p>
<p>對我來說，我也有個夢想就是希望能夠去世界各地，包括非先進國家，做各種文化的認識與交流</p>
<p>在日本，不同的縣就有不同的文化</p>
<p>這次讓我經驗跟有驚訝的是，美國各州也有不同的文化跟不同的環境</p>
<p>我個人覺得蠻新奇也很有趣</p>
<p>再來，我要來提幾點，我學到的觀點</p>
<ul>
<li>英文能力：尤其是聽跟說</li>
</ul>
<p>人類，接觸大部分最直接的方式，是透過耳朵跟嘴巴</p>
<p>除非寫文章或寫論文，需要用到閱讀跟寫作</p>
<p>不然人跟人交談都是聽跟說</p>
<p>此趟旅程，我深感覺得自己的英文聽力實在有夠差</p>
<p>我本人是有勇氣的敢講，但我聽不懂別人在說什麼，我也不知道該說什麼</p>
<p>尤其本人又是一個很會嚴格批判與思考的人，不能講話跟不能和別人討論時再痛苦</p>
<ul>
<li>溝通，社交與談判</li>
</ul>
<p>此次意外上榜(咦？，的是溝通與社交</p>
<p>我發現除了語言能力及專業能力以外，溝通與社交是非常重要的一環</p>
<p>先排除在台灣的情況</p>
<p>在美國跟在我目前上會跟國際學生相處的課程中</p>
<p>用英文溝通與用英文練孝威，都是很重要的事情</p>
<p>像是直白、自信的表達自己，多發言多問問題，多介紹自己以及幽默地講笑話</p>
<p>都是讓別人迅速認識你的方法！</p>
<p>尤其，我在 Juniata 上課的時候，每個學生在上台報告前都會先說自己的名字</p>
<p>每個學生幾乎都會問問題，並分享自己的想法，與大家討論</p>
<p>有的也會在討論中講講一些幽默跟笑話</p>
<p>而在忘記第幾篇，一位 Juniata 的畢業生來給我們演講</p>
<p>在美國，因為每個人都一樣，所以為了區別人不同的特質跟個性，每個人會自信地表達自我，介紹自己，還有分享自己的想法與和人溝通與討論</p>
<p>這樣能夠形成每個人不同的個人特質，也能夠透過這樣的作法，讓人認識你</p>
<ul>
<li>自信及自卑</li>
</ul>
<p>在整個旅程中，因為我算是唯一外系來的，很怕我以前國高中被排擠的陰影，怕被團體排擠，很怕被討厭，所以自卑感作祟，總是很怕自己孤單或者烙單</p>
<p>所以在一開始的時候便很黏我的室友，而且深怕他被人搶走。為了提升我的自信，我還會去說別人的壞話，以貶低人家來提升自己的信心。</p>
<p>在碩士期間，我已經很少會因為自卑還有怕單獨而去刻意討好別人，還有特別害怕一個人跟分不到組</p>
<p>自己的能力變強了，外表變好看了，多的時間我都能夠 Solo</p>
<p>只不過也許這次是因為要分房兩兩一組的關係，我才會又回到國高中被排擠的恐懼中，很害怕沒人要跟我睡，很害怕大家唾棄我</p>
<p>要是林北我能夠付 double 的錢一個人睡就好了，哈哈 XD</p>
<p>不過因為這樣我也認識一個互相說心事的室友</p>
<p>只是這趟旅程，也許是因為真的硬性規定要兩兩一組分房，我才感覺又回到國高中被排擠時代，必須求別人跟我一組，然後極度害怕一個人被排擠，自卑感跟被害感又出現</p>
<p>如果在學校那種分組，我通常都直接 Solo 或者能力比較強而且負責任，多半別人來找我@@</p>
<p>這也就代表了我還沒有完全有自信跟信心，同志們仍需努力 XD</p>
<ul>
<li>survive 的能力</li>
</ul>
<p>其實我很想要在駐美教授給自己建議的後面加上第四點：survive 的能力</p>
<p>與其訂一個夢想</p>
<p>我覺得應該要訓練自己 survive 的能力</p>
<p>survive 是我在 Juniata 跟老師學到的哈哈</p>
<p>基本上我將他解釋為生存能力</p>
<p>但我想把他的功能訂得很廣泛</p>
<p>講到生存，大家的想法，也許是孤島如何求生</p>
<p>也許是在叢林裡怎麼跟大獅子搏鬥</p>
<p>也有可能是怎麼一個人在異地求生</p>
<p>這邊都是，但我認為除了這些以外</p>
<p>生存能力比所謂語言能力跟專業能力更深層</p>
<p>像是溝通與社交是一種生存能力</p>
<p>像是我英文很爛，但我就空有勇氣，所以我敢交流</p>
<p>這算是一種生存能力，就是勇氣讓你度過這個可能會很尷尬的時期</p>
<p>很難描述，像是一個旅人，他武功高強，所以在旅程中不管任何事都難不倒他的意思</p>
<p>不會英文，有不會英文在美國學到新知識的方法</p>
<p>沒有專業，有沒有專業但可生存的方法</p>
<p>所以開頭我就說，與其訂一個夢想</p>
<p>不如將自己的各方面能力做進步與提升</p>
<p>只要機會來臨，夢想就達成，成功也就達成了的意思！</p>
<ul>
<li>一步一步慢慢走，好好經營當下的，但也要規劃未來的大方向！</li>
</ul>
<p>這就如上面所說，羅馬不是一天造成</p>
<p>不會有突然飛天的事件發生</p>
<p>不管是成功，還是成就，或者進步跟效果</p>
<p>都是要一步一步慢慢來，一步一步做改變，一步一步的進步，一步一步累積</p>
<p>才會慢慢到位的！</p>
<ul>
<li>很容易注意力被拉走</li>
</ul>
<p>上面也有提到，很容易被新奇的事情拉走</p>
<p>但後來發現空想跟做不到，或者與現況抵觸，所以又回到現實了</p>
<p>這點需要多多注意，因為畢竟時間很寶貴，應該要多點時間焦點在自己真正想要的事物身上</p>
<ul>
<li>與其煩惱，不如去做跟去解決問題，如果只有想，沒有做，什麼都不會成真！</li>
</ul>
<p>這個真的是，我時常會因為還沒到的事情煩惱</p>
<p>然後會像上一點一樣，空想一堆東西，但都不做</p>
<p>因此，我覺得只要是對的，不管會不會達成，只要去做，一定就會有成果</p>
<ul>
<li>專業能力</li>
</ul>
<p>這不用多說了，在國外工作，自己領域的專業能力真的要夠，才能夠說服別人！</p>
<ul>
<li>健康健身運動</li>
</ul>
