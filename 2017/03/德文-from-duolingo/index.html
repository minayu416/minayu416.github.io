[德文] Basics + Phrases from Duolingo

1 minute read

Danke bis später = thank See you later

Danke bis morgen = thanks see you tomorrow

Danke bis bald = thanks see you soon

Ja, gute Nacht = yes good Night

Auf Wiedersehen = goodbye

Entschuldigung = sorry

Ja, genau = yes exactly

Leider nein = unfortunately no

Es tut mir leid =  I am sorry

Danke in Ordnung

Wie geht’s ? = How it’s going?

Ja, mir geht’s gut = I am doing well

Alles klar = All right

Keine Ahnung = no idea
