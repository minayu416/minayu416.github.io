[留學]媽媽說不准我找代辦，所以我只好自己來

March 23, 2017 in Language

這句話取自於「雖然媽媽說我不可以嫁往日本」

啊啊啊又來到了黃金星期四！！！

因為我星期五就沒課了，所以星期四的晚上等於我的黃金假日

就好像星期五晚上就該像是度假一樣，要馬就是出去走走或者吃大餐！

不然就是買杯茶飲坐在電腦前看海賊王，看日劇，魯拉拉～～

但，我現在因為有訂每週Schedule，似乎不敢做什麼太狂歡的事。(º∀º)√

那麼唯一能讓我稍微舒壓的事情就是寫文辣(‾∇‾)

首先要先提及的就是我昨天為了咨詢我想要去歐洲留學的事。

由於我查網路上好像都只有高學歷的人丟歐洲高學歷的大學。

於是我就萌生一種只有高學歷才申請的了國外大學的想法。

事實上是不是真的這樣我也不知道辣XD

等我丟過，上了才能突破這迷思。

好啦！事實是這樣的，我昨天為了要了解留學的資訊，我就當天打電話給某某代辦。

一開始我當然天真覺得那就是諮詢用的，他不會叫我做其他事。

傷人嘛！哪會這麼簡單，於是他在我講到最興奮最高點的時候跟我說如果你想要接下來的結果！

你就得按照我們代辦的方案繳費來去得到，我就會~~~~~~

好吧！顯然我知道那是商業話術！而且我也感覺很想跳坑

於是，我帶著我的包包離開，總算是得到第一手資料。

坐上公車我就整個很疑惑很困惑很矛盾的在左右小惡魔打架。

因為我覺得爸媽賺錢很辛苦，當初托福補習錢繳下去了，也因為太忙所以都在混。

於是我就開始害怕之類的

跳過

總之，我媽當然是不答應。找學校自己找啊

當然我還記得某某PTT在板上講說「如果你連這點事都要找代辦，那你還確保你在國外能解決問題嗎？」

好吧衝著這句話我就….自己來了QQ

首先我把先把自己在google搜尋上大家提出申請的國家抓出來。

你可以開一個橫式的Word或者Excel

將「學校」「科系」「地點」「申請條件」「特殊條件」「學費」…等等

一欄一欄將查到的資訊網上填。

就在我害怕我根本只有幾所能申請的時候。

我在PTT的論壇上看到了一個DAAD網站的學校篩選欄！

不過只限德國喔！我當初諮詢的時候怕德國沒幾間這種大學(聽說都頂尖大學才有英文授課)

所以我還考慮到荷蘭跟瑞典瑞士(但這些地方就可能沒有免學費了，也會比較難找)

所以我就進去了那個DAAD的網站，看到這網站當下就噴淚了。

於是我把我的需求「computer science」「Master」「English」勾選後就”帕”

出來整行足足15頁「相關的」科系，跟電腦有關的科系都會出來。

然後我怕沒幾個computer science能申請，我就把software engineering, data …

資料分析相關跟多媒體我都有算進去，但最後會被我篩掉因為我沒想走這兩條路。

截至第六頁至少有45間學校跟科系！這樣我就不是很緊張了＠＠！

我應該會花個兩三天把所有的學校選出來後

再花個兩星期，慢慢把每個學校跟科系看過後一個一個篩出來(還有條件)。

基本上太難的跟偏離自己的主題應該不會申請

看看能不能找到至少10所(到底是多怕不上)

最後我要提供一下網路上找到的網頁

這個是找房子的，好像有點久遠不過既然先找到就先筆記起來，不過都是德語就是！

另外這個好像是一個論壇，有問題大家都會在上面討論，可以去爬爬文。

結語就是我覺得並沒有自己想像中這麼的困難跟難找還有這麼的資訊不透明

是自己的害怕淹沒了自己(不過還是要看最後到底能不能篩出我考得上的學校才能知道有多難QQ)

Erasmus Mundus 我也想看看，可是我自己可能覺得不太適合。

所以這方面我應該還會再仔細看看(因為我上去看沒看到跟CS有關的科系)
