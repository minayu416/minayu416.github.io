即將開學的週一

September 18, 2017 in Me

藍色星期一

我還在家裡，並沒有去學校

我這學期的課表集中在週四週五

經過一股挑戰的推力，我始終捨棄了「可以躺著過的課」來選「全英文授課」

在這之前我有退縮過，想說「啊～好累喔～」「工作已經很多了，還有論文」，卻還是得選這種可能會需要戰戰兢兢的課

未來，我們是不會知道的

雖然我都很喜歡將未來碩造成一個很遙遠的時間點

但其實並沒有想像中遙遠

像是我認為我無法負擔全英文授課

原因是我覺得我可能會很忙

我接了三個案子，所以我覺得很忙

但是「很忙」，其實只要時間管理得當，他不會是一個理由

我怕的點在於

  1. 可能會聽不懂

  2. 可能會被當掉

  3. 怕在大家面前說出菜英文

但是，如果不去的話，又怎麼會進步呢？

今年二月，我考了人生第一次英文語言測驗，而這個考試居然就是數一數二難的「托福」

各位GRE跟GMAT看官別覺得我很遜QQ

我是真的覺得托福很難啊！

考完看到分數後憤而向上！

但是當我知道我不用留學後

我就懶散了

才發現我學日文跟學英文抱的態度是不同的

因此學英文對我來說會是一種苦差事

日文的學習是這樣的

有興趣 >> 開始自學字母 >>開始自學文法 >>開始看日劇 >>開始背單字 >>為了測驗自我的語言能力而去參加日檢

而英文的學習過程卻是這樣的

要留學 >> 直接做托福考題 >> 開始背單字 >> 開始摩作文 >>

曾經我以為我的英文已經到了不能在進步的範圍內了

但我後來我發現我錯了

比對上面日文跟英文的過程會發現光是起點就不同，所報的熱忱也不太對

這時候我才發現，我雖然背了一堆單字，但我的文法卻是爛到爆炸

我有開始聽60S 美國科學網

聽力我有些聽得到單字

我也有聽出一點點語感

所以我想領悟出語感才是最重要的

而我並不是沒得進步，而是還有一堆東西還沒去練習

因此，我決定

之後除了持續背英文單字以外

學好文法跟練語感，才是最重要的

碩一下學期的主旨是在於拉「單字量」與練習「英文作文」

碩二上學期的主旨，我想定為「學好文法」跟「練習語感」

需要一個主題式的定義，才會有目標將生活往前進

而碩二上

由於有一個案件在10月即將到期

所以，大部分的時間重心則會在

  • 衛福部的案子

  • 全英文授課

  • 我的論文

  • 網頁(想辦法先推了)

再來，其他的時間，我必須明確定義出來，因為我不想被我的懶散跟玩電動給佔去太多時間

  • 全力唸英文

  • 副力擺西文

  • 運動與健康

再來是想訓練的能力

  • 時間管理

  • 溝通與領導

  • 獨立思考與實行

最後，我想探討的是

精神自由

由於自己耳根子弱

所以我常常聽到別人的意見，我就會被左右

不過大概是慢慢長大了

就會很想堅持己見

我內心很嚮往的是精神自由

因此，我想實行的事

自己決定的事情堅持到底
人生是自己的不是別人的
別被別人影響自己

以上，是我給自己新的學期的課題
