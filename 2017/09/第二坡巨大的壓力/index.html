第二坡巨大的壓力

September 24, 2017 minayu416

1 minute read

  • Twitter

  • Google+

  • Facebook

  • Reddit

  • LinkedIn

  • StumbleUpon

  • Pinterest

  • Email

覺得最近一直持續壓力很大

耳朵就會有坡坡聲～

一開始想說中耳出問題？

但後來發現可能也是壓力造成的

這樣的坡坡聲根據我看醫生的前兩三天大概是9/10開始

我有點搞不清楚是減肥的壓力？還是ETP?衛福部？

在開學的前兩三天，我很確定是ETP的課程讓我感覺壓力很大

很想逃避，又很想修課讓我變得更加強大

開學後，果不其然，第一堂ETP的課讓我壓力好大！

之後，有位熱心的男子推坑我再多選兩門英文課：翻譯與口說

我很爽快地就答應了

事後想想很壓力，但卻又不想放棄

就這樣一顆巨大的壓力壓在我的胸口

透過塔羅的方式，我看到，我有能力可以勝任，也許會多少手忙腳亂

但對於我自己會是好的

以前面對壓力，我會很悲觀很難過甚至是哭泣，任由著悲傷的情緒帶著我走

現在面對壓力，也許是因為有一半是自己決定而且也是很想要做的事

除了巨大的壓力外，更多的是高過壓力的決心與求生本能

第一坡巨大的壓力，我認為是在五月左右

剛接集保＋核銷＋衛福部的時候

由於剛接，跟剛跟我的指導老師磨合

我其實壓力很大，因為都不是自己想做的事

而且，每件事情又很急，那幾個星期簡直急壞自己

而且對我自己自卑的關係

所以我那時候很看重指導老師對我的看法

如果他罵我，我就會覺得世界毀滅了

如果他稱讚我，我就會覺得好一點點

價值觀建立在指導老師對自己的態度

但我一直很自卑，因為我做的工作是偏秘書，而不是寫程式

對資訊業來說，我做的工作叫做行政工作

這樣的工作讓我覺得自己是不是很沒價值

後來兩三個月我才慢慢將衛福部的工作給hold住，用自己的意思在做事

而最近，是新學期的開始

由於選了ETP又加上被應外系的同學拉進兩門課

加上為了去美國企業參訪，跟我的惡魔指導老師交換條件

一個月內用英文寫出proposal

真是惡魔…..

加上衛福部的案子還有集保核銷跟其他00碎碎的事

我真的覺得自己可能會升天

從週四開始我就心神不寧

內心有大石壓住

終於，我今日起床受不了跟我爸媽說

我爸則是跟我說

先畢業比較重要

於是我將自己內心的事情做了點排名

  1. 自己想做的事

  2. 需要畢業的事

  3. 其他的事

這樣算的話

  1. 英文課，寫程式(論文)

  2. 論文，選課

  3. 衛福部，集保

曾經衛福部的案子在我的內心是第一名

因為他是指導老師對我的看法跟想法最直接的媒介

但現在，我已經有自己想做的事了，想怎麼樣去玩我的人生

想怎樣去打破規則

雖然是這麼想

但我還是很有壓力跟緊張

於是，上了網來請塔羅大神指引一下

「我這學期選了好多課＋我必須在一個月內趕出企劃書＋時不時冒出來的衛福部案子，是不是會很有壓力？請牌給建議」

基本意思 : 穩健而認真的計畫是錢幣騎士的標誌。錢幣騎士通常指的是強化你的計畫，並朝確定的目標邁進。它意味著為了實現一個目標而努力工作。就一個人而言，這個人對於承諾非常的認真，不論是對事業、個人雄心或兩性關係。通常，他相信這個格言：如果你想做好一件事，那就自己動手吧。
行為暗示: 負責任、努力工作、實用主義、不解風情而且沒有情調、考慮周到。編碼過的訊息。法律方面人士。一個成熟且可靠的人、有責任感、有條理、有耐心、有恆心的、有能力完成一件事、有規律。
總論 : 這張牌意謂著你似乎比較擅長思考及談話，而不是行動。你做得不夠多，雖然有創意也還不夠成熟，這是你必須學習的重點。
雖然是網路上隨便找到的塔羅網站

點出來的結果卻意外的符合目前的處境

尤其是似乎擅長思考而不是行動，更是貼近我現在，光想光害怕，卻不去實行這點。

基本意思: 聖杯國王暗示透過情感和創作上的訓練而成功，經由落實精力在有創作的目標上，可以達到所追尋的成功。這個國王的成功可說是想像力、訓練及執著的結合。一種成熟、有創意的方法帶來琛功，尤其是在創造和藝術的努力上。這張國王牌以為你應該信賴你本能——別放棄。它暗示一種堅強又冷靜的方式。想像加靈感，再加上實際的努力就會得到回報。
行為暗示: 處變不驚、慈悲寬容、學識豐富、重視婚姻及家庭、創造力。發明家。責任和創意、受教育者、事業者、虔誠的人、科學家、周到的人、好心和可靠、自由的行為、對科學和藝術的好奇。
總論 : 適合從事有創意的工作
這張牌更是切中我現在很害怕我的企劃書生不出來的窘境，加上我現在內心又很慌張

要自己告訴自己「冷靜」，似乎是件困難的事情。

基本意思 : 權杖二暗示因成長而不滿當前環境，需要決定未來行動方向的時機。他表示你目前所擁有的事實是不夠的，你將決定下一步要怎麼做。權杖二代表決定，通常是身體上的決定。例如，搬家、轉換工作，或在生活中作一些具體的改變。如果問到有關你的生涯，這張牌則暗示你正決定是否換工作或換行業。權杖二並不代表換工作或搬家的行動，而是決定本身。行動是由權杖一所代表。在決定行動之前，權杖二代表對選擇的評估，它是你所習慣的東西與你所想擁有的東西之間的一個抉擇。
行為暗示: 思考與計畫、猶豫不決、受限制的行動轉折點、重要關頭。工作被外力強迫中斷。成熟的人、掌權者、達到目標和需要、負責的勇氣、一個強制的個性。
總論 :   決定是否轉變，並不會行動只會去思考且做出決定。猶豫不決的抉擇，而影響了身體出現些小小的不舒服。轉換環境再來思考未來。

「因成長而不滿當前環境，需要決定未來行動方向的時機」「生活中作一些具體的改變」

在暑假只有衛福部的計畫的時候，我特別的不滿意現況，覺得完全就不知道自己在做什麼，也看不到任何未來。

所以我的卻對於我剛開學做的選課的決定很滿意，卻也帶給了自己巨大的壓力。

「猶豫不決的抉擇，而影響了身體出現些小小的不舒服。」「轉換環境再來思考未來。」

不曉得要不要加選還是退選，可是我怕表現不好怕跟國外的同學處不攏，但另外一方面我又好想修這門課，雖然壓力大，但一定會成長很多。

因此帶來了壓力壓到自己心神不寧，睡不好加上壓心，耳朵出問題。

整體而言本來其實只期望網路上的塔羅測驗只要給自己出現加油跟堅持

沒有想到意外的準跟意外地貼近自己現在的狀況。

一堆事情混在一起，分不清哪個該做哪不該先做

其實很害怕的是衛福部的案子，民生學院的助理一直用官腔攻勢狂line自己

造成自己的精神壓力很大

最大的問題應該就是目前手上的案子，實在很想推掉。

綜合一下牌給的建議

  • 就是做就對了，決定好衝就對了

  • 冷靜冷靜再冷靜然後要相信自己一定能夠做到，發揮自己的創意來寫企劃書

  • 不要猶豫不決，好好的思考跟取捨，轉換個思考跟角度去切入

加油！
