我的藻球時代 with 極火蝦

September 23, 2017 minayu416

1 minute read

  • Twitter

  • Google+

  • Facebook

  • Reddit

  • LinkedIn

  • StumbleUpon

  • Pinterest

  • Email

之前的養蝦缸就是買了兩個蝦屋

長這個樣子

蝦子不知道為何都拍不到

由於我從舅媽家ㄎㄧㄤ來的水蘊草一直養不活

我就上網找找什麼草適合養蝦也比較養得活

  • 摩絲

  • 小榕

  • 鐵皇冠

摩絲自己買了蝦屋上面有了，我這次就想說看看要不要買個小榕

於是走進了魚中魚的水族館

還沒找到小榕，卻看到浮在半空的藻球很可愛><

然後就跑去找一個店員

「Hello可以幫幫我嗎？」

：恩？

「我家的水蘊草，一直養不活都不會長」「有沒有比較耐長的？」

：水蘊草已經是最耐長得了

「…」「有適合蝦子然後比水蘊草好養的嗎@@?」

：藻類吧，這些好幾種藻類都可以

光藻球就有很多種，我買的這種浮在半空中的看起來好像假的喔

我本來看重的是比較會往外長

但店員說這種會長得越來越圓，而且不用去修剪

原本是買這種然後也是浮起來的球

不好意思我是辜狗來的，所以侵權請告訴我@@

但最後我買了這種的

然後再詢問店員，小榕是做什麼的？蝦子會吃嗎？

：裝飾用，不會吃

可是我想要買幾個小榕裝飾XD

所以我就買了三送一小榕＋兩顆浮浮藻球

回來裝飾後

再將蝦缸換位置後，就變得很美麗了！！！

沒想到其實水草也需要加營養劑><，根本長知識

我媽還問我兩顆石頭也要買喔？

我說那是我從日本的四萬十川跟划船的老闆要的

至於去年從阿寒湖帶回來的綠球藻

早就被蝦兒們分屍成好幾塊XD

我已經放棄養綠球藻了哈哈，就讓他們在那邊任由小蝦吃吧！

蝦子已經有點懶得數了，尤其又多了水草們，更難數

快祝我蝦孫滿堂XD
