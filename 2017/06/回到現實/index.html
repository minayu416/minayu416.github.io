回到現實

June 27, 2017 minayu416

1 minute read

  • Twitter

  • Google+

  • Facebook

  • Reddit

  • LinkedIn

  • StumbleUpon

  • Pinterest

  • Email

在家都會亂想亂想

本來早晨興高采烈去外面吃早餐順便健走

準備走完要來買杯飲料回家好好的享用配論文加工作

結果指導老師一通電話，就臨時趕到學校加班了QQ

一到系辦，看到我的大學專題指導老師

寒暄兩句

接著聊起我的論文，我稍微做些介紹，在詢問有沒有什麼解決或者建議之後

我再度提起我想在出國留學的事情

其實這個問題我困擾蠻久的

目前暫定去德國唸CS

但跟我當初選我的指導老師一樣

問10個9個反對

但我現在蠻喜歡我的指導老師的

雖然常常被狠狠的操工作，可是是真的學到很多

我也很固執地說，我知道你擺一大堆工作給我，讓我接一堆東西

我有時候是很生氣我的指導老師就是死愛面子，不敢當面稱讚，也拉不下臉安慰我

可是我卻也在他面前說

：我也有想過換指導老師，可是我除了你，我沒有其他想換的老師了

離題了

說穿了我就是希望有個人能夠支持我出國唸書

我知道我這樣到處去問人很常打我自己的臉

因為就算我得到了各方面的建議，我還是很固執的覺得，如果不是我選的路，我會想走嗎？

就算有9個都建議去美國念書，可是我的固執還是會讓我去選別的國家

至少，我常常拼著這股叛逆然後做出一小番成就

接著我遇到我的指導老師，他對我下達加班的工作內容後

我就慢慢往秘密房間前進

然後坐下來先來處理核銷案再來處理案子的內容

這些「現實」硬是戳破了我在家裡幻想三天的粉紅泡泡

我努力的克制自己，結果我一到學校接觸教授們，還有我的學業跟工作後

我便了解了我現在目前應該要面對的是我的工作，學業和我的未來

我的夢想還有我的目標

放學，剛好5點半，我打了通電話給我高中要好的朋友

他在附近上班，我們一起搭車回家

一上捷運，她用帶點期待的語氣及活潑的表情問我

：怎麼樣？有進展嗎？

我則嘆了一口氣回她

：哎，別說了

她則稍微皺眉

：為何？我看你上週一才興高采烈的傳簡訊說巧遇他兩次

我帶著點抱怨的口氣回覆

：恩啊，可是我現在光是面對自己的事情我就顧不得了

：我的論文，專題要開始做了，不知道生不生的出來

：我的指導老師讓我接了三個案子，我還得處理一堆案子的事

：最重要的還是我的夢想，出國留學，我還得拼英文

：實在沒有時間

是呀，這些「現實」，讓我認清事實，但卻也慢慢的讓我離開幻想及不實際

因為我開始要面對我的人生跟未來

我不再是以前那個年輕的小學生了

做什麼事情都要考慮到未來，以後，投資報酬率

也不能耍任性，整天打12場LOL，或者從早上六點打魔獸打到晚上

每一分每一秒每一天都很珍貴

故事拉回小時候

來談談我國小老師問我說，你的偶像是哪個偉人呢？

我則回答：路易十四世

因為當時正在玩世紀帝國

我翻開小學畢業紀念冊，看著我那面自我介紹，居然是當個旅行家

當時正迷著海賊王的尼可羅賓

國中是個正熱血的時期

我一腳跨入學校剛成立的女藍校隊，和同學朋友們一起熱血練習籃球

我的夢想轉為籃球國手，即便我當時是球隊的「三軍」

高中時期

音樂老師則是問我喜歡哪個偶像？

當大家開始議論紛紛：五月天啦，王心凌的

我則回答說：GIRL NEXT DOOR

這個回答，可是把音樂老師弄糊塗了XD

因為我國三的時候迷上日文

高二的時候不顧老爹的反對，天天都看日劇，還有自學日文

幻想著住日本，去日本留學

雖然大學沒有如我所願上了日文系，但卻也練就一番日文功力

矇懞懂懂直到現在

才發現原來我每個階段都比同年齡的人都還「老成」一點

原來我當時答出的答案，不會是同年齡的學生會答出來的答案

原來我是個怪咖

而現在，我卻樂於當個怪咖

長大後，剛開始一樣對每個新奇的東西產生好奇想要了解想要學習

但是現在已經沒那個衝勁

又或者說，當我想接觸一個新語言的時候

我必須了解的是，我已經沒有那樣年輕的時光，可以讓我不顧一切的揮灑

接下來所學習的東西必定是奠定我未來人生的基石

於是我在俄文跟德文之中徘徊了很久

那陣子幾乎讓我抓狂

根據自己念資訊加上科技，我毅然決定先不念西班牙文

直接接觸德文，然後先將日文擺一旁，直攻英文

因為我長大了，我開始慢慢的知道，接下來我下的每一個決定

都會影響到我未來的人生，慢慢的一些零零碎碎的枷鎖就掛在我的身上

他們阻礙我前進，讓我迷茫

我得承擔下決定後的後果

但在這之前我得先問自己

「如果現在不拼，不做，你以後會不會後悔？」

「如果你真的聽取別人的建議而不走自己堅持的路，你會後悔嗎？」

「你要讓別人來抉擇你的道路嗎？還是寧願衝一發，然後自己在為自己選擇的路承擔？」

「你真的甘願放棄或者甘願偏道嗎？」

即便我知道我已經老大不小了

但我下的決定卻需要我年輕時的衝勁

只是這次衝下去的結果，將不能被隨便忽視，不能隨便中斷，不能隨便扔擲

我指的意思是說，他不像我學日文一樣，付出一堆心力，還有時間允許自己猶豫不決，要不要去日本留學，要不要繼續學習日文

他也不像我學西文一樣，不帶夢想的，只為交友的耗在西文班兩年，最後卻做了一堆跟西文無關的決定

不像國中的女籃，反正打得很差，換個夢想就好，不用負責任的

這次做的決定將緊緊扣住未來，緊緊扣住我的人生

只因為長大了，該為自己負責了，做的每一件事都是為了耕耘未來

接好多好多案子，忙到天昏地暗，常常遇人不淑，為的就是賺點錢，當做自己留學的基金

自願當班代，好好的上課，好好的做報告，好好的念書，為的就是讓GPA好看點

利用天天通勤時間背英文單字，每週約定自己要寫一篇英文作文，逼自己看英文論文看英文文章

為了就是考好托福然後申請學校

不同於以往，常常什麼事情都不做然後就擔心自己上不了日本的學校，會不會被排擠

也許就是成熟了，所以開始面對，如果要達成夢想就必須得好好付出努力才行

而我最近領出一個道理

會猶豫，會被外人說動，會鬆散提不起勁無動力，說穿了也就是決心不夠。

不然怎麼會被一堆無關緊要的事情給動搖了呢？

沈默並不是生氣也不是傷心，而是多了一份滄桑，堅定深深埋藏在眼神中

看上去多了一點嚴肅，多了點老成，少了點稚氣，我想這些都是人生畢生之路

好奇妙的感覺…
