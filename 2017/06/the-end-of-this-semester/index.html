THE END OF THIS SEMESTER

June 22, 2017 in Me

好快，我的研究生生啀已經過了一半了

好吧，又是週四的回憶感傷時刻XD

這學期居然結束了，好無感的感覺

想當初這時候剛進來找老師，接著就開始忙碌

雖然感覺好快又過了一年，但是確實經歷了很多坑爹坑娘的事，哈哈

剛剛還不知道怎麼下筆，然後開始打字了之後，腦袋就慢慢的一幕一幕湧出畫面

好險，我都有來部落格記錄當下的心情，這樣就能回去翻翻這一年發生何事

真不知道我的腦袋怎搞得，常常對之前發生過的事情完全沒印象

  • 自我的茫然

  • 人際關係的煩躁

  • 社會的黑暗與利益

我只記得我剛進學校就開始茫然

為的就是系主任的那句話：網頁太多人做了，你要不要改道？

讓我覺得那我對人工智慧有興趣，而開始慌張自己是不是選錯老師了？

然後開始一連串壓力大的班代，英文補習班及功課還有人際關係問題

再加上自己的茫然，就更錯愕了

終於，網頁瓶頸了，一度以為我會以這個當作是未來的目標跟工作

然後開始將希望轉移到指導老師身上

渴望從指導老師身上得到自己的專長

還記得我二月剛拼完托福也是一陣茫然

也記得3月因為德國留學被老爹打槍而茫然

還記得3月因為不知道要學德文還是俄文而煩惱

然後又記得自己就算去出國念也是不知道要念什麼科系，又怕選錯路而煩惱

然後開始對自己的人生覺得無聊沒自信沒有熱情對待每個東西

接著開始我發現，我都是被別人的話語所影響

我從來就沒看過沒有接觸過自己的內心

於是，就開始著手找回自信還有找回自我跟開始去尋找我的道路

最近，老毛病又犯了

太在乎，結果就讓自己的世界又垮一半

我認為這不是一個好的開始跟好的人生

於是，我得喚回我的理性

因為我知道，我不能又再一次失去自我

對人生沒熱情

太在乎一些事

還有失去自我

顯然都不是什麼好事

有一點慘的是，冷血跟感受不到人類的情感

時常在想隱居跟想狂認識人的鋼索上跳來跳去

我真的是感受不到人的情感嗎？

這樣聽起來好像挺悲催的

相處一年的學弟，我居然只用一個晚上的時間哭一哭

就可以跟他們說再見了

我覺得我們很要好，但又怎麼感覺如果之後都見不到也無所謂？

也許從那一天開始，我就把我的內心給封閉了

因為不讓自己再受到更致命的傷害

以前我都會問說：你能不能當我的好朋友？

將自己的心交給人家

結果後果當然是悲催的

如果可以，還有人可以當我的好朋友嗎？

可以讓我把心交出去嗎？

還是，我注定只能當個旁觀者？

誰能把被堅強包圍著的傷痕累累的自己解放出來？

接下來

失去自我

我只有差不多3.4天的時間可以調整這樣的自己

因為下週三就要開工了

我得用這幾天時間把自己調整好

走回原來的步調，以及持續努力自己的目標

調整自己，找回自我及我的個性

那個表面上喜歡玩喜歡裝作不在乎，然後喜歡高自尊貶低人，實質上有智慧，古靈精怪又很調皮的自己

喜歡挑戰，喜歡冒險，喜歡獨立行事(挑戰)

那個看起來有自信，默默觀察每個人，在必要時偷偷拉人一把的那個人

為了找回人生的方向跟道路

我必須做幾項反思

  • 擬定Schedule，沈思並找回夢想和目標

  • 找回自己的特點跟個性(這樣就能繼續以自我過生活)

擬定Schedule，沈思並找回夢想和目標

首先是這個，我想最重要的應該還是衝刺英文了

再來我在想是不是唸德文

至於專業，我想交給學校跟老師，畢竟我還沒有找到自己的道路，我就先不擔心這個

所以，我認為還是以衝刺英文(衝到托福90分)為主

次要是有時間可以念念德文，看有沒有機會考過歌德學院的基礎

再來是專業的部分，現在是什麼都會什麼都學，看起來沒有一個契機，可能要等入暑假才知道了

這個不擔心，反正學校跟科系還沒找@@

好好地探索完，決定了申請國外學校才能好好專心鑽研

不過倒是希望科系能夠跟科學，數學有關就是了

以及能夠衝擊到腦袋創意及思考的科系

然後，還有處理好學校的工作跟專題

最後，還是希望自己持續每天重訓跟留長髮燙捲變美啦XD

喔！還想要能會點武功XD，保護自己強身健壯

夢想喔

世界旅行和世界各國的人跟學者交流，還有瞭解世界的知識跟文化種種

好像太廣泛了@@!

那麼就還是以學習知識跟留學為主

打算真的出得去，就想辦法出去看看

看能夠得到什麼想法上的衝擊跟矛盾

所以還是重視「知識交流」跟「思緒衝擊」

因為台灣很多觀念跟想法還是蠻傳統跟侷限

看看國外的思想究竟能夠開放及創意到什麼程度

總結的話

就是需要做一個小的Schedule

那大致的方向則是

  • 衝刺托福

  • 學習德文

  • 處理專題及論文(從中找到未來想鑽研的科系)

  • 處理學校及指導老師交代的事

  • 每天重訓及留長頭髮燙捲

找回自己的特點跟個性

不曉得自己是怎樣，個性常常變來變去，連自己都摸不著自己

有時候聰明，有時候又很現實，有時候又很逗比，有時候又很調皮

姑且

我還是把自己稱作為「裝乖寶寶的小惡魔」好了囧

因為知道自己無法裝乖呀XD

但冷血跟壞也不會壞到哪去

只知道自己有一點點聰明，然後很愛裝愛玩裝不在乎，又很調皮很愛想一些事情

然後去實行，然後來驗證到底是可以還是不可以XD

尤其超愛看不認識的學妹崩潰的樣子

很愛逗弄學弟妹XD

好吧….只能裝個乖然後來掩飾內心古靈精怪的一面了

但事實上自己卻很愛自己獨特跟怪咖的內心呀…

本來想說要將自己的怪咖表現出來，但發現還是加一層乖寶寶面具比較好XD

至於人際關係，雖然常說自己冷血跟感受不到情感

但絕大多數都是覺得….維持關係太麻煩了

還有超級怕人家黏，一黏世界都塌了…. 自由多好呀！

  1. 就好比煞到一個男生 >> 狩獵開始 >> 啊，他不理我，好討厭 >> 恩…維持關係好麻煩 >> 算了不要好了XD

  2. 煞到 >> 狩獵開始 >> 他的個性不太OK，他的腦袋沒有這麼聰明，他好笨 >> 冷掉

  3. 煞到 >> 狩獵開始 >> 對方也很熱烈 >> 對方主動接近 >> (恩？不太對好像太快，需要觀察？) >> 退一步 >> GG BYE BYE

  4.  那講沒有煞到的 >> 純聊天 >> 對方聊出感情 >> 我無感 >> GG

反正不管怎樣都悲劇呀，所以自己好像還是比較適合相處久一點的人QQ

到目前為止沒有出現過聊出感情的案例

喔對了，最近煞到很多會雙語甚至是三語的人

所以我有點想要學習像他們一樣

會雙語或者三語轉換

然侯很有自信，有些會經營部落格有寫會拍youtube

因為我感覺這樣的語言者，他們就會懂很多跨國的知識，真的很讚！

然後這樣獨特的人，就會散發出很獨特的氣質跟個性，還有過很充實跟豐富的生活的感覺

所以我也會想像他們那樣！

總結以上，就是想當一個獨特的怪咖(喂)，還有帶乖寶寶面具的小惡魔(或者愛玩不在乎但內心卻嚴格觀察跟認真的人)

我會努力朝著精通語言者前進的XD ， 目標是去學習不同國家的各種的知識

最後，我要分享我那陣子很糾結德文跟俄文

我一直覺得時不時用別人聽不懂的語言講話很帥XD

有時候會對於自己能夠看懂別國語言這樣的狀況感到開心跟成就

為什麼想學俄文

主要是因為看了馬路須加學園3有個俄日混血的演員講俄文，覺得俄文發音超美

再加上俄文的字母真的是直接就讓人家看不懂

因此我認為學會一定很有成就感

為什麼會猶豫，主要是覺得廣用性不高，只有俄羅斯跟部份東歐

我要學這個語言前有做足好功課

  • 相較於其他語言，德文，法文，日文，英文及西班牙文，實用性沒這麼高

  • 雖然是工業聯盟國家之一，但科學跟科技發展已經被美國日本還有德國佔去

  • 老了現實，認為要學習新的語言，希望能學一個自己有興趣而且對未來工作有幫助的語言

事實上，我現在在學德文，有時候看到莫斯科有些科技發達的新聞跳出，還是會動搖

不過當我接觸德文的時候又是另外一個世界

來說說為什麼後來唸德文

  • 其實很多科學家出自於德國

  • 德文據說是歐洲最多人使用的語言

  • 雖然字母看起來都是a,b,c,d,e,f，可是組合起來也是打啪一堆人看不懂XD，這點讓我覺得學起來有成就感

  • 發音！ 發音我覺得很困難，聽很少德文，但之前聽到瑞典話，真心覺得喜歡，據說很像所以想來學學看(至少我覺得他的發音跟西文跟日文還有英文都不太懂)

  • 抓不到規則跟發音讓我覺得很神秘抓不住的感覺有挑戰性

  • 工業國家發展之一，除了美國英國澳洲跟日本以外次要的留學選擇

好啦，雖然有時候看到莫斯科有些很厲害的發明或者厲害的新聞出來會動搖

但目前怎麼列都還是德文贏呀XD
