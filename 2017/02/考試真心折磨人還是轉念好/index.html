考試真心折磨人，還是轉念好

February 11, 2017 minayu416

1 minute read

  • Twitter

  • Google+

  • Facebook

  • Reddit

  • LinkedIn

  • StumbleUpon

  • Pinterest

  • Email

從1/1號開始便漸漸的閉關念英文，每天都按照自己規定的計畫，要背多少單字，要寫多少模擬考，要讀……算了

這樣的過程是每個考生必經之路，但隨著這樣的過程，便漸漸讓我覺得這個世界很無聊。

就算要說我沒定性，穩定不下來，或者遇到壓力就跑也沒關係，可能我真的遇到壓力就跑？

一直以來，我不太會表達自己的情感，在學校或者在任何地方可能會出現暴怒，傷心，難過，各種情緒，但內心中就是覺得這樣也不是表達情感的感覺。

有點像是，內心的靈魂抽出來，站在旁邊看著我自己難過，傷心，生氣，發洩各種情緒。

一進寒假的開始，我就一直在找各種電影，還有偶像劇，甚至想出去運動，我完全沒有心準備考試。

還蠻討厭自己這樣的，總是對考試逃避逃避的。就讓我想到以前大學的時候，期中考的前兩週開始，我就出現了這樣的狀態：切斷對外聯絡(除了家人跟以前的男友)，每天睡到11點起床看書，看到晚上12點，立馬切換狂看電影跟…..種種，劇情還研究得很透徹，然後開始想學習各種東西。不過考試結束完，又變回原本的樣子。

幾乎每次遇到期中期末考都這樣，所以我常責怪自己，沒有抗壓能力，這樣就想逃跑。

然後，又讓我回想起，前陣子，我常常發自內心的很愛學日文，融入日本的文化，了解他們。日文就在不知不覺的情況下成長很多很多。

可是，每次我要考日檢的時候，都會呈現上面說的狀態，面對考試會煩躁，不想讀。甚至在考完試後，有陣子會非常討厭日文。又或者，在考完試的當下會發誓好好準備下一次。

後來我才知道，我其實根本不適合考試吧？

「考試」，是為了檢測人的能力到達的標準，但也為了「檢測」，所以刻意針對來設計出一套考題。

但問題就出在這，一個人的能力真的能夠用「考試」來檢測出來嗎？ 我這個寒假幾乎困在這個問題裡。

在小時候，可能我只要提出這個問題跟質疑，早就被學校老師跟爸媽(一般家長)給批評跟質疑想太多了吧？

就算是現在我也還是會怕提出來被人家念想太多，於是就把這問題拋一邊。

幸好現在長大了，以前在國高中，為了要考試考出好成績，學生都會逼迫自己去學習，在這樣逼迫自己的情況下，其實不太會有什麼好效果。

我認為，學習一件事情，是要人自己發自內心想要了解其「知識」，才會去學習。

但往往會在「考試」「利益」中朦朧和迷茫。

我在考試前一週半開始對考試緊張，各種不安，因為我去用我模擬考檢測出來的成績遠遠不足我期望的成績，然後我又想趕在什麼時候把這事情結束，又一昧的一定要交換學生！

種種的逼迫讓我把自己逼出了點焦慮，甚至我念到最後產生了「我為什麼要念這個？他讓我好辛苦？我到底是為了什麼？」

我身邊的人就開始勸：要考試了還在亂想？你不好好撐一下，你還想出國？你在逃避？

但我覺得好辛苦，也許會有人說，想出國跟準備考試本來就是辛苦的。

這裡先打個岔。

入寒假後，我跟我老爹透過我的命盤去寓個言，預測今年我老爹不是升官就是退休。

果真年後，我老爹就跟我說他想退休，他辭呈都遞出去了，也問過菩薩了。

於是歷經了幾天反反覆覆，在遞出辭呈的兩天後被高層挽留，於是又繼續工作。

這時候也是我準備考試壓力的最大的時候，我老是跟我爸媽抱怨說，我怎麼唸都念不下。

可是，我不在現在就把這考試結束的話，我的交換學生就沒了。

所以我把自己逼到整個緊張。

我原本以為我爸媽會逼我讀書的，不過我老爹跟我說：

你不能去交換學生也可能是天意啊！你還記得你當初申請大學的時候，你狂丟日文系申請，只丟一個資管，就算兩個面試強撞，你也還是選擇先去日文系學校面試，結果你最會中資管。你不覺得，你之前在急在緊張都沒啥用，因為你根本就是走別的路。

我媽也跟我說：

不要把自己侷限在這條路，逼死自己幹嘛呢？一定還有別條路。

聽到這麼開明的話，我反而嚇著了。我不知道是因為他們不想看到我這麼緊張才這樣說的，還是？

拉回正題。

於是，我在考前6天釋懷了。我回想起來，我從1/1開始致力精心準備考試，突然我發現，人生不是這樣過的。我不願意去賭那個命運，換句話說，就是不要去相信考試的時候成績就突然變好。因為那根本就不是我的實力呀！

我在說什麼蠢話。

也就是說，我為了考試去背考試的單字，考試的文章，考試的技巧，再去賭考試的運氣。考出來的不是我的實力。

正確來說，應該是我先累積自己的實力，然後在才去做上述的步驟才對。

我在追求交換學生的同時，我失去了初衷，熱情跟享受。我對這世界感到厭煩，對我的人生覺得無趣煩悶，也是因為我在過度追求一個「利益」跟「目標」

有目標跟夢想是很好的事情，但並不是要「為了＿＿跟為了達成＿＿而做＿＿事」

這樣偏離了目標跟夢想的原意，對我來說，應該是「發自內心的想要學習英文，想要學習知識，想要從＿＿學習什麼」的態度，來追求目標跟夢想，這樣才會永遠保持自己的熱情跟初衷，才會想好好的享受這一切。

我了解我這樣的舉動不是逃避考試，也許我已經沒有心去準備了，因為我知道我的實力跟別人實力的差距。

但我想改變一下我對這件事情的態度跟做法，換個做法跟態度，會使人生更快樂，也能更能了解什麼是辛苦的但卻是快樂的。

所以，我這兩天，不是忙著準備跟苦讀我的考試，而是好好改變想法，並且運用在下學期，讓我的人生更快樂更享受，雖然是用了5000元的報名費跟一個半月的時間去想通，也沒關係。

被質疑我的做法跟想法也沒關係，因為我一直覺得，人生沒有所謂的生活規則，沒有所謂「應該要怎麼生活」的框架才是。
