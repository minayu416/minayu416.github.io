Duolingo x 親身體驗 x 語言學習過程

February 22, 2017 in Language

語言學習真的是一條漫長又久遠的路程。

我個人小學五年級學習英文，然後國三學習日文，大四學西文，現在碩一最近剛被俄文煞到。

那麼學習的過程又是怎樣呢？先說說自己的故事。

=====2018/12/13

後來我有再重新寫了一樣主題的文章在這，可以參考最新的文章唷！

語言學習過程與Duolingo

我英文在小學四年級的時候有去一般小學補習班補習(就是考不好會打人的)，這時候好像英文還是很差，沒什麼進步。

小五進科見美語補習，科見的教學就屬於美式教學。

優點就是基本上透過玩遊戲(填空，連連看，拼字…)或者真正在使用和反覆運用，讓英文一點一滴滲透進您的腦袋，並透過使用加強記憶跟應用，進步可以說是很快的，當時都能聯合各種小朋友欺負外師了(壞蛋)。

缺點的話，就是我上國中的時候，面對我國中英文老師的教法，整個趴掉了。

國中英文老師，不外乎開始教字母/單字詞性(不及物/及物)/五大句型跟什麼不定詞後要加什麼？

如果我沒記錯，科見裡是沒有這些所謂的「定義」，所以當初我在國中銜接的時候英文直接爛掉了。

在科見我跟外師溝通完全沒問題，但在學校英文的考試都吊車尾，我完全不知道為什麼要考什麼句型什麼不及物。

因為這些所謂中文給他冠上的「定義」或者「規則」，早就在科見的教學，遊戲，運用中自動建立在腦子裡反應了。

有些人學語言適合用國中英文老師教法：學單字，學詞性，教文法。

但科見的教法我比較喜歡是因為透過他的教法，我是可以自然說出英文，而不用特別在腦子想應該怎樣加為什麼要這樣。

這邊也不是說要推薦還是讚美科見美語，只是拿出來比對兩者的不同。

之後大學畢業有段時間想考多益的時候有去補個人家教，內容就是做題目/背單字

前陣子考托福有補托福密集班兩個月，內容是講解考題/技巧/背單字跟做題目

後來我發現，這樣的補習：背單字/做題目/技巧/背五大句型什麼的，對我的英文壓根沒有進步。

日文的話淵源就深了，剛開始的50音是自己學的，學法是看一看50音然後去日系商店，把每個看到的物品(有日文)都想盡辦法叫出來。

這樣的學法，我大概是一個月還是兩個月就把50幾乎建立在腦子中，可以用脊椎反應的程度。

反觀我看過有些人都在學文法單字了，還不太會念50音。

中間有去地球村把基礎50課文法上一上，不過真正在進步的時候，還是我自己看書整理跟學文法的時候+上大量砸影片砸日劇跟砸視覺的書籍。

還有我會去網路上玩pigg日文社交遊戲，這邊也是進步很多，也自己用日文打部落格網誌。

有幾段時間我會為了通過日文考試而去買專用單字被跟聽還有看。

其實這種準備考試跟補補習班對語言進步跟提升完全是 無 效

接著是西文，西文的話也是去補習，剛開始我西文就跟不上，除了發音我會。

單字跟什麼文法大致我都跟不太上，可能可以說我對西文其實沒有這麼有feel。

西文完全都是補習，沒有額外的練習，西文補習班的教法還算不錯，就是教文法，然後給大家時間運用文法句子造句跟大家聊天。

這讓我的西文到現在還有一點點記憶，主要是上課的時候，老師會讓大家互相聊天跟練習。

這樣偏向生活的運用跟應用，反而比較能讓自己學好一個語言。

不過，因為除了補習以外的時間都沒在自學西文，所以我的程度也就只有那樣了。

這樣綜合下來，也就是說要學好一個語言其實是從生活跟運用下手。

好了我接下來要推薦這個手機app

最早是載下來大概擺了半年甚至一年沒用又刪掉。

然後再載了又刪掉！

最後還是載了，之後想說被俄文煞到就來試試看這個app。

本來我也是想說要像一般學語言開始先背字母。

所以在每個字母都還不知道的時候。

我用蘋果手機，我先去我的鍵盤把俄文輸入給加到打字鍵盤中。

然後按照Duolingo app的進度，每天20分鐘做他的小遊戲。

今天應該是第三天，我已經記得幾個字母怎麼念，然後會一點基礎的文法(This is / my/ here)

我個人是覺得蠻經典的啦，如果我自己自學的話可能第三天還在學字母跟字母發音吧！

要知道俄文不像英文是我們從小就根深蒂固的abcd，也就是說我三天的時間，每天20分，到今天已經會一點字母跟基礎的文法。

我個人蠻推Duolingo這個app的，可以利用每天20分鐘學習自己想學的語言。

現在的進度還蠻滿意的。而且我感覺自己沒有硬去背什麼東西到我腦袋裡，可是看到了就是會反應。

最後，如果想要學好一個語言，建議不要為了考試而去學，先慢慢好好的去熟悉語言。

如果非必要(要求證照)，少去碰語言考試的東西了。

我的觀點，我覺得語言考試會把我的熱忱磨掉，二來我覺得考試不太會真的增進我的語言能力。

所以除非前提建立在已經有一定的能力了再去碰考試。

不然直接一開始就立志我要考____，最後可能會因為煩躁煩悶無趣還有考試的壓力弄的自己討厭這個語言一點也不好！

我考日文檢定6次，每次的日文檢定考完我都很痛恨日文，明明最愛日文，考完立馬恨透。

都要用一段時間才能平復，剛開始完全不曉得是怎麼回事，結果後來發現原來是考試惹得錯。

對語言的喜好程度也會因為你考好幾次都考不上，就消耗更多對語言的愛。

三四天前考完托福，考完我也很痛恨英文。

其實只要調整好心態，學語言其實是很快樂的！

最後我覺得台灣一般教育的語言教學完全不行啊！！

這樣的教育跟語言考試都會摧殘一個人對於學語言的愛！
