[正面面對英文] 托福準備紀錄

人生第一次英文檢定考試

Posted on Feb 18, 2017

今天剛完成了人生第一次的英文檢定考試-TOEFL

真的是強者在考的，考完瞬間覺得鬆了一大口氣。

看似四個小時長的時間，其實寫題目的時候時間過得神速。

考的不是考生的英語能力，考的根本就是考生的反應能力。

即便這樣我還是準備考第二次了。

考完就自己一個人搭車到淡水躺在草坪上好好思考了。

當前幾週在準備聽力的時候，我就發現我的能力沒到。

本來以為61分交換門檻很簡單，結果我現在反而覺得40分都很困難了吧？

今天，主要最困難的就是，考聽力的時候一直想偷聽人家口說，所以我分心了。

口說即便真的聽到一點，我還是來不及準備。

下次別再這麼貪心了，聽了口說送了聽力。

聽力，我自己實際聽我覺得沒有TPO40回的速度，感覺上沒有TPO這麼快，考點也大致都抓得到。

口說其實沒有自己想像的這麼可怕，我一直害怕考口說，因為講不出來就完蛋了。

但其實讓我害怕的不是口說而是我自己對口說產生的恐懼感。

寫作的話，整合寫作我個人覺得應該還…..可以辣xD。

好險整合寫作合格把模板套上去破200字。

我比較擔心我的獨立寫作，因為我根本就….看不懂題目，只好硬掰掰看。

閱讀的時候也是覺得時間過好快來不太急寫。

對我而言難度大概是，聽力>閱讀>寫作>口說。

一開始我認為的困難度，口說>寫作>聽力>閱讀。

結果實際考完倒過來。

最痛苦的就是考寫作的時候，我的腦袋滿滿的日文文法。

所以這就是問題了。

長期都用日文去思考，長期用日文轉換和人對話，現在考英文的時候完全轉不過來。

這個大概就是我接下來需要克服跟面對了。

雖然我最近都在一個學俄文還是學西文的圈子中打轉。

後來去仔細研究了一下我的日文學習過程我才真的發現。

  1. 日文因為是一見鐘情，簡單來說就是自己喜歡有興趣，所以還沒去補習班補，就立馬回家自己找資訊學習。

  2. 日文有用大量的電影跟日劇還有綜藝(無字幕)去砸出來。

  3. 很愛在洗澡或在路上用日語跟自己對話，用日語跟別人對話(這個過程會在腦袋中選字，所以很習慣跟熟練了)。

  4. 至少學了八年，歷經6次檢定考試。

這樣綜合下來，大概也能知道為什麼我日文還不錯了。

回想一下，我的英文從剛開始就是為了通過托福而考，自然就是等進補習班才開始。

也就是說不是在補習班的時候跟不準備考試的時候基本上是不太碰英文的。

光這點在對比日文就缺乏了1,2,3,4 ～，咦！那這樣當然考不好！

再來舉學習西文的例子，當初我記得學西文是為了旅行。

而當初學西文的時候也是覺得去報補習班，然後被動等補習班教。

去補習班學，下課後回家就擺在旁邊不太念，然後等下一堂課(那個時候密集班每天都有課)。

當初學兩週就跟老師說我覺得跟不上進度，我想休息了。

神奇吧，如果一個語言是自己打從根本就想學的，其實應該會一直想念很練，自然就進步神速。

但居然從最一開始的abcd教，然後還是這個樣子。

同班的同學倒是進步神速，可能就是因為他對西文有興趣的關係。

即便我知道西文在全球算是第三大語言，我還是顯得有點懶懶的。

當初我不知道對一個語言有沒有興趣居然影響這麼大。

因為我學西文的時候，就是懶懶得去報補習班。

在學習的時候，我也不太會去把西文相關的電影或者劇集拿出來聽。

就像現在的英文一樣。

上面也說過了，即便現在覺得西文可能很重要，但我還是提不起勁，可以稍微K書一下。

只是，我猜我可能會用寫程式很忙，念英文很忙的藉口推掉去學西文。

再來，來談談最近打中我內心的俄文。

三天前在某部日劇裡聽到俄文立即打中我的內心。

還記得最一開始的日文打中內心是因為

  1. 和日本人溝通成功很有成就

  2. 我覺得日文寫起來很優美跟英文不同

  3. 那個時期學日文的人很少。

再來對比一下，現在的俄文。

  1. 我覺得是發音吸引到我

  2. 俄文字寫起來也是跟英文不同

  3. 俄文對比德文法文跟西文，在台灣算很少人學了

打中內心的當下，當天我就睡不好了，因為整個興奮(喂~)

也就是這樣我連續興奮了三天到今天，還在為了俄文興奮。

而且我也立馬去載學語言app開始學習俄文基礎。

然後也去找俄文的電影來狂聽是不是真的我想像中的優美，還是我聽錯了。

不過也是讓我矛盾跟煩惱幾天了。

因為

  1. 學太多語言，我怕人家會覺得我怎麼不好好把三個語言學好就好

  2. 聽說俄文很難，現在開始學不知道未來用不用得上

  3. 我有去找如果第二外語選俄文的話對工作有沒有幫助，(不過IT業至少是英文好重要)

  4. 我發現我媽可能會反對。

當初我爸也是反對我念日文，但是我媽支持我。

現在念俄文可能會被兩邊都反對，讓我覺得好卻步。

可是又好想學。

然後糾結的點繪在於，如果付出俄文的時間學不起來，那那個時間拿來學西文會不會剛好能把西文學精？

類似這種腦殘還沒發生又過度擔憂的狀態。

所以這樣的矛盾跟煩惱存在於我的腦中有幾天了。

整個讓我很心神不寧的，不曉得該如何。

可是說真的，我學俄文的動力跟積極力真的是高出英文跟西文很多，而且，我不希望去補習班學俄文。

我希望可以的話，讓俄文在我的興趣跟無壓的環境下長大。

也基於爸媽可能反對，所以我就不去補習班補，我認為也沒那個需要，畢竟真的有這麼大興趣，不用透過第三方去學習，自己就會去找資源跟硬擠出時間的！

雖然說是這樣，但一想到以後萬一對工作沒幫助或者被爸媽反對的話，想起來就煩惱。

但，我只想說就去做吧，在這邊光想，想破頭了還真什麼都沒有學到。

就像我開學前幾天就在擔憂我會不會開學工作論文學業混雜，無法學英文，可是俄文就可以，因為興趣所以我一定會擠出時間。

所以差別就在這了，其實根本不用擔心這麼多，就是去做就對了。

只要有心，時間都擠得出來的！

語言這門領域一定是要經過大量砸時間練習才練得出來的。

現階段，先以英文為主吧！ 俄文就是自己每天擠一點小時間出來學囉。

把日文邏輯腦先改成英文邏輯腦。

有的時候覺得很奇怪，學日文的時候不會擔心日文會不會沒工作。

學英文的時候也不會擔心英文有沒工作。

現在想學俄文反而就害怕學了當第二外語會沒用到。

但反過來想，我西文也是學了，但沒什麼興趣擺在一邊，怎麼反而不去想為什麼西文找不到工作哩？

如果說IT相關就是英文，那麼第二外語我真的就想學一個自己有興趣的呢！
