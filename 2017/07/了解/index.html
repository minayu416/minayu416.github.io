了解

July 3, 2017 minayu416

1 minute read

  • Twitter

  • Google+

  • Facebook

  • Reddit

  • LinkedIn

  • StumbleUpon

  • Pinterest

  • Email

未知，其實才是最令人害怕的謎團

當你慢慢了解之後，恐懼跟迷惑便會漸漸的退去

我曾經說過，對我來說，讓我害怕的是迷茫徬徨，而不是高標

因為高標，只要努力就能達成

而迷茫則是不曉得該如何下手

那天，我向我朋友說

：我認為應該要找能發展性的工作，這樣就能夠爬上去，賺更多錢

而朋友則是對我說

：對你來說，工作就是很重要，你會優先以工作為主，而對我來說，工作只是為了維持生活的一部分

這時我才恍然大悟，原來我真的很在意經營工作這部分

我沒有辦法讓自己去做一份沒有發展性的工作

然後領一樣的薪水

原來我已經跟一般的女生有點不太一樣，把工作當作人生努力的奮鬥目標

所以到現在趨於穩定，我才會有點不滿

因為我認為穩定=沒進步

今日把工作告一段落後，我開始來查詢去德國留學的資訊

雖然這已經被我爸給否決了

但在短時間內我應該是準備不來GRE的

所以估計就不考慮美國了

爬了很多很多文章，包括比較Software Engineering 和 Computer Science，以及我找到如果有幸上了國外的學校，要趁寒暑假去Intern，好像就是實習吧！

包括在德國畢業後有三個月可以找工作，包括畢業後只要工作幾年就能拿到永居

很多很多都找到，但我發現一個很有趣的問題，就是「語言」

不管是永居還是找實習，都要會德語

因為現在已經24歲了，如果再找一門新語言來學的話的卻有點懶散了

現在就蠻希望自己當初是煞到德語不是日語XD

不過我認為，如果還要再選一門新語言的話，就是德語了

而且這年齡之後，應該就不會再有任何想接觸新語言的想法了

因為差不多要開始磨專業

雖然是晚了點，不過定在現在開始，學一門以後可能會用到半輩子的語言

或者可能會以他為找工作

還算是可行

大概過30就不會想要有什麼學新語言的想法了XD

不過，也因為爬很多文，所以我有看到有去申請德國的網友

Software Engineering, Computer Science 跟 Data Analysis 都有丟

我估計如果我往後的半年沒有蹦出其他想法

之後應該也會丟這三個領域

而我發現老師訓練我來整理弱勢家庭案，其實也是訓練我開始寫論文

也就是廣泛收集資料，然後一一分析跟整理重點

當勞工是沒錯啦，但因為老師漸漸讓整個弱勢案都給我處理

讓我也多一份信心，能夠Handle一個案子

這也是為什麼當初簽這個案子的時候老師是找我，而不是找寫程式的學弟

接下來，就是Proposal了，估計12月多丟，3月左右初試(?

總之我不希望太晚丟啦~

因為之後要寫一些扔國外的文件，可能時間要喬一下@@

還有英文是時候該衝刺了

也剩下沒多少時間了，間天載了Lecture來聽，還是有點聽不太懂的說QQ

希望聽力能趕緊加強

所以！！！！！！

假日的時候可千萬不要再亂想碗糕了，英文都沒時間唸了嗚嗚!!
