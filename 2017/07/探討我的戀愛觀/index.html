探討我的戀愛觀

Posted on Jul 01, 2017

有時間，我就窩在這個小小的家，記錄我生活的一點一滴

也許是天性，讓我覺得任何事情都記不住，很容易忘記

越長越大，發現自己沒有很喜歡跟朋友相處

慢慢的，工作追夢求學之餘，最喜歡的就是聽歌寫文章

但也不是什麼邊緣人

以前總是會嘲笑自己是邊緣人

但到最後發現，由於自己的個性吧，身邊的朋友跟同學是尊敬自己

而不是將自己當作練蕭瑋的朋友，也許是受過太多傷，慢慢的不相信人

另外一種則是大家都長大了，沒有人有時間一天到晚陪著你練蕭瑋

也許是長大了或者成熟了，不然就是自己想當個人生旅人/旁觀者

於是，變成時常一個人

以前很喜歡聽嗨歌，當然現在也是啦XD

不過打文章的時候常聽的是帶點滄桑的歌曲，以及慢歌

也許是我看透了自己容易被周遭環境影響

於是讓自己建一座高牆，好好的做自己

至少，我喜歡現在這樣做自己的感覺

從旁觀角度觀察每個人

最近探討的議題，其實是自己很想交男朋友

可是又覺得為什麼要交一個男朋友來內憂外患XD

常常很瘋迷就跑去塔羅牌去知道自己的什麼未來六個月桃花之類的

每次知道要是有，就會期待，萬一沒有落空就會失望

後來經過好幾次塔羅牌甚至是易經算桃花後

發現其實是自己的問題居多

也許是因為自己在追夢，然後時常忙碌

所以不會花太多的心思在顧另外一半，因為在追夢，也不想要

有句話說得不錯「當你不想談戀愛的時候，戀愛就會悄悄將靈」

這句話有在前男友身上靈驗

而太想談戀愛的自己經由算命去得知，然後又沒信心覺得自己差

後來最近就在檢討這個事情

以下言論不是針對男生，也不是在責罵男生

當自己所有的時間都奉獻給工作學業，而且還要抓緊時間唸英文準備留學

我哪來的時間額外跟人聊Line

而且說不定連男生朋友約我出去，我還得瞧時間XDD

跟人聊Line其實有時候會打斷正在做的事情

比方說正在認真工作，就不會想要回line

而且偏偏曖昧又需要大量的回應訊息

這點對於我時常在忙碌，感覺…..

另外一種就是，最近爬文發現

其實有時候不只是外貌身材

如果你太能幹，太精鍊，太聰明，男生也是不敢追

因為男生會自卑

可是…. 我總不能以後結婚的時候給男生養吧

另外，台灣低薪的狀態，要組成家庭最好也是雙薪家庭

越賺越多，生活才會過的好呀！

至少我是這樣想的啦

但我去爬文會發現男生普遍不喜歡比自己強的女生

那你們也變強不就得了QQ

又想要過好日子，又不讓女生幫忙賺錢，想要展現自己大男人的魅力跟思想

每天做到快GG了，為的就是展現大男人的包袱

為什麼不雙薪然後住在大樓頂樓喝紅酒咧

不過當然，展現大男人的包袱，就會開始被親戚，鄰居，好朋友們給稱讚

How about woman?

在家做家事跟帶小孩的媽媽/女生就會接受各種攻擊啦XD

也許是從小看太多身邊長輩的例子

加上自己個性跟思想的問題成份也大

首先是經濟權掌握在男人身上，啥時被休了都不知道，在家帶兒煮飯，然後還會被各種親戚親人問候「我兒子在外打拼這麼辛苦，你在這什麼事都不做，真好」

你在外面打拼很辛苦，我總不能拿著你的錢到處環遊世界吧

因此，沒有經濟權，意味者你得在家帶小孩煮飯還得接受問候，然後什麼地方都不能去

因為去了就會被問候，你在家什麼都不用做，我在外打拼這麼辛苦#$%#$&@#^

靠XD講到這還是單身好，根據台灣的風氣，我嫁一個老公同時代表我嫁給公公婆婆叔叔嬸嬸大伯二伯

接著每次過年聚會，可能會被逼問，生了沒呀？小孩子幾歲辣

我兒子上台大，你兒子呢？(抱歉台大躺著中槍QQ)

有女朋友男朋友了沒呀？什麼時候要結婚呀？

乾你屁事

小時候談戀愛根本玩家家酒

大學時候在煩惱沒有男朋友

看對眼就在一起

分手後煩惱的是下個男友會不會是恐怖情人

然後再長大一點，會發現談戀愛會背負各種各種的包袱

大學那時候還在那邊想，我大概25.26歲交一任男友

28歲結婚！

我現在想都不敢想，我只知道嫁一個男人，會附帶一堆套件跟插件

我眼神就已死

我再強調一次，我沒有在怪罪或責罵男生

強調歸強調，但我還是覺得會附帶一堆套件跟插件XD

也許是看太多交往很久的情侶，因為爸媽不喜歡就散了

我真的真心覺得

付出這麼多，然後過不了爸媽那關就吹了

不覺得浪費了大把時間跟鈔票在上面，然後什麼都沒得到咪….

好吧我太現實了

所以時常去算算命，或者感嘆自己沒有男友或沒男人追

不過檢討之後我發現，自己的問題才是最大的

這不是貶義，也不是說自己有什麼缺點

而是單指思想跟個性，還有時機，會覺得現階段不太適合交男友

也許還年輕必須得衝，所以我不曉得現在的想法會不會因為之後長大又在變，也是有可能的

但至少釐清了並不是什麼沒男人追，也不是自己不好

有時候就是因為可能真的不需要不適合(這邊指的不是說心裡所想，指的是從旁觀者來看自己)

旁觀者角度會覺得我本人現在過的就不錯了這樣

另外，也有可能是自己的個性跟思想還有散發的氣質

基本上有很多因素，而不是因為自己差自己沒自信，也不是因為自己醜身材不好

也可能是自己活在象牙塔當中

可是覺得稍微成熟之後，很多事情都不能成單一角度切入，或者不是是非題就是。

這樣說服自己就能讓自己好很多辣XD

不過我也確實認為，不管是人生還是戀愛

這些事情這些領域其實不能用是非題去定義的

更多時候，他其實包含著一堆不同層面的涵義跟成分

最後，彩卷中頭獎的機率很低

可是戀愛「中頭獎」的機率很高

如果平凡還好

如果一個不小心中到結婚或者戀愛中的「大獎」

這輩子可能就GG了

Good Game / Game over了…
