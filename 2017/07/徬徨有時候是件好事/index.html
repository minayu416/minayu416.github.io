徬徨有時候是件好事

July 15, 2017 in Me

搞清楚自己真正要的是什麼

有的時候其實是一件很困難的事情

因為，當身上背負著一堆枷鎖

再加上世俗的道理，傳統的規則

以及媒體的渲染還有大眾的習慣

很容易就影響自己的決定

老了或者到了這個年紀，已經不像以前那個樣子

耍叛逆說我要幹嘛就幹嘛

也許就是因為長大了成熟了，才想得更多更躊躇不進

一心想要明年出國留學，其實造成的原因有很多種

第一點無非就是覺得自己在台灣念的書根本不夠

認為自己根本什麼都不會，覺得去國外接受更高等的教育會學到更多

二，其實就是為了快速結束自己畢業的徬徨

換句話來說也算是”逃避”工作或者”逃避”面對自己徬徨的理由

第三點，目的就是為了明年是2018年

在我命盤裡面是很重要的黃金一年，而且剛好走到遷移宮(就是會換一個地方呆)

命理有時候很有趣，究竟是注定那年會變動，還是要自己去讓他變動

為了這一年，我「刻意」讓自己變動，而不是像命盤寫的「變動」

這樣的探討很有趣

這樣感覺複雜，舉個例子

在不知道自己的命的情況下，走到明年，自然就會有任何事件或者際遇讓自己「變動」

但正是因為現在知道，所以才想辦法讓自己「變動」

兩者其實是不太相同的，但也正因為「知道了」，所以「刻意」安排自己變動

到現在，這樣的「刻意」反而化為泡沫，因為不管怎樣「刻意」去安排

目前都是被阻止或者環境還有條件不足

好比說：我刻意讓自己2018年要變動，所以我刻意安排自己要出國去留學

但到目前為止，透過學校的雙連計畫，是卡住的，因為念的不是自己想念的CS

那如果單純交換學生，也是被我老爹給阻止，原因是不要浪費時間

再來，如果是再去唸一個碩士，申請美國，也不行，因為要準備GRE，時間遠遠的不夠

去英國留學的話，自己不想去加上去英國要考雅思，我已經準備托福到一半了

去澳洲也是怪，雖然自己蠻喜歡澳洲的，但就是一個字「卡」

最後找到了德國，但還是被老爹阻止了

原因是我爸是美國派的，而我自己最後檢討了自己也發現我根本是因為不想考GRE才被迫選擇去德國念書

這樣為了想要剛好卡在明年「變動」而刻意「變動」，我已經遭到算是報應的，任何刻意都無法成真

但正是因為這樣的徬徨，我才開始正視自己真正的需求

因為我還困擾的是在CS內，我究竟要走哪一條分支？

老實說，不曉得這個，就算出國唸書，面對的窘境也會是現在這樣

Computer Science? Soft Engineering? Data Analysis?

是我認為我自己會喜歡？還是我真的嘗試過，我是喜歡的？

在沒有搞懂這些之前，如果就這樣跑去留學，會照成更徬徨

而且，定力跟決心不夠，畢竟完全不知道自己究竟要朝哪個方向前進

就像我昨天跟我學長說，我在想要去國外念CS還是Soft Engineering?

我學長則用很驚訝的表情回：軟工喔？很硬噎？

我則是回他說：是的，就連我自己也都無法知道我到底是「以為」自己會對軟工有興趣還是真的有興趣？

這兩個差很多喔！

學長則又回了說：其實現在工作也沒有分的這麼的細，這麼多分支，是因為學術研究跟論文的分支不同，外面工作並沒有這麼多分支

：最常見的還是

  • 網頁前後端

  • 開發app

  • 資料分析

學長：最常見的還是那幾個語言，你看看自己對哪個語言有興趣

我則是回答他說：我現在跟著老師做，就是一開始寫網頁，跟現在他可能要讓我開始近資料分析，大數據的部分。

：我現在也沒有那個機會去開發app，寫Java跟IOS

：所以目前我也就是接觸網頁跟現在即將要開始學習資料分析

學長：網頁不外乎用php, node.js呀，你網頁用什麼語言寫？

：Python

學長：Django? Python 我不行，我對那完全不了解

：是啊，老師要我開始學R跟SAS

：所以，你覺得我要先工作在留學，還是先留學在工作？

老實說，要我很理直氣壯的說留學，我反而也覺得卡卡的，畢竟「留學」是我「刻意」卡2018「變動」所安排的路，但目前好像任何一條路都卡住。

難道，你們不會覺得，我畢業都25歲了，還再去國外念兩年，畢業都27.28了還沒有工作經驗，一定很慌吧？

再加上，我現在唸這兩年研究所，被老師操到都快吐血了，再加上我很討厭同樣的Schedule，Loop18次(一樣的課表跑18週)，我還會想再申請一個研究所荼毒自己嗎？QQ

而我覺得，從進研究所用Python寫網頁開始，到現在要學建立資料庫，學R學SAS，總覺得這方向像是朝向「資料分析」的專業前進。

：學長，我覺得網頁設計好累，動不動就跑出不同的語言，你知道完成一個網頁要用的語言有時候多於5種嗎？

於是，我打開Google，Key入「資料分析師」，跳出來的語言不外乎「Python」，「R」，「SAS」，或者「Hadoop」

怎麼好像跟我現在的事有點像？

揉了揉眼睛，又看到「資料庫」「演算法」「數學Model」「機械學習」「Data Mining」，我笑了笑，又是一群「有趣」的東西

想到之前「資料庫」的課是大學裡面最高分的課，那時候還想一鼓作氣地去考Oracle的資料庫證照。

沒想到這樣看起來會是我之後的路？ 但我要學的還好多好多喔

在今天搭老師的車的時候，我則是在沈默中蹦出了一句話

：老師，我覺得，我可能不一定會去留學耶

老師則一如往常老人碎念的回：想這麼多幹嘛？英文念好就是！

雖然花的時間有點久，不過大致上還是找到了方向跟努力的目標

  • 英文

  • Data Analysis 相關

我想這會是我之後發展的方向

至少，我一開始會以為我未來可能就當網頁設計師，朦朦朧朧的摸了一年

在2017的一半7月暫時是決定了這樣的方向，而且看起來也是老師現在叫我做的跟以後會發展的。

不同於以往，想要寫網頁，老師不讓我像學弟那樣寫網頁。想要學加解密，老師不讓我碰資安部分

雖然一開始被現在這個弱勢案給壓得氣死，但現在卻反而覺得這可能會是帶出我未來路的一個契機。

為了讓自己不再徬徨，還有為了刻意讓明年的「變動」成真，我硬是選了一條留學路

但卻在最彷徨的時候，看清了這條路。

現在的自己，已經不會這麼的Care留不留學，反而開始會去想，那我要怎麼去發揮跟學習我現在找到的方向？

我已經不敢去想像明年可能的「變動」會如何產生 而是開始為了明年可能的「變動」而「做準備」，我想這樣才是對的吧？

為了不能去美國而去德國，稍微刷了德語

為了想學德語，而迫自己放棄八年的日文

現在想想，應該要看看自己有什麼，而不是一昧的去補自己沒有的

會日語，加上現在在衝刺英文，還有誤打誤撞學了兩年的西文

感覺上以後會派上用場

人生真是奇妙

我不該為了變動而刻意「製造」變動，而是為了可能會「變動」的機會或者際遇而做好準備。

傻呆啊自己，過了一年半終於想通了？
