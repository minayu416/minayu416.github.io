2017 四國之旅 Day3 高知～桂濱海灘

May 28, 2017 minayu416

1 minute read

  • Twitter

  • Google+

  • Facebook

  • Reddit

  • LinkedIn

  • StumbleUpon

  • Pinterest

我記得前一天好像停在…

搭高速巴士前往高知的過程

到了之後就拉行李來旅館了

旅館是松榮第二別館

離路面電車はりまや橋站跟南はりまや橋巴士站很近，

快要過橋往右走，看到白色建築就是旅館了唷！

旅館外還會寫有誰定房間很貼心

然後最令人訝異的其實就是日式和房有內服乾濕分離的浴室跟房間

玄關

超美～～

這…由於我早上才記得拍哈哈！

好的，去玩旅館後，就去南はりまや橋公車站等前往桂濱的車，真的大概半小時會到

然後因為實在沒什麼人讓老爹他一直緊張

也因為店都沒開，我們只停留一小時就走了

接著就回高知站，確認明天的電車時間還有畫指定席！

然後到高知站旁邊的居酒屋吃晚餐

最後再回家（一樣搭回はりまや橋）
