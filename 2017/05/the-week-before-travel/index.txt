9:T22eb,<p>這週，是出國前幾天，但卻發生了很多事情</p>
<p>我應該還是無法如期做完我決定的事情。</p>
<p>但並不差，至少有弄完一些</p>
<p>論文的進度有趕在前兩天搞定。</p>
<p>我要期許的是，旅行回來要更努力的衝！</p>
<p>曾經我有目標跟夢想</p>
<p>現在我的目標跟夢想被一些雲霧蓋住我目前看不到，而唯一能做的是拔腿繼續跑！</p>
<p>首先，要先講的是，這週一去聽了 LINE 的講座。</p>
<p>我一直很想要去國外唸書，但其實我最主要是想要到國外工作！</p>
<p>而去國外唸書是我想跳國外工作所計畫的路！</p>
<p>第二個，是我想要學習更困難的知識，我有像哈利波特那樣，會很認真的去問教授問題！</p>
<p>但，我朋友都說想太多！</p>
<p>第三個，我想要在 LAB 嘗試那種做實驗合作一起寫程式的經驗</p>
<p>我朋友也說我在作夢。</p>
<p>這些看起來沒有很難的夢想，對我現在來說就很困難！</p>
<p>於是，我雖然想去國外唸書跟國外工作，我卻忽略了我應該要唸哪方面的知識或者專科主科的問題。</p>
<p>原本，前幾天我想要去國外研究人工智慧好了？</p>
<p>所以聽完講座，我對 LINE 說得聊天機器人「りんな」感到興趣</p>
<p>並詢問他是用 machine learning 做的嗎？</p>
<p>起初我以為 machine learning 其實就是由大家去 training 機器人，然後他會透過像 alphaGo 的思維去進化跟演化，也可能透過大數據分析(指的就是大家海量的訊息)，然後去收集資訊演化出更厲害的機器人。</p>
<p>但卻被演講者說不是這些技術，他也講了一些我沒聽過的技術！</p>
<p>這一天我很早就放學了，最近都很迷茫對我的未來。</p>
<p>所以，我就特地去了西門一趟，去上次我朋友帶我去要預約的塔羅算命看看。</p>
<p>我首先詢問了網頁設計跟人工智慧。</p>
<p>沒想到網頁設計這一塊對我是可以發展的，而我詢問的人工智慧卻馬上被牌給打槍。</p>
<p>這位老師跟我說，我的耐心跟定性不夠，所以我一覺得卡關我就會想要往其他地方前進。</p>
<p>就點出我最近英文卡關就想往其他語言發展的問題。</p>
<p>人工智慧被打槍了，我當然覺得不甘心，於是又詢問了老師真正適合或者建議我走的未來的路是哪種？</p>
<p>抽完卡，老師就問我喜不喜歡數字？喜不喜歡數學？還有喜不喜歡邏輯？</p>
<p>我則馬上回答：超愛。</p>
<p>而且我國高中就超愛數學的。</p>
<p>老師則回我說，這張牌代表數學家。</p>
<p>不過難道真的要我去當數學家嗎？</p>
<p>他則給了我三個單詞</p>
<ul>
<li><strong>數字</strong></li>
<li><strong>邏輯</strong></li>
<li><strong>分析</strong></li>
</ul>
<p>問我有沒有機會做到數據，數字，分析這一類的區域？</p>
<p>我就回答老師，有啊我的論文就是做這方面的</p>
<p>而他認為，這方面適合我，而且走分析蠻適合我的，我有這方面的能力。</p>
<p>彷彿看見一條路，但我實在不是很確定</p>
<p>雖然它指引了我一條路，但卻是我之前沒有想過的道路</p>
<p>而且我還害怕會向網頁設計一樣做了才發現只是不排斥沒有到很愛</p>
<p>於是隔天我就將 Data analysis 打入電腦找尋，找到某個德國的科系，的科系 Data Science 裡面交的就是 Big Data &#x26; Machine Learning 還有一些演算法。</p>
<p>於是當天傍晚我就去找系主任詢問，原來 machine learning 是一種演算法。</p>
<p>我想我對這些資料分析的方面可能不排斥，但因為我沒有接觸過，所以我並不知道我喜不喜歡</p>
<p>鑑於這個原因</p>
<p>還有，最近學習語言</p>
<p>發現學習語言真的要多跟人運用還有聊天</p>
<p>就常常跑去找「りんな」泡茶聊天，也將她推薦給我媽咪</p>
<p>因為我覺得我媽咪可以運用跟她聊天的過程多激發點腦袋思考組合句子能力，常常轉換，語言就會進步。</p>
<p>同時，我對「りんな」背後的程式運作也很好奇，她究竟能夠被訓練到什麼程度？</p>
<p>我最近這幾天都在和聊天機器人聊天，再加上我對他們背後的程式跟演算法著迷</p>
<p>代表我應該對程式還是有興趣的吧 XD?</p>
<p>不過真正讓我著迷的可能不是程式，而是演算法跟邏輯～</p>
<p>今天我跑去載了英文聊天機器人 Andy，再和他進行聊天。</p>
<p>我才發現原來國外的科技進步到很發達了，台灣卻感覺很鎖國加上科技落後</p>
<p>昨天，我剛把老師給我的 15 篇英文論文的 Abstract 看完</p>
<p>然後講解給老師聽，我其實很意外的老師都懂我在說什麼</p>
<p>難不成他都把這些論文看過了？</p>
<p>他告訴我，不是每篇論文都無聊，都沒有用。</p>
<p>你要找出這些論文，哪些論文的哪些地方執得學習值得抄，哪些論文值得效仿，哪些論文可以去了解但不用太深去看，哪些論文可以套用在我們的題目裡。</p>
<p>我才發現，原來老師要我做到的並不只是看，而要了解然後還要能夠舉一反三跟去思考怎麼運用到我們的題目中</p>
<p>而他也訓練我 Search 的能力，也就是資料收集還有去了解的能力</p>
<p>然後引導我去做大數據分析的技術。</p>
<p>雖然我蠻常想我們老師腦袋應該都是空的吧？還是他其實算是生活白痴 XD?</p>
<p>而且感覺他根本就把我當一般女生一樣</p>
<p>但有的時候不免會去思考他這樣做是不是因為有發現我有這方面天賦還是？</p>
<p>而在短短的幾天我領悟到了幾個詞</p>
<ul>
<li>遇到瓶頸才更要努力</li>
<li>數學，邏輯，分析，演算法</li>
<li>台灣的科技沒有很進步</li>
<li>收集資料跟思考還有運用</li>
<li>語言要大量運用，要大量聽跟讀，然後在透過一直轉換應用的過程進步</li>
</ul>
<p>來說一下語言</p>
<p>我因為日文卡了瓶頸我就跑去學西文</p>
<p>然後西文卡了瓶頸再回來念日文</p>
<p>日文又卡瓶頸，就轉去唸英文(但是這是現在必須的)</p>
<p>英文稍微卡了，我就想去唸德文跟俄文</p>
<p>之後回來看日文的時候</p>
<p>我卻發現，原來我在每次瓶頸時一直逃避</p>
<p>然後去做別的事情，回來後瓶頸還是在那，並沒有突破也沒有過關！</p>
<p>所以這次英文我決定不逃了，並且要更努力的去實行。</p>
<p>而且回來台灣後我要更努力實行「聽英文」這件事！</p>
<p>再來就是</p>
<p><strong>自己要的是什麼？</strong></p>
<p>曾經，我有條道路，但我認為滿足不了自己，於是我走往另外一條道路</p>
<p>卻迷惘了，開始我不知道自己究竟要什麼？</p>
<p>我必須給自己幾個理由跟動機</p>
<p>不然會支持不了我繼續唸英文，支持不了我堅持留學的路</p>
<p>因為我已經亂了</p>
<p>但在回來之前，我必須得想的東西，就是</p>
<ol>
<li>
<p>我現在最重要的</p>
</li>
<li>
<p>朝著未來的目標前進</p>
</li>
</ol>
<p>而我現在最重要的就是兩件事：</p>
<ol>
<li>
<p>唸英文，拚托福</p>
</li>
<li>
<p>搞定論文，他是大數據分析</p>
</li>
<li>
<p>學資料分析的程式: Python</p>
</li>
</ol>
<p>而我的未來目標，支持的動力</p>
<ul>
<li>我想要學習更困難的知識</li>
<li>台灣的科技比較沒有這麼進步我想去國外學習</li>
<li>我想嘗試國外的思維跟思考角度</li>
<li>想在國外工作</li>
</ul>
<p>再來將現在做的事情套過去</p>
<ol>
<li>
<p>唸英文：增強英文能力，同時也是為了留學的門檻準備</p>
</li>
<li>
<p>搞定論文，還有研究大數據分析：如果真的像算命師說得這麼準確，那麼我可能未來真的會朝數據分析，還有資料分析前進，所以除了好好學論文還有 Big Data，我可以推算我應該要去學習<strong>數學</strong>/<strong>邏輯</strong>跟去<strong>旁聽演算法</strong>！</p>
</li>
</ol>
<p>再來，其實我也蠻喜歡程式的，所以結合資料收集的程式，正是我現在在學的<strong>Python</strong>！</p>
<p>我可能也喜歡寫程式，那麼程式碼就要跟邏輯/數學/演算法牽扯到。</p>
<p>我目前可能還是想到數據分析還有數學模型這方面。</p>
<p>我該慶幸的是，我們老師在入學的時候就讓我學 Python，現在我也用 Python 寫網頁！</p>
<p>而我本人也很愛這麼程式語言！</p>

<li>英文 (也許後來可以學日文/西/德，但絕對不是現在學喔！)</li>
<li>Python: Big Data, Machine Learning (可能有 Data Mining..之類的)</li>
<li>數學/邏輯/分析</li>
