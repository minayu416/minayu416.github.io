和日本朋友見面

May 13, 2017 minayu416

1 minute read

  • Twitter

  • Google+

  • Facebook

  • Reddit

  • LinkedIn

  • StumbleUpon

  • Pinterest

  • Email

今天和去年認識的日本爺爺朋友，看上去其實感覺身體力壯，但年齡我覺得應該是介於50~60。

去年帶他遊玩龍山寺跟西門町認識的。

當時一見面，他講著超級神速的三重口音日文，學了日文好歹八年，可是卻完全聽不懂他在說什麼，非常的錯愕。

去年見面的時候就是很錯愕，聽不到就笑

然後抓到訣竅就是：讓他說，然後「恩～」「喔～」的回應，但其實都聽不懂XD

而這次，因為是帶一整天，所以剛開始一樣處於一個聽不太懂的狀態，然後我自己也處在一個日文講話是一段一段組合的狀態。

中期的時候，開始進步到對方講完兩秒後，我反應過來是什麼然後再回話。

而最後要回來了，已經進步到大概知道意思，而回話已經可以回完整且迅速的句子

這次的狀態，是在我經過兩個月成功切到英文模式後接觸到的第一次日文衝擊。

因為我腦袋已經是英文模式了，講日文的時候腦袋整個卡住只有英文單字

在這向口譯敬個禮！！

不過我因為在意我的日文能力，我邊玩邊在坐車的時候思考了一下

由於我會一直鬼打牆的說，學了西文在學個德文吧之類的，然後再告訴自己會日文會英文再會一國或者兩國的語言。

然後最後覺得自己應該可以學四國語言之類的想法XD

可是其實都沒有很精通

這次日文能力的震撼後，反而讓我更想持續並且更努力地唸英文

  1. 我認為達到托福可能80分都困難了

  2. 我覺得我英文卡瓶頸了

  3. 因為卡瓶頸所以我就開始會懶惰

  4. 因為卡瓶頸了所以會開始想學其他語言來去分擔瓶頸的壓力

大概就是以上的模式

但，如果要達成出國留學的話，必須全心全意的Focus在英文上

再來，我目前覺得我完全無進步，可能就是因為我一直在跟最好比，眼睛一直看著前面。

卻發現其實我已經走了很多路了。

套一句我指導老師跟我說的，「我當初規定自己GRE單字5000個沒背完別想出國」

我現在其實正在背托福的高頻2500個單字，但其實我算了一下，包含他列舉的同義字應該有5000個字左右，再加上托福的學科單字跟之前美加上課筆記的單字。

應該保守估計6500~7000個字，我應該也要告訴自己，在沒有把這6500~7000個字吃下去且滾瓜爛熟之前，別在那邊怕什麼考不好，別在那邊怕出不了國，先自亂陣腳了。

再來就是，因為我開始卡瓶頸就會自暴自棄但其實，我比之前更少且更懶惰去接觸英文了

原因就在於卡瓶頸，但我不應該這樣，因為卡瓶頸我就去尋求其他語言的安慰。

反而更是要更Focus在英文且更專注於練習英文才對

然後要給自己多點信心了，告訴自己，努力一定會有回報的。

現在五月了，別在迷茫了直接油門踩下去吧～！

告訴自己6500個字沒吃下去之前不要在那邊慌慌該該叫XD

為什麼要這個樣子呢？

因為我當初日文就是好幾次卡瓶頸，卻又不敢面對，然後就開始透過其他的興趣去轉移注意力，去透過其他的興趣去企圖說服自己至少還會什麼

然後我漸漸的就學了八年的日文，也因為遲遲沒有去面對它，導致於我現在跟2.3年前卡的瓶頸是一樣的完全沒有變。

一樣的關卡沒有去突破它就永遠卡在那個地方，因為我沒有做任何能改變的事情去改變，所以就算轉移注意力後，尋求其他的安慰後，回來我還是發現我停在原本的地方，卡一樣的點，絲毫沒有進步，接著在用一樣的方式去逃避。

但這次，我想我終於領悟到了，我趁這次跟日本朋友溝通後我發現我日文能力依舊很弱。

接著好好地仔細想一想，我知道為什麼我的日文跟我去年跟他見面的時候是停留在一樣的階層。

即便，我去年在跟他聊天後，就去報N2的考試然後努力的背單字加上刷題目，當然還是Fail了

Fail後，因為想去國外唸書，就開始轉戰托福了。

這一路來跌跌撞撞，直到今天，我發現我逃避後回來還是卡一樣的點。

這時候我可能有產生要回去念日文的想法。

但仔細想想我現在很努力在英文上，之後也要考托福還有準備出國。

我因為卡了停滯期所以認為我考不過那的托福的界線。

跟當初日文的狀態是一樣的，所以這次即便深感自己日文能力不足真的需要加強，但也讓我深深了解這道理，並把它套用在目前最需要努力跟一樣犯錯的英文身上。

「卡瓶頸或者卡停滯期才更要堅持與努力下去」

阿突然想到這句話用在減肥身上也行得通吧XD

即便，我可能真的想學習很多語言，但我覺得在這個節骨眼上，我必須好好的去思考我現在眼前要的是什麼？

剛剛碰巧翻到了會25國語言的台灣人，以前我也想當這樣的人，我覺得很帥很帥，會不同國家的語言。

但由於我並不了解我自己的個性，開始了解自己也是去年開始的事情，真正開始了解自己的時候，的卻碰到了很多迷茫跟迷惘，因為不知道自己要的是什麼且深怕自己會失敗或者害怕自己的未來。

現在的話，之前也是蠻想精通好幾國語言，但是我慢慢發覺到自己的個性也許在實行上有點困難，也許不適合，而這點還有待我去釐清。

所以我即便真的想要像他一樣精通很多國語言，但我在深刻思考後發現不太適合目前的我。

漸漸的就會發現，想要成為跟真的適合自己的其實有時候會是不同的。

突然領悟的到這些，真的帶著我的人生又向前跨了一大步！

加油！
