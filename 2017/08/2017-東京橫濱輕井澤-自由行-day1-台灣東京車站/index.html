2017 東京/橫濱/輕井澤 自由行 Day1 台灣~東京車站

August 26, 2017 in Travel Diary, 日本－東京

以往的遊記都在車上完成

因為乘車的時間夠長

這次東京我也是繞來繞去搭不同的地鐵跟工具跑來跑去

我都想吐啦XD

時間為8/21~8⁄25 五天

晚進晚出

人數：家庭3人，爸爸, 媽媽, 我

五天的住宿

  1. 東京車站 Sardonyx Tokyo

  2. 輕井澤 Chisun Inn Karuizawa

  3. 橫濱 元町中華街 Monterey Yokohama

  4. 東京 芝公園 Mielparque Tokyo

皆為三人房

三個人的房間如何定？

在Agoda或者相關訂房app, 搜尋的條件選3個人1間房

行程

  1. 東京車站 [住東京車站]

  2. 東京車站, 輕井澤, 雲場池, 王子購物廣場 [住輕井澤]

  3. 輕井澤, 聖保羅教堂, 增上寺, 諏訪神社, 橫濱, 山上公園, 21港未來 [住橫濱]

  4. 橫濱八景島, 東京芝公園, 東京鐵塔 [住東京芝公園]

  5. 上野, 阿美橫 回家

因為是第一天所以在文章開頭先把此次旅行細節交代一下

第一天的飛機是下午16點10分

到日本的時間應該是20點50分

我當時認為可以搭成田Express去東京，大約是一小時吧

沒有想到飛機有稍微的誤點，登機時間先是慢了40分

實際到的時候是晚上的20點55分

過海關的時候排得非常的長

因此真正出來搭車的時間為22:00左右

因為沒有事先查好去東京的列車

所以22:00之後已經沒有成田Express號了

22:30之後只有JR可以搭

因為我那時候還很茫然，所以想說先沈澱一下在看看能坐什麼

可是我爸實在是超級急，要我隨便找個人問

而我因為隨便找，我突然也就不知道說什麼了

突然要我問我也不知道問什麼

上了JR就開始跟我爸吵架了

我在飛機上看了一部電影叫：失落之城

看到一半看得出來快到高潮點，結果飛機廣播下降後

就沒得看了

害我在JR線上都很獄卒，因為看到一半，我又還蠻喜歡這部電影

很想知道後來在演什麼QQ

有JR可以搭

然後就一路摳囉摳囉搖搖遙搖了一個半多小時

約12點到東京車站

飯店，我記得在八丁掘站

出來走三四分就到了

今晚的飯店，我老爹他特別滿意

而我也很滿意

雖然桌子跟房間是稍微窄一點

但整體來說是蠻舒適的！

桌子旁邊還有一個大的液晶電視

床蠻軟的

今日住宿：東京車站 Sardonyx Tokyo / 東京寶石飯店

對於家庭旅行的飯店我給4分！

[2019 北海道 札幌/道東] DAY 5 釧路/塘路/釧路湿原/帶廣

 Feb 2, 2019

前往釧路湿原

原先是想搭乘早上八點的摩周號前往釧路湿原遊玩，規劃行程拜訪了其他部落客的行程，可以在釧路湿原站下車一路遊玩漫步到細岡站在搭列車到塘路站。沒想到後來我仔細觀察時刻表後才發現冬天早上的班次沒有一班列車會到達釧路湿原。

[2019 北海道 札幌/道東] DAY 4 網走/流冰破冰船/前往釧路

 Feb 2, 2019

網走

原先以為網走會是整躺行程最冷的地方，畢竟處於北海道右上方，加上海上的破冰船可能是開放空間氣溫直下零下10幾度，結果早上出太陽，體感溫度只有3-4度，特涼爽，破冰船船內也有冷氣。對比於我們對網走的幻象就是一個大雪大風的印象好像不是這麼的符合

[2019 北海道 札幌/道東] DAY 3 前往網走/札幌雪祭/旭川平和通買物公園

 Feb 2, 2019

第三天的行程被我媽稱為一波三折，受到前一天的大雪，使得今天大部分前往網走的列車都停駛，雖然最後晚上11點順利到達網走，但原先規劃下午在網走的行程都被迫取消，這也是我們在幾次自由行中第一次面臨的突發事件與抉擇。

不過雖然今日在網走的主要行程沒有去成，但也多兩個行程，包含參觀大通公園的2019年札幌雪祭與旭川的平和買物公園健行。最重要的是兩個人在討論進退的過程，挺充實的呢！
