盡量安排好自己的時間

August 3, 2017 minayu416

1 minute read

  • Twitter

  • Google+

  • Facebook

  • Reddit

  • LinkedIn

  • StumbleUpon

  • Pinterest

  • Email

當研究生，就是大部分的時間將自己奉獻給老師

這樣的模式我花蠻多時間才適應的

其中一個原因，主要是我想做的事情太多

我一直覺得老師讓我接的案子跟操我的內容

常常會讓我吃不消

好幾次我以為壓力的關係讓自己崩潰大哭

認為世界超級黑暗又很悲觀

但其實充其量也就是「沒有自己的時間」跟「沒有自己的空間」的關係

對於愛好自由又重自我空間的自己

的確會讓自己抓狂

誰希望在自己正在放鬆的時間，突然被call一聲，就要馬上趕去學校？

第二點，其實就是自己太急躁和想做太多事情

英文才認真念半年而已，真的半年而已，已經從矇懞懂懂的情況下，練出能寫一篇文章

研究生就是隨時隨地On call,

所以當初四月很徬徨的原因

主要就是因為，我想要自學的東西太多

但是我根本就沒這麼多時間自學

自學JS, Python還要自學英文德文跟日文

要做這麼多事情的情況下

其實在有限的時間內很難達到

為什麼會想學這麼多東西呢？主要就是因為不知道自己真正要的是什麼

但根據每次跟老闆還有老爹抱怨完之後

得到的答案就是好好唸英文

好像任何的難關都卡在英文

然而，也是真的卡在英文沒錯

不管是要進外商還是繼續念國外的學校

都卡在英文

而專業，其實已經在老師給的案子中慢慢的磨練了

即便我一開始很心慌，對於老師給予的東西，我不是很能理解

也認為老師是故意把自己不想做的事情丟給我

但後來慢慢發現，我們老師的確都在訓練自己，不管是寫網站還是收集資料

還有對於未來的老闆，研究所念的不是書的內容

而是如何有效在第一時間內收集資料，做出分析跟論證

這在大學其實是做不到的

但我當初抱的夢想的卻是能夠在研究所裡磨練我的程式能力

但我自己卻被一連串的徬徨給殺掉了

也許自己正在進步，但自己卻看不到

知曉這些事情後

我發現我能擁有自己的時間其實沒有這麼多

我不能夠再浪費時間在「同時做這麼多事」

於是我將眼前最重要最需要也是卡關的事情找出來

雖然每天的時間會被工作跟案子給壓著只有零星時間能用

但一有長假，二話不說先訂去玩的機票XD

再來就是用長假的時間充實自己

反而不會是耍廢

於是

我認為現階段的是

  1. 英文

  2. 讓自己健康

雖然我還是很喜歡日文跟西文還有之前誤打誤撞念的一些德文和俄文

但為了讓自己能夠成功，這次我不再逃避，而是更努力地一直念英文

以往日文考試的時候Fail我都會討厭日文一陣子

搞到自己念了八年日文，可能還沒有現在的英文程度好

讓自己健康，主要是常常處於高壓的狀態下

會睡不好，或者擔心徬徨，久而久之健康就不好

其實該感謝現在還活蹦亂跳的自己

別人工作下班後未必有精神能夠做自己想做的事

而我至少還能打起精神做自己的事情

最後

我還是告訴自己不要著急

日文我有八年的時間學習，但大部分其實都是因為每次日檢跟每次日文卡關我都讓自己逃避跟放棄

所以其實花了很多的時間在於逃避跟放棄

另外，熟能生巧

我背英文單字的方法是用17天GRE的方法

但我實在懶惰加上只有公車上時間才能背

背的進度雖然慢，但還是有進入腦袋

一開始托福都不曉得在幹嘛

但是隨著1000個單字進入腦袋後

我發現大部分托福的文章都看得懂了

再來就是每天聽不知所以然的60Sec 科學美國網還有每天川普川普的CNN

其實我只有泛聽沒有精聽，主要也是因為我都在交通時間聽，還蠻吵的XD

但是一段時間後雖然覺得沒什麼進步

但回來聽托福的時候有進步很多

其實很多事情都是熟能生巧

並沒有什麼輕鬆或者聰明就能拿高分

於是

雖然現在還是不知道我未來的道路是什麼

但我除了跟著老師工作還有接案子

一有自己的時間

我決定就要好好地拿來念英文還有運動讓自己更健康
