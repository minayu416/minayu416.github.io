人生道路茫然不孤單

計畫擬定

Posted on

是說，突如其來的行程亂了本來的Schedule.

去年進學校後開始很精密的計劃我的Schedule.

但，後來的結果不符合預期，一考完托福一戰。

也就不了了之。

在考試失敗後，整個計畫都被打亂了。

而在這茫茫然又徬徨的時刻

一進研究所，本來朝著我的夢想(網頁設計)前進！

結果被某某老師說寫網頁沒前途

我然就傻住了。

那….我就轉往….人工智慧或者資安前進囉？

但這些又覺得根本要去更高學府的地方或者更艱深的研究場所研究才能達到

這樣的東西，也許困難，而且要達成也不容易

但…除非要去研究，不然要靠這樣太專業的東西找工作，我覺得可能有點困難。

考慮到自己籌碼就是會寫網頁！

加上我突然被叫去國外幫忙！

可能種種的狀況暗示著自己

這麼看來，在這個我一進研究所就開始徬徨，然後沖托福沖到一半又不知道自己在奮鬥什麼！

重要的還是，我的各種夢(轉變了好幾次)，都能一一的被各種原因打碎。

未來是幻想啊！腳踏的現在才是真的，至少我現在是這麼想的！

受到學長的指導＋上去國外幾天回來。

我更確定些我現在該做的事情。

即便我真的不知道我的未來是如何。

我聽說，國外的人都是有自信的自薦自己。

因此，我只要有足夠的實力，就能推薦自己得到機會！

那麼接下來就不用去思考跟安排未來了

再來訂一下努力的方面跟Schedule

接下來的方向

  • 語言能力

  • 學業/論文

  • 專業能力

  • 啊～運動！

項目

  • 語言能力

這方面還是以拚托福為主要！畢竟不管交換生還是碩士都需要托福

但還是希望以增進英文能力來唸托福

我可不想托福考得不錯結果不會跟人說話(N1考試不會跟人講話？)

如果有時間在K日文好了，但我覺得應該是沒時間，剛好我有買日文的程式書，加減看加減進步XD

  • 專業能力

有別於之前把這個剔除，但我回來唸研究所就是要更增進這方面的知識才對！

讓學長指導兩句後，我的網頁夢又回來了XD

之前是為了想試試看當UX設計師才回來

但不知道為什麼問了一堆人都不知道UX是什麼= =

而且才赫然發現自己其實大學申請的時候最想要學習寫網頁@@!

結果忘記這個夢也太久了

列出幾個出國前筆記的

  1. VM

  2. Plone

  3. JavaScript

  4. Python

  5. Database

  6. UX

  7. 數據分析(論文)

  8. 資安(網安),破解,加密

SSD~ <好吧這是學長希望我可以凹到經費去買來放在學校的研究室。

  • 學業/論文

看起來學業是不用有什麼太大的困擾，就是期中跟期末考需要專注一下！

論文就比較困難了，要秉持著每週進度，然後要去了解，依照我這叛逆的個性，我不會只讓自己只忙一個論文，這樣一定混boring。

但也許會偷偷被罵萬一論文不過怎辦XD

人在求生的時候會使出渾身解數的!!!!

況且我還要用英文寫論文呢！！！！！別忘記，所以英文能力很重要的！！

  • 運動吧～！

好吧，夏天要到了，估計自己訂了幾個運動可以變化比較不會無聊

  1. 萬用健走

  2. 游泳

  3. workout

有時間就是健走，游泳是輔助(因為會覺得無聊，而且游泳消耗熱量大)

再者沒時間就做workout 用youtube找

我現在還在煩惱我有這麼多事情要做，我哪有這麼多時間？

就像書上說的，做就對了，哪管有沒有時間

也像某位PPT仁兄說的，時間就像乳溝一樣擠一擠就有！

我這次不打算特別偏向哪一個

但說真的我要考托福，應該是要偏向全準備英文

全準備英文的風險就是放掉學習專業能力跟論文

英文考好了，專業能力跟論文自然也就ＧＧ了

這樣算是分散風險，現在是4月，訂個7月跟9月檢視一下自己的英文能力

把唸英文的時間拉長好像還是比較保險

所以綜合以上種種關聯跟any connection, 我結合出以下的結論XD

前期

  • 語言能力

以增強英文能力為主，主要考托福

聽說讀寫都要運用到，大量看跟大量聽去砸！

偶爾要寫跟說！

  1. 背單字2500

  2. 看英文論文

  3. 聽60sec,TED,CNN

  4. 每週寫文章，偶而用部落格寫文章

  5. 有時候可以用說的錄音來記錄每天做的事情！

  • 專業能力

  1. VM (應該是指可能會需要遠端遙控)

  2. 切換使用者(是說這功能也要學習)

  3. Plone

  4. JavaScript

  5. Python

以做網站為主

(將日本買的書看一看)

  • 論文

收集資料，多看論文，寫proposal

中期

  • 語言能力

托福技巧，托福考試

實力應該是起來了，就開始做題然後繼續練！

  • 專業能力

  1. Database

  2. UX

  • 論文
  1. 數據分析

應該是要用大數據去分析了，這階段通常就是測試分析資料跟準備寫論文！

後期

  • 專業能力

資安(網安),破解,加密

沒有接觸過的東西，看起來有點困難的東西，自然先擺在最後在學習囉！

然後再把時間切為3或者4等分！

目的就是不要有一方佔超過時間！

要平均去進行！

接下來就…..自己在用每個星期的Schedule去計劃吧XD!
