分散式學習

April 15, 2017 minayu416

1 minute read

  • Twitter

  • Google+

  • Facebook

  • Reddit

  • LinkedIn

  • StumbleUpon

  • Pinterest

  • Email

首先下這個標題，我就覺得根本不知道要放在哪個分類

來趁因為生日多聚會多派對的情況下無法好好有時間唸書或者學習來生個文章XD

我想來分享我最近覺得很有效的學習方法

對我很有效XD

我先來分析一下我自己的個性，我不知道這方法適不適用每個人，但我的個性是這樣

  • 要學的事情多(不同領域)

  • 喜歡有自己的時間自學，不喜歡給老師教或侷限

  • 容易對一件事感到厭煩

  • 不愛專注於一件事

  • 個性很急或者太熱情

  • 找到開心的方法學習(或學習你喜歡學得)

  • 無法短期努力拼

從以前補習的時候我就知道自己比起老師教學或者去上補習班我更喜歡自己學

但自學的效果就是怕會懶惰，因為我以前常常有這種懶惰事

我本來的命好像就不太適合考試，偏偏我又得面對一堆考試

於是不斷嘗試後發現我自己準備考試的一堆習慣之類的

因為現在我必須得準備托福，但以我這個很難長期抗戰準備考試的人真的困難

首先，我從日本回來我就突然發現我有一堆一堆一堆事情跟知識等著我去學。

依照我去年準備導遊考試跟以往都會專心準備日檢或者專心準備____。

我發現最要不得的就是專注於準備一個東西

先來說一下什麼是分散式學習

這詞是我自己發明的XD

分散式學習

就是把一堆想學的事情分類，然後打散開來學習，以週為單位。

很難理解吧。

基本上就是把自己有空的時間做最有效的利用。

假設自己是一個學生(好吧我就是學生)

畫一個星期一~星期天的計劃表

假設這是我的計劃表，我先把星期一~星期四我有課的時間給填上去

然後要去扣除每天通車三個小時(包含午餐晚餐)

接下來就去把自己想學的東西寫下來

  • 準備托福(分聽,說,讀,寫)

  • 學架網站，Open Source (Plone, Wordpress)

  • 學程式語言：JS, Python, JQuery….

  • 論文時間/課業時間/Big Data/爬蟲

  • 運動時間/減肥時間

看….起來好多好多好多

喔對了，基本上是沒什麼休假跟玩樂時間的，這點可能要注意。

我個人目前最大的放鬆活動就是：健走, 聽歌寫程式, 寫部落格。

由於我之前花了2.3年的時間狂打魔獸，爬天梯前幾然後當各種會長團長再推進什麼進度團首殺團。

所以遊戲的巔峰基本上我是打過了也打膩了，現在才不會想打什麼遊戲。

首先就是準備托福，他分聽說讀寫四個部分，剛開始我也覺得很困難去準備，覺得我根本沒時間。

後來我找到一點方法，累積後發現自己英文有點進步，所以應該照做就行

托福準備

  • 閱讀：將手機, 電腦全轉成英文, 上資訊爬文都是看英文, 論文也是看英文, 英文單字就在公車上通車時間背

  • 聽力：[每天早上]花一個小時~半小時聽60sec Scientific American, 因為我很討厭聽寫，所以我選擇跟讀的方法，聽力還沒練成不太能講什麼，[晚上]偶爾聽TED/CNN/college 101

  • 口說：用手機錄音，用講英文的方式錄日記，或者介紹一個主題，大約15-20分，每兩天一次，或者每天，[通常在早上]。

  • 寫作：每週寫一篇TPO的題目給英文老師改，[每兩三天]用英文寫日記寫在部落格上，不要放過任何可以寫英文跟練習英文的機會

架網站/程式語言

這項目是日本回來後硬是加上去的

基本上這些不太可能一次學，所以我分成兩三層部分來學

就是近期學, 第二期學, 第三期學

第一期學完換第二期學的東西

原則上我分[每天晚上]的時間在這塊部分

第一期

  • Plone (一三五)

  • JS, Python (二四六)

第二期

  • Git/Github

  • JQuery, Ajax, Python

第三期

  • Wordpress

  • 程式語言

類似這樣，一三五晚上學其中一個，二四六學另外一個

論文時間/課業時間/Big Data/爬蟲

由於現階段指導老師沒空理我這小妹妹，所以論文的時間我把他暫時放在

五六日的下午

因為幸運的話，通常…..兩個下午努力一點就能達成本週進度

運動時間/減肥時間

高效能運動為主

像是夏天了，我就用每週兩天,45分鐘自由式游泳

持續個幾週，搭配飲食喔～

游泳對我來說算是有天賦的運動，但是唯一覺得無聊的就是….他不能聽歌

我最愛的運動是聽歌健走(因為很舒壓)，一次走一個半到兩個小時。

但太花時間了，所以可能會先以游泳為主

游泳畢竟就是以減肥為目的當然無法舒壓，所以到了真的不想做再換別的

健走或者去找youtube的workout來做

大致上就是這樣

因為要做的事情要學習的東西很多所以一定要制定一個日程表！！！！

  • 要學的事情多(不同領域)

基本上看起來真的學很多很多東西XD！太貪心了我！！但也是因為以前花了2.3年時間在打遊戲，現在才會矛起來學一堆東西，還有就是我確定好我要走的路了，才會努力的去學！

  • 喜歡有自己的時間自學，不喜歡給老師教或侷限

這部分就是每次我想學什麼會先考慮到補習，可是我後來真的發現我很討厭一週在固定的時間做固定的事(就是”必須”在那個時間點去補習)，愛好自由！ 而且通常補完習回家就不會唸了！

  • 容易對一件事感到厭煩

有朋友跟我說我不太適合一次只做一件事，因為很容易膩，很無聊，雖然我可能會被罵說你連好好準備一個托福要準備到高分都很困難了還覺得無聊，但我就是覺得一直唸英文就很無聊QQ，會開始想做其他事情

  • 不愛專注於一件事

跟上一點很像很像很像，之前因為太專注英文，搞得自己壓力很大，太專注於準備導遊，這樣分散式學習有點分散投資，但就是都有在學習就是都有進步

  1. 只專注於一件事，萬一那件事Fail就完了，壓力就很大

  2. 只專注於一件事，萬一那件事情Fail，那段時間就等於浪費了，你那段時間也沒有其他東西在進步

可能我比較悲觀吧….就是覺得全投資在一個東西上，失敗了，就等於什麼都沒有，所以才會分散投資到其他地方，可能有人會覺得就是全投資在一個東西上才會讓那個東西成功。

經過我自己精密的分析，分散投資還是好得多…

  • 個性很急或者太熱情

以前我會在專心拼日文的時候，一直想著拼完日文後我就要怎樣怎樣，讓自己拼日文的時後先期待考完後的事情，因此，乾脆準備時間拉長快樂學習，也把之後考完想做的事情挪到前面一起做！

這就好比，我不想要先學完plone在學js，所以我乾脆分成一三五學plone，二四六學js，因為我現在就想學js

  • 找到開心的方法學習(或學習你喜歡學得)

討厭或沒興趣的一路不碰，但如果是像英文這樣必須學的，那就想辦法讓它變得有趣。

我是迷上用英文寫部落格跟講英文錄日記，因為我蠻尊敬崇拜一個youtuber。

既然要時間拉長分散投資，那學習東西的時候要有開心的學習過程！

  • 無法短期努力拼

我完全無法短期努力拼，我這個人完全經不起高壓，一有壓力我就焦慮，患得患失然後緊張兮兮怕東怕西。所以拉長時間每天準備。
