Re-Adjust

October 4, 2017 minayu416

1 minute read

  • Twitter

  • Google+

  • Facebook

  • Reddit

  • LinkedIn

  • StumbleUpon

  • Pinterest

  • Email

我猜應該就是今晚，或者明天了

面對這樣的狀況，我大概是少數還能冷靜地看著他

早上我看過我表姐在床前面哭一次了

我偷偷走進阿公的房間裡

對著他小聲地說了好幾句很冷靜的話

不同於其他面對親人臥床也許淚灑大哭

我則是很冷靜的面對他，這大概就是因為我是很怪咖的人吧

靈魂困在老化無法使用的軀體裡一定很痛苦，希望之後能夠讓靈魂自由自在的 我知道你一定很擔心你其中一個女兒，可是她應該不知道這樣的事情，我們也沒告訴她 也許哪天她知道了以後，會崩潰大哭吧 人的一生就是出生，幼年，青年，青壯年，壯年，偏老年，老年 相信你青壯年的時候一定很勇敢，相信你年輕時一定過得很精彩 抱歉，我也許無法理解你之前對我做了多好的事情，因為我現在也在徬徨裡，正在經歷青年時期應該要做的事情

另外一方面，我正在一個覺得自己很孤單

我大概煩悶了三四天，去躲避任何的人

論文，衛福部，上課也不想上

但我還是有去思考有去做，並不像真的傷很重的人完全放棄

今天，我深刻的記得，我在我手機的行事曆上寫下

該重新調整腳步了 你這樣不行喔

面對情感上的孤僻跟孤單就是因為自己沒有時間社交

但因為這樣而讓自己整體的狀況停滯不前

懲罰的會是自己

於是，我想起之前的目標跟想做的事情

不該懶惰而是前進讓自己進步

這些事情沒有終點，而是要告訴自己會持續的進步的！

重新調整好腳步，繼續出發

也許孤單是一時的，但磨鍊出來的技能是真實的，當你以後站上大舞台時，你會感謝自己一直在一個人奮鬥努力的時光

Our mind is free, we are learning SOFTWARE PROGRAMMING, we can CREATE everything!

也許台灣這塊小天地，無法達成夢想，那就跨出去到自由的國度去自由的實現自己想做的事

  • 程式

  • 英文

  • 健康(身材/保養)

當踏到這步，你會發現，你當初所以為自己可能適合「外語系」「應用外語系」，其實自己更適合的是「資訊科技」

無論多麼的孤單

你只要好好的Hold住自己的夢想跟理想就好

夢想就是

自由地去做自己想做的事 也許是去旅行，也許是去嘗試不同的冒險，也許是用自己的技能創業與其他...... 不要被現有的框架給侷限住，因為你的夢想就是思想自由，大膽用自己的雙手跟頭腦去開創世界
