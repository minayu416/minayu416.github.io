在短短的兩年從極度自卑沒自信到現在

正所謂經歷一連長串困難致人於死地的磨練後

通過的人將成為更屹立不搖的王者

2015年，ˋ是一個具有代表性的年份

在2014年的12月多，我在睡夢中突然驚醒，接著我的頭像是火在燃燒一樣的痛，然後頓時我就像無法呼吸一樣，大力的呼吸可望吸到更多氧氣

我好害怕，我到底怎麼了，於是我當天就去掛急診，半夜。

沒事，開了藥回來吃，結果更慘，後來發現發高燒跑去看小診所才好

我那次沒有意識到，但我想那個應該就是恐慌吧？

那個時候我不知道什麼是壓力，原來從那時候開始我就已經埋下了焦慮跟恐慌的種子

2015/02我在電玩世界裡，經歷了好多好多事情

這時候的我，是一個很胖很胖的女生，每天坐在電腦前打魔獸世界

沒有幾張照片，有一個男朋友，但不知道為什麼漸漸不喜歡了，交往了兩年半，我就跟他分手了

很殘忍的在情人節前一天分手。

然後每天都在電玩世界裡和我所謂的網友們混在一起

在4月達到電玩的人生高峰

天梯前幾名，還有一堆人想挖我去進排名前面的拓荒團

後來大概是吵架了，團也散了，我眼看著大概沒有工作了？

我就準備收山了，那時候跟電玩的一位男子曖昧

就這樣一路曖昧到7月，畢業了，我仍然不知道要做什麼

在資訊系裡面，為了擺脫跟大家一樣的叛逆，我故意跑去學日文

結果在快畢業的時候，我的日文也沒有達到所謂能找工作的N2

2月跟男友分手後就開始出去健走，接著就瘦一大圈

剛畢業，我認為自己需要考個英文證明，於是找了一陣子英文家教

後來想應徵美甲師，但預約了沒去

之後我免強的將我大一到大四上的多媒體課程的作品集結起來作成作品集

讓媽媽的朋友看到後建議我去上一個空間設計班

也就是室內設計

在室內設計中，一樣想脫離跟大家一樣，於是我再次認為我自己是資訊系的

班上的電腦常常壞，修電腦也就修出感情了

知道自己對電腦還有興趣，但剛從室內設計班出來，為了不要讓自己沒事做

聽了我爸的意見去報日文導遊

那幾個月也是很喜歡筊杯，在菩薩面前根本就一直無杯

後來才發現自己根本就沒要找工作，而是出現另外一個選項

對的，我決定回研究所好好的唸書

我只投我們學校的資訊研究所

我只想進這個研究所

因為只投一家，我當時壓力很大

大約2016/1月，我開始有失眠睡不著的狀況出現

當時正在準備研究所的被審資料

2月還好，就在準備導遊考

但是到3月左右，整個大爆發

開始我的心跳跳得很大力又很快，每晚都失眠完完全全睡不著

每天就是怕東怕西，眼睛還很澀很模糊很模糊，很緊張很緊張

2016堪稱人生最黃金時段

我於2016年1月1日戒掉魔獸世界

專心報考我的研究所

所有的焦慮和壓力壓垮我

在逛街的時候被媽媽朋友，阿姨們發現我可能有自律神經失調症狀

聽起來頗悲劇

但是在我每天都生活在那樣不正常的環境下，連續幾天的失眠

讓我大哭崩潰著求我媽帶我去精神科看醫生

被回絕

最後到了失眠的第四天，真的是我人生中最痛苦的時間點

我感覺我再也快樂不起來了，感覺我的體力全數被抽乾

那天走在那條熟悉的路上

我真的再也受不了了，彷彿再睡不著就想自殺

聽我媽的建議，我去了1月失眠曾經求診過的中醫

中醫一個安穩的話，拿了藥回家吃

還是很害怕，晚餐後吃完一包中藥

我開始慢慢地感覺到

我為什麼要想這麼多？

心情安定很多很多

然後那天我就睡了一個好覺

終於，睡著了。

並不是睡一個好覺就一切都好

在後續的幾個月，已經造成的傷害「一直緊張非常焦慮，心悸，眼睛模糊，腦袋趴帶」

聽從專題老師的建議，我去台大重頭上C語言

接著我選擇了大學中令人聞風喪膽的程式老師當我的論文指導老師

開始了我的程式之路

這個階段，我非常的沒有自信覺得自己很爛

當我接到我的指導老師想要我試圖抄抄看MIT網站，作為我學習HTML, CSS跟程式語言的入門關卡

考驗就開始了

> > ### 如果沒有挑戰過，你怎麼知道你的極限在哪？ > >

指導老師拎著我到他的秘密研究室

這裡面坐了三個學弟

然後問我，你會寫什麼程式？

我什麼都不會，你要收我嗎？

好大的膽字哈哈…..

於是我教授又問我了，你對程式有多少覺悟？

我想要成為一個很厲害的電腦工程師

第一週，七天內學會HTML並拉出一個簡單網頁的框架

一開始我看著HTML的語法，連續練習跟背了三天，三天後我看完了所有HTML的語法

然後指導老師跟我說開始學JS，我就開始看JS的語法

我知道我離Deadline剩下四天，但我還是花了一天背JS的語法

背到第五天，我終於發現

這樣不行，我得開始做

於是我一邊想辦法拉框架，一邊查遇到問題怎麼辦

終於在第七天拉出了簡單簡單的網站架構，給教授看

一切就像打電玩一樣

所以我之前有寫過「我的電玩哲學」

做出東西，我解開了我的新成就：我得到了研究助理的工作

是幫學校的新單位寫網頁

第二週 把MIT的網站全版抄下來試試看

這個時候我已經覺得很難了，我覺得我不可能達到

我就跟我的指導老師說我覺得我不行

而我的指導老師回我說：你可以的要相信自己，我相信你

第二週如願以償的，我將MIT整版抄下來

第三週之後，我開始學後台語言PHP

雖然是蠻無感的

但後來我漸漸發現自己自學能力變得很強

PHP倒是卡了蠻久

經歷4個月後，我開始接觸到比較困難的程式架構：Django in Python

第一次接觸到終端機，第一次接觸到伺服器，好多的第一次，我開眼界

而我旁邊的那個學弟很喜歡把我踩在地板上

很愛嗆我，讓我覺得自己很爛很爛

於是，我就下定決心，我就是不要問你，我自己想辦法

我一定要比你厲害

雖然在他旁邊我還是多少受到影響

因為我的指導老師歷年來的學生都是程式高手

而我？就是一個什麼都不會的菜鳥

當時我有個夢就是希望能夠在碩三去美國拿雙聯學位

於是我很努力的拚托福，剛開學我就開始拚托福

碩一上整個就是忙到炸掉了

而且還經歷很多同學上老師上的衝康

那幾個月的精神壓力也很大，每週有四天要去補習班補托福

隨著第一次的模擬考考出來分數”47”吧？ 我忘了

離所謂的雙聯學制門檻81還差很多

常常我會因為報告太多，指導老師還想操我，沒時間唸英文而跟他吼

暑假我考完我的第一次托福”57”分，這個分數很詭異，不巧剛好等於托福的550~600

所以用這個分數底調研究所英文門檻，加上用這個分數報名美國企業參訪團

發現雙聯學制不是唸資訊的，我有想要去德國留學

但一直一直一直被我爸唱衰跟阻止

那個時候我也很不開心，覺得我的命運跟我作對

我什麼都無法做決定，因為永遠達不到

開始耍任性，開始覺得有夢沒用，這個世界好爛

無法留學的事實讓我大受打擊

2017⁄4 我回來找我的指導老師

然後開始被抓去做一些行政的工作

行政工作的繁瑣讓我哭過幾次

再來就是讓我很討厭的衛福部的案子

那段期間是一波超大的壓力，每過幾天就會哭

還在別的教授的課上大哭

在研究室，學弟們面前大哭

在公車上在捷運上在路上突然就掉眼淚是很平凡的事情

我覺得自己依舊很爛很爛，我做出來的東西無法獲得教授的賞識

他認為我都亂做，做的東西都不能看

後來我才聽學弟們說，教授都會偷偷趁我不再跑來研究所叫學弟們多關心我

說我其實很自卑很沒自信

撐過那段時期後，核銷工作越來越上手

而我開始作出老闆想要的東西

在某次開會的時候，我老闆直接用我做出來的東西

我當下真的是差點想跪在地上大哭

但也漸漸的習慣了這樣極大壓力的方式

最近很少哭了

因為哭完事情還在

雖然在某次極大壓力的情況下，有出現之前恐慌的症狀

但後來慢慢知道，慌沒有用，哭沒有用

事情下來就是把握時間做好

不管多忙，時間控管好，還是OK的

大概在8.9月吧，我開始覺得我的身體健康很差

我希望能夠開始改正作息讓自己變得健康

加上在6.7月的時候也是有疑似憂鬱的狀況出現

那幾天我就冒出了一個觀念

沒自信幹什麼呢？

與其沒自信自卑憂鬱到極點

不如試試看樂觀點，試一次讓自己自傲讓眼睛長在頭上

無論是憂鬱還是焦慮甚至是恐慌我都覺得，任何精神上的問題，都會造成非常非常痛苦的生活

於是我告訴自己打死我都不要在憂鬱了，打死都別在焦慮了事情來了就做

從沒自信到有自信

從有點快精神病的狀態到現在慢慢克服了

克服精神病後的心理會更加強大

再來就是意識到健康很重要

關卡其實就是我的電玩哲學

唯一的難點就是-沒有重來/也沒有無限條命

不然我認為自己在電玩裡交得到朋友，到現時也可以

電玩裡爬到天梯前幾，我在研究所也能在前幾

電玩裡受到高手賞識，我在研究所也能夠受到教授賞識

電玩內我可以技能練很強什麼的

我在人生中一樣可以練好我的程式學好我的英文

練好情緒控管練好時間管理

雖然我也沒有到眼睛長頭上啦XD

但我深深覺得不要讓自己悲觀，因為悲觀只會讓自己的精神惡化，更加痛苦

如果沒有挑戰過，你怎麼知道你的極限在哪？

我就是什麼都不會，但我就是敢挑戰，我現在寫程式的功力也不會差到哪去

有自信點，大膽地給他Try下去吧！

現在的我，我很滿意現在的生活

也慢慢確定我未來的方向和目標了

-獻給我這朵可愛的八年級溫室小花朵
