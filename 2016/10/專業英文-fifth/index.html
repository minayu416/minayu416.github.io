However 不等於 but

主要是副詞，所以不能跟but混用

But is coordinating conjunction and can link two independent clauses.

However is an adverb and cannot link two independent clauses.

不要把However 當作一個連接詞

下次記得帶血型閱讀

下週單字

  1. Library n.圖書館
    I study English in library.
  2. Comfortable (adj.) 舒服的
    There is a comfortable chair.
  3. Convenient (adj.) 方便的
    7-11 are convenient.
  4. Far (adj.) (adv.) 遠
    school is very far.
  5. Investigate (vt.) (vi.) 研究 調查
    I am investigate his life now.
  6. Program (n.) 節目程式 規劃
    I write a program on computer.
  7. Script  (n.) 手稿 劇本 原本
    He write a script for this movie.
  8. Structure (n.) 結構 構造 建築物 (v.) 建造 建築 構成
    Data structure is a good cource in university.
  9. Python (n.) 蟒蛇 (n.) 程式語言的軟體
    Python is a computer language.
  10. Debug (vt.) 除錯 (v.) 調試
    programmer is debugging now.
