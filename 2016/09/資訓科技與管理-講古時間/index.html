＃講古時間

Parallel Computing and Networking 平行處理概念

**-Parallel Computing **
Computers rely on interconnected central processing and/for memory units that increase processing speed.

Multi-tasking 多功能處理
某個時間載入一個工作，然後在某個時期又載入另外一個工作。在同個時段，有記憶體讓出去又有記憶體載進來。

[](https://www.blogger.com/null)
**-Networking 交換網路 **

1.LAN (Local Area Network) 區域網路
-在小範圍的地方有，大量密集的電腦
2.WAN 網域網路
-長呈特性

“光”通訊的發展，在傳遞的時候減少了衰弱性，所以讓網際網路變得簡單多了。
如果說，從台北拉一條光纖到新竹，還是能跑很快。

Ethernet(乙太網路) connects small computers to share resources
File servers connect PCs in the late 1980s

Google 光纖網路大概是25Tbes

強調網路可以互聯所以需要整合 > 互聯網

**APPANET and LAN > Internet **

互聯網，早從70年代就從美國國防部開始發展。
國防部DoD計畫在1970年開始發展，初步的成果>APPANET
加入了很多人之後轉為民營，慢慢轉變為現在的網際網路。

**Supercomputer **
-Cray 2 電腦泡在液體裡，因為高速運轉下，發熱效果高。

**Network computing **
-LAN computing
-Blade server 刀鋒伺服器，各個小刀鋒擁有自己的CPU跟RAM
一次可以裝10幾個，後面有一個後台在處理每台小刀鋒的資料交換
-Data center #Google的資料中心在彰化
雲端運算最難的就是，到底要如何連結幾十萬台電腦的資訊交換，誰在做誰的事，誰知道。

**Applications of Internet **
-BBS
最麻煩的是每天都要記一堆網址。
-Gopher
美國明尼蘇達大學所發明出來，希望解決使用者需要記住一堆網址的問題，選單式程式。

**WWW **
-Netscape
-IE
-Mosaic
剛開始網路還都是全用英文交流。
剛開始台灣想要中文化，但是那時代還在各自發展，各個公司開發各個自己的中文化系統然後互不相容。

所以那時候在歐洲有一堆很強的科學家為了傳遞圖片所以開始研發可以傳遞圖片的網路。
WWW > 開始讓大家可以在網路上看到文字以外的東西，例如：圖片

但剛開始想使用WWW必須先有：1.server 2.瀏覽器
一開始的Server就是Mosiac
然後一群人就創立了Netscape公司也是瀏覽器。

90年代的時候微軟如日中天。

90年代以後，賈伯斯回到蘋果後，蘋果慢慢變強。

90年代後期，全世界60%電腦在台灣組裝，100%的筆電也在台灣組裝。

1997年，Win95開始出現，對微軟是一個非常非常大的躍進。

命令提示字元(終端機)：就是當時95的畫面。

Win95被啟動之後，附送IE：只要看到https:// 就自動幫你用IE打開網頁

Netscape 要錢，搶資源又輸IE自然就被淘汰。

有了WWW之後就進入的電子商務的時代

**e-Commerce 電子商務 **

在電子商務時代的公司，其實大家都是在追求夢想。

但那時候微軟已經達到一個神的階級，無人能敵。

有兩個年輕人去找比爾蓋茲說他們有個公司跟瀏覽器希望你買，被蓋茲發卡後。

這兩個年輕人就是現在的Google。

＃程式寫不出來最好的就是轉換環境跟思想(發呆)。

無名小站原本是七個交大的同學，結果沒多久就把交大的網站弄掛了。
被學校趕出去，被投資人找上，49%/51%分成立公司，投資人就去找奇摩併購無名小站，用七億買。

Facebook跟youtube都是個傳奇。

-Facebook是哈佛的學生剛開始為了想要認識學弟妹的學生。
-Youtube 四個好朋友的家庭想要分享生活跟人生。

First Generation Software (1951-1959) 
-Machine Language 
Computer programs written in binary (1s and 0s)

-Assembly Language and Translators 
Programs written using mnemonics, which were translated into machine language

-Programmer Changes 
Programmers divide into two groups:application programmers and systems programmers(寫系統的人).

資工：偏向工程，偏向硬體，去研究電腦的架構跟記憶體。
資科：比較偏重於平台，比方說你要怎麼去研發一個高效率高搜尋的資料庫。
資管：如何製作做一個應用系統，和使用者溝通如何製作，把使用者的需求轉換成電腦語言。
＃所以資管的人才必須要會一點各領域的皮毛。

App是在手機上運行的應用程式
手機網頁，就是需要用瀏覽器去啟動的

Second Generation Software

-高階語言

-低階語言

Third Generation Software

-System Software 

作業系統本身就是一個電腦和使用者的互動

-Separation between Users and Hardware

區分使用者跟硬體

檔案總管就是一個系統的軟體，可以用高階語言和低階語言來寫，現在大部分都用高階語言來寫。

Unix 本身是用C寫的

Forth Generation Software (1971-1989)

-Structured Programming

Pascal

C++

-New Application Software for Users

Spreadsheets

Word processors

Database management systems

舉BASIC為例

Fifth Generation Software
-Microsoft
-Object-Oriented Design
-World Wide Web
-New Users
